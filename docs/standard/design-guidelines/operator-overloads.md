### <a name="conversion-operators"></a><span data-ttu-id="8c1b0-129">Konvertierungsoperatoren</span><span class="sxs-lookup"><span data-stu-id="8c1b0-129">Conversion Operators</span></span>  
 <span data-ttu-id="8c1b0-130">Konvertierungsoperatoren sind unäre Operatoren, die Konvertierung von einem Typ in einen anderen zu ermöglichen.</span><span class="sxs-lookup"><span data-stu-id="8c1b0-130">Conversion operators are unary operators that allow conversion from one type to another.</span></span> <span data-ttu-id="8c1b0-131">Die Operatoren müssen als statische Member auf den Operanden oder der Rückgabetyp definiert werden.</span><span class="sxs-lookup"><span data-stu-id="8c1b0-131">The operators must be defined as static members on either the operand or the return type.</span></span> <span data-ttu-id="8c1b0-132">Es gibt zwei Typen von Konvertierungsoperatoren: implizite und explizite.</span><span class="sxs-lookup"><span data-stu-id="8c1b0-132">There are two types of conversion operators: implicit and explicit.</span></span>  
  
 <span data-ttu-id="8c1b0-133">**X DO NOT** Geben Sie einen Konvertierungsoperator aus, wenn eine solche Konvertierung nicht eindeutig von den Endbenutzern erwartet wird.</span><span class="sxs-lookup"><span data-stu-id="8c1b0-133">**X DO NOT** provide a conversion operator if such conversion is not clearly expected by the end users.</span></span>  
  
 <span data-ttu-id="8c1b0-134">**X DO NOT** Konvertierungsoperatoren außerhalb der Domäne des Typs definiert.</span><span class="sxs-lookup"><span data-stu-id="8c1b0-134">**X DO NOT** define conversion operators outside of a type’s domain.</span></span>  
  
 <span data-ttu-id="8c1b0-135">Z.
