# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>Modernisieren Sie vorhandene .NET apps für Cloudoptimierte Anwendungen

> Vision: Modernisieren Sie Ihre vorhandenen .NET Framework-Anwendungen zu Cloudoptimierte Anwendungen drastisch Ihre Bereitstellung zu verbessern, sodass Sie schneller liefern und senken die Kosten für die Bereitstellung der Anwendung.

Um die Vorteile der Cloud und neuer Technologien wie Container nutzen zu können, sollten Sie Ihre vorhandenen .NET-Anwendungen zumindest teilweise modernisieren. Letztlich wird die Modernisierung Ihrer Unternehmensanwendungen Ihre Gesamtbetriebskosten (Total Cost of Ownership) senken.

Teilweise Modernisierung einer app, nicht unbedingt eine vollständige Migration und Neuentwurf bedeuten. Sie können zunächst mit wichtigen, aber einfach Modernisierung Ihrer vorhandenen Anwendungen modernisieren. Sie können Ihre aktuelle Codebasis, die in vorhandenen .NET Framework-Versionen geschrieben wurde, mit allen Windows- und IIS-Abhängigkeiten pflegen. Abbildung 4-1 zeigt, wie durch die Cloud optimierten apps, die im Azure-anwendungsmodernisierung positioniert werden.
