# <a name="grpc-in-production"></a>gRPC in der Produktion

ASP.net Core 3,0-Anwendungen, einschließlich GrpC-Dienste, können unter Windows, Linux und in Containern unter Verwendung moderner Plattformen wie docker und Kubernetes ausgeführt werden. In diesem Kapitel werden die verschiedenen Optionen zum Ausführen ihrer GrpC-Dienste in der Produktionsumgebung erläutert. Außerdem werden die Überwachungs-und Protokollierungs Optionen erläutert, um den optimalen Betrieb von Systemen sicherzustellen.
