## <a name="handling-partial-failure"></a><span data-ttu-id="75b10-114">Behandeln von Teilfehlern</span><span class="sxs-lookup"><span data-stu-id="75b10-114">Handling partial failure</span></span>

<span data-ttu-id="75b10-122">Ein ähnliches Szenario kann in einer APP auftreten, die HTTP-Dienste nutzt.</span><span class="sxs-lookup"><span data-stu-id="75b10-122">A similar scenario might occur in an app that consumes HTTP services.</span></span> <span data-ttu-id="75b10-123">Das Netzwerk oder der Dienst selbst ist in der Cloud möglicherweise nicht während eines kurzen vorübergehenden Fehlers verfügbar.</span><span class="sxs-lookup"><span data-stu-id="75b10-123">The network or the service itself might not be available in the cloud during a short, transient failure.</span></span>
