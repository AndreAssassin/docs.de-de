# <a name="azure-serverless-platform"></a>Serverlose Azure-Plattform

Die serverlose Azure-Plattform umfasst Azure Functions, Logic Apps und Event Grid. Diese Dienste arbeiten zusammen und stellen eine Verbindung mit unzähligen anderen Ressourcen her. Die serverlose Plattform arbeitet mit allen Komponenten zusammen, von Datenbanken über Speicher und Analysen bis hin zu Machine Learning/künstlicher Intelligenz.

Sie können auch Application Insights verwenden, eine serverlose Plattform zum Erfassen von Diagnoseablaufverfolgungen und Telemetriedaten. Application Insights ist für Anwendungen aller Art (Desktop, mobil oder Web) und serverlose Implementierungen verfügbar. Die Plattform wird in der folgenden Abbildung dargestellt:
