# <a name="azure-serverless-platform"></a>Server lose Azure-Plattform

Die Server lose Azure-Plattform umfasst Azure Functions, Logic apps und Event Grid. Diese Dienste arbeiten zusammen und stellen eine Verbindung mit unzähligen anderen Ressourcen her. Die Server lose Plattform funktioniert mit sämtlicher Datenbanken, Speicherung und Analyse und Maschinelles Lernen/künstliche Intelligenz.

Sie können auch Application Insights verwenden, eine Server lose Plattform zum Erfassen von Diagnose Ablauf Verfolgungen und Telemetriedaten. Application Insights sind für Anwendungen aller Typen (Desktop, Mobil oder Web) und Server lose Implementierungen verfügbar. Die Plattform wird in der folgenden Abbildung dargestellt:
