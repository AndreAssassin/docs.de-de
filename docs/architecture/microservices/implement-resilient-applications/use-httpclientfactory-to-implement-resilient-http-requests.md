### <a name="httpclient-lifetimes"></a><span data-ttu-id="218d4-148">HTTPClient-Lebensdauer</span><span class="sxs-lookup"><span data-stu-id="218d4-148">HttpClient lifetimes</span></span>

<span data-ttu-id="218d4-151">Das Zusammenlegen von Handlern ist wünschenswert, da jeder Handler in der Regel über seine eigenen HTTP-Verbindungen verfügt.
