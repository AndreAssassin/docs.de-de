# <a name="breaking-changes-in-windows-forms-net-framework-to-net-core"></a>Breaking Changes bei Windows Forms (.NET Framework zu .NET Core)

WPF- und Windows Forms-Unterstützung wurde zu .NET Core in Version 3.0 hinzugefügt. Wenn Sie eine Windows Forms- oder Windows Presentation Foundation-App von .NET Framework zu .NET Core migrieren, können sich die in diesem Artikel aufgeführten Breaking Changes auf Ihre App auswirken.

Breaking Changes sind nach der .NET Core-Version gruppiert, in der sie eingeführt wurden.
