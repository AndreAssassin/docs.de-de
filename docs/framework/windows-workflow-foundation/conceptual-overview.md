 Beschreibt die Grundlage, mit der Benutzersystem-oder Benutzer Workflows in Ihren Anwendungen erstellen können, die für die Betriebssysteme Windows Vista, [!INCLUDE[wxp](../../../includes/wxp-md.md)], [!INCLUDE[ws2003](../../../includes/ws2003-md.md)]und [!INCLUDE[lserver](../../../includes/lserver-md.md)] geschrieben wurden.  
  
 [Grundlegende Konzepte für Windows-Workflows](fundamental-concepts.md)
