# <a name="2026---compilevbexpressionstart"></a>2026 - CompileVbExpressionStart
## <a name="properties"></a>Eigenschaften  
  
|||  
|-|-|  
|ID|2026|  
|Schlüsselwörter|WFRuntime|  
|Ebene|Ausführlich|  
|Kanal|Microsoft-Windows-Application Server-Applications/Debug|  
  
## <a name="description"></a>Beschreibung  
 Gibt den Anfang einer VB-Ausdruckskompilierung an.  
  
## <a name="message"></a>Meldung  
 Der VB-Ausdruck '%1' wird kompiliert.  
  
## <a name="details"></a>Details  
  
|Datenelementname|Datenelementtyp|Beschreibung|  
|--------------------|--------------------|-----------------|  
|Ausdruck|xs:string|Der zu kompilierende VisualBasic-Ausdruck.|  
|AppDomain|xs:string|Die von AppDomain.CurrentDomain.FriendlyName zurückgegebene Zeichenfolge.|
