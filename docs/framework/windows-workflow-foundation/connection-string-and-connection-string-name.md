# <a name="connection-string-and-connection-string-name"></a>Verbindungszeichenfolge und Name der Verbindungszeichenfolge
**Verbindungszeichenfolge** Eigenschaft gibt die Verbindungszeichenfolge, die die SQL-Workflow-Instanz Store verwenden soll, eine Verbindung zur persistenten Datenbank herstellen. Dieser Parameter ist ein optionaler Parameter. **Name der Verbindungszeichenfolge** Eigenschaft gibt den Namen der benannten Verbindungszeichenfolge an, die die SQL-Workflow-Instanz Store verwenden soll, eine Verbindung mit der persistenten Datenbank herstellen. Dieser Parameter ist ein optionaler Parameter. Wenn Sie nicht über die SQL-Workflow-Instanz Store die standardmäßige Verbindungszeichenfolge verwenden möchten, geben Sie einen Wert für die Verbindungszeichenfolgenname-Eigenschaft oder die Verbindungszeichenfolge sollte **DefaultSqlWorkflowInstanceStoreConnectionString** .
