# <a name="migration-activity-in-wf"></a>Migrationsaktivität in WF
[!INCLUDE[netfx_current_long](../../../includes/netfx-current-long-md.md)]stellt die <xref:System.Activities.Statements.Interop> Aktivität zum Ausführen von Aktivitäten bereit, die von einer Aktivität in einem Workflow abgeleitet <xref:System.Activities.Activity>werden, der auf basiert. Weitere Informationen finden Sie im Abschnitt [Migrations Leit Faden](migration-guidance.md) .  
  
> [!NOTE]
> Die <xref:System.Activities.Statements.Interop> -Aktivität wird nicht in der Toolbox des Workflow-Designers angezeigt, es sei denn, für das Projekt des Workflows ist die **Ziel Framework** -Einstellung auf **.NET Framework 4** oder höher festgelegt.
