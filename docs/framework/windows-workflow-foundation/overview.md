# <a name="windows-workflow-overview"></a><span data-ttu-id="88328-102">Übersicht über Windows-Workflow</span><span class="sxs-lookup"><span data-stu-id="88328-102">Windows Workflow Overview</span></span>
<span data-ttu-id="88328-103">Ein Workflow ist eine Reihe von elementaren Einheiten mit dem Namen *Aktivitäten* , die als Modell, das einen realen Vorgang beschreibt gespeichert sind.</span><span class="sxs-lookup"><span data-stu-id="88328-103">A workflow is a set of elemental units called *activities* that are stored as a model that describes a real-world process.</span></span> <span data-ttu-id="88328-104">Mit Workflows können die Reihenfolge der Ausführung sowie abhängige Beziehungen zwischen kurz- und langfristiger Arbeit beschrieben werden.</span><span class="sxs-lookup"><span data-stu-id="88328-104">Workflows provide a way of describing the order of execution and dependent relationships between pieces of short- or long-running work.</span></span> <span data-ttu-id="88328-105">Diese Arbeit durchläuft das Modell vom Anfang bis zum Ende, und Aktivitäten werden unter Umständen von Personen oder Systemfunktionen ausgeführt.</span><span class="sxs-lookup"><span data-stu-id="88328-105">This work passes through the model from start to finish, and activities might be executed by people or by system functions.</span></span>  
  
## <a name="workflow-run-time-engine"></a><span data-ttu-id="88328-106">Workflowruntime-Engine</span><span class="sxs-lookup"><span data-stu-id="88328-106">Workflow Run-time Engine</span></span>  
 <span data-ttu-id="88328-107">Jede ausgeführte Workflowinstanz wird von einer prozessinternen Runtime-Engine erstellt und verwaltet, mit dem der Hostprozess durch eines der folgenden Objekte interagiert:</span><span class="sxs-lookup"><span data-stu-id="88328-107">Every running workflow instance is created and maintained by an in-process run-time engine that the host process interacts with through one of the following:</span></span>

- <span data-ttu-id="88328-109">Ein <xref:System.Activities.WorkflowApplication>-Objekt für die explizite Kontrolle über die Ausführung einer einzelnen Workflowinstanz.</span><span class="sxs-lookup"><span data-stu-id="88328-109">A <xref:System.Activities.WorkflowApplication> for explicit control over the execution of a single workflow instance.</span></span>  
  
- <span data-ttu-id="88328-110">Ein <xref:System.ServiceModel.WorkflowServiceHost>-Objekt für meldungsbasierte Interaktionen in Szenarien mit mehreren Instanzen.</span><span class="sxs-lookup"><span data-stu-id="88328-110">A <xref:System.ServiceModel.WorkflowServiceHost> for message-based interactions in multi-instance scenarios.</span></span>
