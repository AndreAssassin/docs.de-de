# <a name="getting-started-with-wcf-data-services"></a>Erste Schritte mit WCF Data Services
Die Themen in diesem Abschnitt helfen Ihnen, schnell zu verstehen [!INCLUDE[ssODataFull](../../../../includes/ssodatafull-md.md)] und Verwendung [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] Verfügbarmachen und verarbeiten [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] feeds, indem die zugrunde liegenden Technologien. Dieser Abschnitt enthält sowohl konzeptionelle Inhalte und [Schnellstart](../../../../docs/framework/data/wcf/quickstart-wcf-data-services.md) Tutorial.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 In den folgenden Themen wird beschrieben, wie Sie Datendienste mit [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] erstellen.  
  
 [Verfügbarmachen der Daten als Dienst](../../../../docs/framework/data/wcf/exposing-your-data-as-a-service-wcf-data-services.md)  
 Beschreibt die erforderlichen Schritte zum Erstellen eines Datendiensts mit [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)].  
  
 [Zugreifen auf Datendienstressourcen](../../../../docs/framework/data/wcf/accessing-data-service-resources-wcf-data-services.md)  
 Beschreibt das Arbeiten mit [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] -feeds.  
  
 [Verwenden eines Datendiensts in einer Clientanwendung](../../../../docs/framework/data/wcf/using-a-data-service-in-a-client-application-wcf-data-services.md)  
 Beschreibt das Arbeiten mit einer [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] feed in einer .NET Framework-Clientanwendung.  
  
 [Schnellstart](../../../../docs/framework/data/wcf/quickstart-wcf-data-services.md)  
 Zeigt, wie erstellen und Zugreifen auf eine einfache [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]-basierten Diensts, der einen Feed basierend auf der Northwind-Beispieldatenbank verfügbar macht.  
  
 [Anwendungsszenarios](../../../../docs/framework/data/wcf/application-scenarios-wcf-data-services.md)  
 Erläutert die wichtigsten [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]-Szenarien, die von [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] unterstützt werden.  
  
 [Ressourcen](../../../../docs/framework/data/wcf/wcf-data-services-resources.md)  
 Enthält Links zu [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] und [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] Ressourcen.
