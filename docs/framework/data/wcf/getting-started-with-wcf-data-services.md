# <a name="getting-started-with-wcf-data-services"></a>Erste Schritte mit WCF Data Services
Die Themen in diesem Abschnitt helfen Ihnen dabei, [!INCLUDE[ssODataFull](../../../../includes/ssodatafull-md.md)] schnell zu verstehen und [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] zu verwenden, um [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] Feeds verfügbar zu machen und zu nutzen, indem die zugrunde liegenden Technologien erläutert werden. Dieser Abschnitt enthält sowohl konzeptionelle Inhalte als auch ein [Schnellstart](quickstart-wcf-data-services.md) -Tutorial.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
 In den folgenden Themen wird beschrieben, wie Sie Datendienste mit [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] erstellen.  
  
 [Verfügbarmachen der Daten als Dienst](exposing-your-data-as-a-service-wcf-data-services.md)  
 Beschreibt die erforderlichen Schritte zum Erstellen eines Datendiensts mit [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)].  
  
 [Zugreifen auf Datendienstressourcen](accessing-data-service-resources-wcf-data-services.md)  
 Beschreibt das Arbeiten mit [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] Feeds.  
  
 [Verwenden eines Datendiensts in einer Clientanwendung](using-a-data-service-in-a-client-application-wcf-data-services.md)  
 Beschreibt, wie mit einem [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] -Feed in einer .NET Framework Client Anwendung gearbeitet wird.  
  
 [Schnellstart](quickstart-wcf-data-services.md)  
 Zeigt, wie Sie einen einfachen [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]-basierten Dienst erstellen und darauf zugreifen, der einen auf der Northwind-Beispieldatenbank basierenden-Feed verfügbar macht.  
  
 [Anwendungsszenarios](application-scenarios-wcf-data-services.md)  
 Erläutert die wichtigsten [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)]-Szenarien, die von [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] unterstützt werden.  
  
 [Ressourcen](wcf-data-services-resources.md)  
 Stellt Links zu [!INCLUDE[ssAstoria](../../../../includes/ssastoria-md.md)] - [!INCLUDE[ssODataShort](../../../../includes/ssodatashort-md.md)] und-Ressourcen bereit.
