2. <span data-ttu-id="1cf5b-119">Führen Sie an der Eingabeaufforderung den folgenden Befehl ohne Zeilenumbrüche aus:</span><span class="sxs-lookup"><span data-stu-id="1cf5b-119">At the command prompt, execute the following command without line breaks:</span></span>

