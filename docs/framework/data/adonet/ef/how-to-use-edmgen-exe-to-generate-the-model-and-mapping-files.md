### <a name="to-generate-the-school-model-for-a-visual-basic-project-using-edmgenexe"></a><span data-ttu-id="2961c-116">So erstellen Sie mithilfe von 'EdmGen.exe' das Modell 'School' für ein Visual Basic-Projekt</span><span class="sxs-lookup"><span data-stu-id="2961c-116">To generate the School model for a Visual Basic project using EdmGen.exe</span></span>

2.  <span data-ttu-id="2961c-119">Führen Sie an der Eingabeaufforderung den folgenden Befehl ohne Zeilenumbrüche aus:</span><span class="sxs-lookup"><span data-stu-id="2961c-119">At the command prompt, execute the following command without line breaks:</span></span>

    ```  
    /c:"Data Source=%datasourceserver%; Initial Catalog=School; Integrated Security=SSPI"   
    /project:School /entitycontainer:SchoolEntities /namespace:SchoolModel /language:VB  
    ```
