Der Microsoft SQL Server-ODBC-Treiber unterstützt neben den allgemeinen schemaauflistungen die folgenden spezifischen schemaauflistungen:

- Tabellen

- Indizes

- Spalten

- Verfahren

- ProcedureColumns

- ProcedureParameters
