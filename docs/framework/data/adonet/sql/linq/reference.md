# <a name="reference"></a>Referenz
Dieser Abschnitt enthält Referenzinformationen für [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)]-Entwickler.  
  
 Außerdem wird empfohlen, Microsoft-Dokumentation auf bestimmte Probleme zu durchsuchen und besonders an dem [LINQ-Forum](https://go.microsoft.com/fwlink/?LinkId=76488)teilzunehmen, in dem Sie komplexere Themen ausführlich mit Experten erörtern können. Darüber hinaus steht Ihnen ein White Paper zur [!INCLUDE[vbtecdlinq](../../../../../../includes/vbtecdlinq-md.md)]-Technologie zur Verfügung. Dieses enthält auch Codebeispiele zu Visual Basic und C#. Weitere Informationen finden Sie unter [LINQ to SQL: .NET Language-Integrated Query für relationale Daten](https://go.microsoft.com/fwlink/?LinkId=93205).
