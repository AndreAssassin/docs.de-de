- <span data-ttu-id="e362f-113">Die Authentifizierung ist der Prozess der Anmeldung bei SQL Server. - <span data-ttu-id="e362f-115">Bei der Autorisierung wird ermittelt, auf welche sicherungsfähigen Ressourcen ein Prinzipal zugreifen kann und welche Vorgänge für diese Ressourcen erlaubt sind.</span><span class="sxs-lookup"><span data-stu-id="e362f-115">Authorization is the process of determining which securable resources a principal can access, and which operations are allowed for those resources.</span></span>
