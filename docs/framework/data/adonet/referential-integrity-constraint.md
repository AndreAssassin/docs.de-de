- <span data-ttu-id="1e4d5-109">Das Prinzipalende der Einschränkung.</span><span class="sxs-lookup"><span data-stu-id="1e4d5-109">The principal end of the constraint.</span></span> <span data-ttu-id="1e4d5-110">(Ein Entitätstyp, auf dessen Entitätsschlüssel durch das abhängige Ende verwiesen wird.)</span><span class="sxs-lookup"><span data-stu-id="1e4d5-110">(An entity type whose entity key is referenced by the dependent end.)</span></span>  
  
- <span data-ttu-id="1e4d5-111">Den Entitätsschlüssel des Prinzipalendes.</span><span class="sxs-lookup"><span data-stu-id="1e4d5-111">The entity key of the principal end.</span></span>  
  
- <span data-ttu-id="1e4d5-112">Das abhängige Ende der Einschränkung.</span><span class="sxs-lookup"><span data-stu-id="1e4d5-112">The dependent end of the constraint.</span></span> <span data-ttu-id="1e4d5-113">(Ein Entitätstyp, der über eine Eigenschaft oder Eigenschaften verfügt, die auf den Entitätsschlüssel des Prinzipalendes verweisen.)</span><span class="sxs-lookup"><span data-stu-id="1e4d5-113">(An entity type that has a property or properties that reference the entity key of the principal end.)</span></span>
