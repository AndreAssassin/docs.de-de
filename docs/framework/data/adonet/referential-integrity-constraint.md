-   <span data-ttu-id="fa621-111">Den Entitätsschlüssel des Prinzipalendes.</span><span class="sxs-lookup"><span data-stu-id="fa621-111">The entity key of the principal end.</span></span>  
  
-   <span data-ttu-id="fa621-112">Das abhängige Ende der Einschränkung.</span><span class="sxs-lookup"><span data-stu-id="fa621-112">The dependent end of the constraint.</span></span> <span data-ttu-id="fa621-113">(Ein Entitätstyp, der über eine Eigenschaft oder Eigenschaften verfügt, die auf den Entitätsschlüssel des Prinzipalendes verweisen.)</span><span class="sxs-lookup"><span data-stu-id="fa621-113">(An entity type that has a property or properties that reference the entity key of the principal end.)</span></span>
