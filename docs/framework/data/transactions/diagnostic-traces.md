## <a name="enabling-tracing"></a><span data-ttu-id="366c3-116">Aktivieren der Ablaufverfolgung</span><span class="sxs-lookup"><span data-stu-id="366c3-116">Enabling Tracing</span></span>  
 <span data-ttu-id="366c3-117">Um die Ablaufverfolgung während der Transaktionsverarbeitung zu aktivieren, müssen Sie die Konfigurationsdatei der Anwendung bearbeiten.</span><span class="sxs-lookup"><span data-stu-id="366c3-117">To enable traces during transaction processing, you should edit your application’s configuration file.</span></span> <span data-ttu-id="366c3-118">Im Folgenden finden Sie ein Beispiel.</span><span class="sxs-lookup"><span data-stu-id="366c3-118">The following is an example.</span></span>  
  
```xml

<system.diagnostics>  
     <sources>  
          <source name="System.Transactions" switchValue="Warning">  
               <listeners>  
                    <add name="tx"
                     type="System.Diagnostics.XmlWriterTraceListener"
                     initializeData= "tx.log" />  
               </listeners>  
          </source>  
     </sources>  
</system.diagnostics>  
</configuration>  
```
