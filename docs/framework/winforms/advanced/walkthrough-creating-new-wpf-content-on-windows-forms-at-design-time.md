   <span data-ttu-id="edcc6-118">Das Hinzufügen des neuen Steuer Elements hat die folgenden Auswirkungen:</span><span class="sxs-lookup"><span data-stu-id="edcc6-118">Adding the new control has the following effects:</span></span>

   - <span data-ttu-id="edcc6-119">Die Datei UserControl1.xaml wird hinzugefügt.</span><span class="sxs-lookup"><span data-stu-id="edcc6-119">File UserControl1.xaml is added.</span></span>

XAML. vb) wurde hinzugefügt.</span><span class="sxs-lookup"><span data-stu-id="edcc6-120">File UserControl1.xaml.cs (or UserControl1.xaml.vb) is added.</span></span> <span data-ttu-id="edcc6-121">Diese Datei enthält das Code-Behind-Modell für Ereignishandler und andere Implementierungen.</span><span class="sxs-lookup"><span data-stu-id="edcc6-121">This file contains the code-behind for event handlers and other implementation.</span></span>

5. <span data-ttu-id="edcc6-127">Legen Sie im Fenster **Eigenschaften** den Wert der <xref:System.Windows.Controls.TextBox.Text%2A>-Eigenschaft auf **gehosteter Inhalt**fest.</span><span class="sxs-lookup"><span data-stu-id="edcc6-127">In the **Properties** window, set the value of the <xref:System.Windows.Controls.TextBox.Text%2A> property to **Hosted Content**.</span></span>

   > <span data-ttu-id="edcc6-128">Normalerweise sollten Sie anspruchsvolleren WPF-Inhalt hosten.</span><span class="sxs-lookup"><span data-stu-id="edcc6-128">In general, you should host more sophisticated WPF content.</span></span> <span data-ttu-id="edcc6-129"><xref:System.Windows.Controls.TextBox?displayProperty=nameWithType>-Steuerelement wird hier nur zur Veranschaulichung verwendet.</span><span class="sxs-lookup"><span data-stu-id="edcc6-129">The <xref:System.Windows.Controls.TextBox?displayProperty=nameWithType> control is used here for illustrative purposes only.</span></span>

<span data-ttu-id="edcc6-134">So fügen Sie einem Windows Form ein WPF-Steuerelement hinzu:</span><span class="sxs-lookup"><span data-stu-id="edcc6-134">To add a WPF control to a Windows Form:</span></span>

    - <span data-ttu-id="edcc6-140">Verweise auf WPF-Assemblys werden dem Projekt hinzugefügt.</span><span class="sxs-lookup"><span data-stu-id="edcc6-140">References to WPF assemblies are added to the project.</span></span>
