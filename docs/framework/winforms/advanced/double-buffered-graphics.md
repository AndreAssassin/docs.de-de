# <a name="double-buffered-graphics"></a><span data-ttu-id="978af-102">Doppelt gepufferte Grafiken</span><span class="sxs-lookup"><span data-stu-id="978af-102">Double Buffered Graphics</span></span>
<span data-ttu-id="978af-103">Flimmern ist ein häufiges Problem beim Programmieren von Grafiken.</span><span class="sxs-lookup"><span data-stu-id="978af-103">Flicker is a common problem when programming graphics.</span></span> <span data-ttu-id="978af-104">Grafikoperationen, die mehrere komplexe Zeichenvorgänge erfordern, können dazu führen, dass gerenderte Bilder flimmern oder anderweitig nicht ordnungsgemäß dargestellt werden.</span><span class="sxs-lookup"><span data-stu-id="978af-104">Graphics operations that require multiple complex painting operations can cause the rendered images to appear to flicker or have an otherwise unacceptable appearance.</span></span> <span data-ttu-id="978af-105">Um diese Probleme zu beheben, stellt .NET Framework Zugriff auf doppelte Pufferung bereit.</span><span class="sxs-lookup"><span data-stu-id="978af-105">To address these problems, the .NET Framework provides access to double buffering.</span></span>
