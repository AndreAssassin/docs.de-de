# <a name="how-to-split-a-window-horizontally-using-the-designer"></a>Vorgehensweise: Horizontales Teilen eines Fensters mithilfe des Designers

In diesem Beispiel wird der Splitter, der <xref:System.Windows.Forms.SplitContainer> das-Steuerelement horizontal dividiert.

> [!NOTE]
> Die <xref:System.Windows.Forms.SplitContainer.Orientation%2A> -Eigenschaft <xref:System.Windows.Forms.SplitContainer> des-Steuer Elements bestimmt die Richtung des Splitters, nicht des Steuer Elements selbst.

## <a name="to-split-a-window-horizontally"></a>So teilen Sie ein Fenster horizontal

Legen Sie im Fenster **Eigenschaften** <xref:System.Windows.Forms.SplitContainer.Orientation%2A> die-Eigenschaft des <xref:System.Windows.Forms.SplitContainer> -Steuer Elements <xref:System.Windows.Forms.Orientation.Horizontal>auf fest.
