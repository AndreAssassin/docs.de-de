# <a name="how-to-move-a-toolstrip-out-of-a-toolstripcontainer-onto-a-form"></a>Vorgehensweise: Verschieben eines ToolStrip aus einem ToolStripContainer auf ein Formular
Verwenden Sie das folgende Verfahren, um <xref:System.Windows.Forms.ToolStrip> einen aus einem <xref:System.Windows.Forms.ToolStripContainer> in ein Formular zu verschieben.

## <a name="to-move-a-toolstrip-out-of-a-toolstripcontainer-onto-a-form"></a>So verschieben Sie ein ToolStrip aus einem ToolStripContainer auf ein Formular

1. Wählen Sie das <xref:System.Windows.Forms.ToolStrip>-Steuerelement aus.

2. Drücken Sie STRG + X, oder klicken Sie <xref:System.Windows.Forms.ToolStrip> mitderrechtenMaustasteaufdas,undwählenSieausdemKontextmenüdieOptionAusschneidenaus.<xref:System.Windows.Forms.ToolStrip>

3. Wählen Sie das Formular aus.

4. Fügen Sie <xref:System.Windows.Forms.ToolStrip> den durch Drücken von STRG + V ein, oder wählen Sie im Menü **Bearbeiten** die Option **Einfügen** aus.

5. Legen <xref:System.Windows.Forms.ToolStrip> Sie <xref:System.Windows.Forms.ToolStrip.Dock%2A> die-Eigenschaft von auf **oben**fest.
