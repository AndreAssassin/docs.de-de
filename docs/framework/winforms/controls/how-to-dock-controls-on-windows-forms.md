### <a name="to-dock-a-control"></a><span data-ttu-id="0b37a-109">So docken Sie ein Steuerelement an</span><span class="sxs-lookup"><span data-stu-id="0b37a-109">To dock a control</span></span>  
  
1. <span data-ttu-id="0b37a-110">Wählen Sie das Steuerelement, das Sie andocken möchten.</span><span class="sxs-lookup"><span data-stu-id="0b37a-110">Select the control that you want to dock.</span></span>

     <span data-ttu-id="0b37a-112">Ein Editor wird mit einer Reihe von Feldern, die darstellt, die Kanten und der Mitte des Formulars angezeigt.</span><span class="sxs-lookup"><span data-stu-id="0b37a-112">An editor is displayed that shows a series of boxes representing the edges and the center of the form.</span></span>  
  
3. <span data-ttu-id="0b37a-113">Klicken Sie auf die Schaltfläche ", die den Rand des Formulars darstellt, in dem das Steuerelement angedockt werden soll.</span><span class="sxs-lookup"><span data-stu-id="0b37a-113">Click the button that represents the edge of the form where you want to dock the control.</span></span> <span data-ttu-id="0b37a-114">Klicken Sie auf das mittlere Feld, um den Inhalt des Steuerelements Formular oder Containersteuerelement zu füllen.</span><span class="sxs-lookup"><span data-stu-id="0b37a-114">To fill the contents of the control's form or container control, click the center box.</span></span> <span data-ttu-id="0b37a-115">Klicken Sie auf **(keine)** zum Andocken zu deaktivieren.</span><span class="sxs-lookup"><span data-stu-id="0b37a-115">Click **(none)** to disable docking.</span></span>  
  
     <span data-ttu-id="0b37a-116">Das Steuerelement wird automatische größenanpassung die Grenzen der verankerten Kante.</span><span class="sxs-lookup"><span data-stu-id="0b37a-116">The control is automatically resized to fit the boundaries of the docked edge.</span></span>  
  
    > [!NOTE]
    >  <span data-ttu-id="0b37a-117">Geerbte Steuerelemente muss `Protected` angedockt werden kann.</span><span class="sxs-lookup"><span data-stu-id="0b37a-117">Inherited controls must be `Protected` to be able to be docked.</span></span> <span data-ttu-id="0b37a-118">Um die Zugriffsebene eines Steuerelements zu ändern, legen Sie dessen **Modifizierer** Eigenschaft im Eigenschaftenfenster angezeigt.</span><span class="sxs-lookup"><span data-stu-id="0b37a-118">To change the access level of a control, set its **Modifier** property in the Properties window.</span></span>
