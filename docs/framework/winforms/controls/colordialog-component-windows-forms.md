# <a name="colordialog-component-windows-forms"></a>ColorDialog-Komponente (Windows Forms)
Die Windows <xref:System.Windows.Forms.ColorDialog> Forms-Komponente ist ein vorkonfiguriertes Dialogfeld, in dem der Benutzer eine Farbe aus einer Palette auswählen und benutzerdefinierte Farben zu dieser Palette hinzufügen kann. Es entspricht dem Dialogfeld, das in anderen Windows-basierten Anwendungen zur Farbauswahl angezeigt wird. Verwenden Sie es in Ihrer auf Windows basierenden Anwendung als einfache Lösung, anstatt ein eigenes Dialogfeld zu konfigurieren.
