## <a name="example"></a><span data-ttu-id="aa941-140">Beispiel</span><span class="sxs-lookup"><span data-stu-id="aa941-140">Example</span></span>

```xml

         <dependentAssembly>

      </assemblyBinding>
   </runtime>
```
