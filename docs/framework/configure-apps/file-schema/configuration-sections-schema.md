# <a name="configuration-sections-schema"></a>Schema für Konfigurationsabschnitte

Allgemeine Informationen zu Konfigurationsdateien und Schemas finden Sie unter [Konfigurationsdateischema für .NET Framework](~/docs/framework/configure-apps/file-schema/index.md).
