# <a name="policyimporters"></a>\<policyImporters>
Dieses Konfigurationselement gibt alle Richtlinienimporter an, die den Import von benutzerdefinierten Richtlinienassertionen zu Bindungen steuern. Jedes untergeordnete Element ist eine`policyImporter`< >, die verwendet wird, um benutzerdefinierte Richtlinien Assertionen über Bindungs Features zu suchen. Außerdem wird ein benutzerdefiniertes Bindungs Element angefügt, das die für die Voraussetzung erforderlichen Funktionen implementiert.
