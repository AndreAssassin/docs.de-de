# <a name="tokenrequestparameters"></a>\<tokenRequestParameters>
Dieses Konfigurationselement enthält untergeordnete Elemente, die zum Angeben zusätzlicher XML-Elemente verwendet werden können, die beim Anfordern eines Tokens im Nachrichtentext an den Sicherheitstokendienst gesendet werden sollen.
