# <a name="cordebughandletype-enumeration"></a>CorDebugHandleType-Enumeration
Gibt den Handletyp an.  
  
## <a name="syntax"></a>Syntax  
  
```  
typedef enum CorDebugHandleType {  
    HANDLE_STRONG                  = 1,  
    HANDLE_WEAK_TRACK_RESURRECTION = 2  
} CorDebugHandleType;  
```  
  
## <a name="members"></a>Member  
  
|Member|Beschreibung|  
|------------|-----------------|  
|`HANDLE_STRONG`|Das Handle ist stark, dadurch wird verhindert, dass ein Objekt durch die Garbagecollection freigegeben wird.|  
|`HANDLE_WEAK_TRACK_RESURRECTION`|Das Handle ist schwach ist, die ein Objekt verhindert nicht, dass durch die Garbagecollection freigegeben.<br /><br /> Das Handle wird ung√ºltig, wenn das Objekt gesammelt werden.|
