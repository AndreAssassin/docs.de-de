# <a name="icordebugstepperenum-interface"></a>ICorDebugStepperEnum-Schnittstelle
ICorDebugEnum-Methoden implementiert, und listet ICorDebugStepper-Arrays.  
  
## <a name="methods"></a>Methoden  
  
|Methode|Beschreibung|  
|------------|-----------------|  
|[Next-Methode](../../../../docs/framework/unmanaged-api/debugging/icordebugstepperenum-next-method.md)|Ruft die angegebene Anzahl von `ICorDebugStepper` Instanzen aus der Enumeration, die an der aktuellen Position ab.|  
  
## <a name="remarks"></a>Hinweise  
  
> [!NOTE]
>  Diese Schnittstelle kann weder computerübergreifend noch prozessübergreifend remote aufgerufen werden.
