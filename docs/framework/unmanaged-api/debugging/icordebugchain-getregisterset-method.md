# <a name="icordebugchaingetregisterset-method"></a>ICorDebugChain::GetRegisterSet-Methode
Ruft den Register Satz für den aktiven Teil dieser Kette ab.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT GetRegisterSet (  
    [out] ICorDebugRegisterSet **ppRegisters  
);  
```  
  
## <a name="parameters"></a>Parameters  
 `ppRegisters`  
 vorgenommen Ein Zeiger auf die Adresse eines [ICorDebugRegisterSet](icordebugregisterset-interface.md) -Objekts, das den Register Satz für den aktiven Teil dieser Kette darstellt.
