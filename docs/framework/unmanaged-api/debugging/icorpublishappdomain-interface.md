# <a name="icorpublishappdomain-interface"></a>ICorPublishAppDomain-Schnittstelle
Stellt Informationen zu einer Anwendungsdomäne dar und bereit.  
  
## <a name="methods"></a>Methoden  
  
|Methode|Beschreibung|  
|------------|-----------------|  
|[GetID-Methode](../../../../docs/framework/unmanaged-api/debugging/icorpublishappdomain-getid-method.md)|Ruft den eindeutigen Bezeichner für dieses `ICorPublishAppDomain`ab.|  
|[GetName-Methode](../../../../docs/framework/unmanaged-api/debugging/icorpublishappdomain-getname-method.md)|Ruft den Namen der Anwendungsdomäne ab, auf die von diesem `ICorPublishAppDomain`verwiesen wird.|
