# <a name="cor_prf_gc_reason-enumeration"></a>COR_PRF_GC_REASON-Enumeration
Zeigt den Grund, weshalb die Garbage Collection stattfindet.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
typedef enum {  
    COR_PRF_GC_INDUCED = 1,  
    COR_PRF_GC_OTHER = 0  
} COR_PRF_GC_REASON;  
```  
  
## <a name="members"></a>Member  
  
|Member|Beschreibung|  
|------------|-----------------|  
|`COR_PRF_GC_INDUCED`|The garbage collection was induced by a <xref:System.GC.Collect%2A> method.|  
|`COR_PRF_GC_OTHER`|The reason is unspecified.|
