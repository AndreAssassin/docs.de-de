# <a name="icorprofilerinfo10suspendruntime-method"></a>ICorProfilerInfo10:: suspendruntime-Methode
  
 Hält die Laufzeit an, ohne eine GC auszuführen.   
  
## <a name="syntax"></a>Syntax  
  
```cpp
HRESULT SuspendRuntime();
```
