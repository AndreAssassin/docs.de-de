# <a name="imetadataimportgetnestedclassprops-method"></a>IMetaDataImport::GetNestedClassProps-Methode
Ruft das TypeDef-Token für die übergeordnete <xref:System.Type> des angegebenen, nicht angegebenen Typs ab.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT GetNestedClassProps (  
   [in]   mdTypeDef      tdNestedClass,  
   [out]  mdTypeDef      *ptdEnclosingClass  
);  
```  
  
## <a name="parameters"></a>Parameter  
 `tdNestedClass`  
 in Ein TypeDef-Token, das den <xref:System.Type> darstellt, für den das übergeordnete Klassen Token zurückgegeben werden soll  
  
 `ptdEnclosingClass`  
 vorgenommen Ein Zeiger auf das TypeDef-Token für den <xref:System.Type>, in dem `tdNestedClass` eingefügt ist.
