# <a name="imetadataemitsetmoduleprops-method"></a>IMetaDataEmit::SetModuleProps-Methode

## <a name="syntax"></a>Syntax  
  
```  
HRESULT SetModuleProps (   
    [in]  LPCWSTR     szName  
);  
```
