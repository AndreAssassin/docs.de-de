# <a name="imetadataemit2savedeltatostream-method"></a>IMetaDataEmit2::SaveDeltaToStream-Methode
Speichert Änderungen aus der aktuellen Edit-and-Continue-Sitzung im angegebenen Stream.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT SaveDeltaToStream (  
    [in] IStream     *pIStream,
    [in] DWORD       dwSaveFlags  
);  
```  
  
## <a name="parameters"></a>Parameter  
 `pIStream`  
 [in] Ein Schnittstellenzeiger auf den beschreibbaren Stream, in dem Änderungen gespeichert werden sollen.  
  
 `dwSaveFlags`  
 [in]: Reserviert Dieser Wert muss null (0) sein.
