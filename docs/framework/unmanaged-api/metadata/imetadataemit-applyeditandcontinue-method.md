# <a name="imetadataemitapplyeditandcontinue-method"></a>IMetaDataEmit::ApplyEditAndContinue-Methode
Aktualisiert den aktuellen Assemblybereich mit den Änderungen, die in den angegebenen Metadaten vorgenommen wurden.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT ApplyEditAndContinue (
    [in]  IUnknown    *pImport  
);  
```  
  
## <a name="parameters"></a>Parameter  
 `pImport`  
 \[im\] Zeiger auf ein [IUnknown-Objekt,](/cpp/atl/iunknown) das die Deltametadaten aus der portablen ausführbaren Datei (PE) darstellt.
  
 Die Deltametadaten sind der Metadatenblock, der die Änderungen enthält, die an der Kopie der tatsächlichen Metadaten des Moduls vorgenommen wurden.
