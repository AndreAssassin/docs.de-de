# <a name="imetadataemitsetmethodimplflags-method"></a>IMetaDataEmit::SetMethodImplFlags-Methode
Legt die Metadatensignatur der geerbten Methodenimplementierung fest oder aktualisiert sie, auf die vom angegebenen Token verwiesen wird.  
  
## <a name="syntax"></a>Syntax  
  
```cpp  
HRESULT SetMethodImplFlags (
    [in]  mdMethodDef   md,
    [in]  DWORD         dwImplFlags
);  
```  
  
## <a name="parameters"></a>Parameter  
 `md`  
 [in] Das Token für die zu ändernde Methode.  
  
 `dwImplFlags`  
 [in] Eine Kombination der Werte der [CorMethodImpl-Enumeration,](../../../../docs/framework/unmanaged-api/metadata/cormethodimpl-enumeration.md) die die Methodenimplementierungsfeatures angibt.
