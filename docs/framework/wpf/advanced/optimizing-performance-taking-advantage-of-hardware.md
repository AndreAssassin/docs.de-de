 <span data-ttu-id="16191-121">Um dies zu erreichen [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] , stellt die Funktionalität bereit, um die Grafik Funktion eines Systems zur Laufzeit zu bestimmen.</span><span class="sxs-lookup"><span data-stu-id="16191-121">To achieve this, [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)] provides functionality to determine the graphics capability of a system at runtime.</span></span> <span data-ttu-id="16191-122">Die Grafik Funktion wird bestimmt, indem die Grafikkarte als eine von drei renderingfunktionsstufen kategorisiert wird.</span><span class="sxs-lookup"><span data-stu-id="16191-122">Graphics capability is determined by categorizing the video card as one of three rendering capability tiers.</span></span> [!INCLUDE[TLA2#tla_winclient](../../../../includes/tla2sharptla-winclient-md.md)]<span data-ttu-id="16191-123">macht eine API verfügbar, die einer Anwendung das Abfragen der renderingfunktionsebene ermöglicht.</span><span class="sxs-lookup"><span data-stu-id="16191-123">exposes an API that allows an application to query the rendering capability tier.</span></span> <span data-ttu-id="16191-124">Abhängig von der von der Hardware unterstützten Renderingebene kann die Anwendung dann unterschiedliche Codepfade zur Laufzeit verwenden.</span><span class="sxs-lookup"><span data-stu-id="16191-124">Your application can then take different code paths at run time depending on the rendering tier supported by the hardware.</span></span>

- <span data-ttu-id="16191-137">**Renderingebene 1** Beschleunigung der Teil Grafikhardware.</span><span class="sxs-lookup"><span data-stu-id="16191-137">**Rendering Tier 1** Partial graphics hardware acceleration.</span></span> <span data-ttu-id="16191-138">Die DirectX-Versions Ebene ist größer als oder gleich Version 7,0 und **kleiner** als Version 9,0.</span><span class="sxs-lookup"><span data-stu-id="16191-138">The DirectX version level is greater than or equal to version 7.0, and **lesser** than version 9.0.</span></span>
