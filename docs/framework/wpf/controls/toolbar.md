# <a name="toolbar"></a>ToolBar
Die <xref:System.Windows.Controls.ToolBar> Steuerelement ist ein Container für eine Gruppe von Befehlen oder Steuerelementen, die in der Regel in ihrer Funktion verknüpft sind.
