# <a name="wrappanel"></a>WrapPanel
Die <xref:System.Windows.Controls.WrapPanel> Element positioniert untergeordnete Elemente nacheinander von links nach rechts und umbricht den Inhalt in die nächste Zeile am Rand des enthaltenden Felds.
