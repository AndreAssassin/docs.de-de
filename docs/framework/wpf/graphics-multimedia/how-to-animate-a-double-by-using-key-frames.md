2. <span data-ttu-id="9980a-110">Am Ende der vierten Sekunde wird eine Instanz der dem <xref:System.Windows.Media.Animation.DiscreteDoubleKeyFrame> Klasse, um das Rechteck sofort an die nächste Position zu bewegen.</span><span class="sxs-lookup"><span data-stu-id="9980a-110">At the end of the fourth second, uses an instance of the <xref:System.Windows.Media.Animation.DiscreteDoubleKeyFrame> class to suddenly move the rectangle to the next position.</span></span> <span data-ttu-id="9980a-111">Diskrete Keyframes wie <xref:System.Windows.Media.Animation.DiscreteDoubleKeyFrame> ermöglichen abrupte Sprünge zwischen Werten.</span><span class="sxs-lookup"><span data-stu-id="9980a-111">Discrete key frames like <xref:System.Windows.Media.Animation.DiscreteDoubleKeyFrame> create sudden jumps between values.</span></span> <span data-ttu-id="9980a-112">In diesem Beispiel befindet sich das Rechteck an der Startposition und wird anschließend plötzlich an der Position 500 angezeigt.</span><span class="sxs-lookup"><span data-stu-id="9980a-112">In this example, the rectangle is at the starting position and then suddenly appears at the 500 position.</span></span>
