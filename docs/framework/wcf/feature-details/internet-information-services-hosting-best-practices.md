## <a name="optimizing-performance-in-middle-tier-scenarios"></a><span data-ttu-id="a0dc4-122">Optimieren der Leistung in Szenarien der mittleren Ebene</span><span class="sxs-lookup"><span data-stu-id="a0dc4-122">Optimizing Performance in Middle-Tier Scenarios</span></span>
