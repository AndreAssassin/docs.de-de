# <a name="peer-nodes"></a>Peerknoten
Ein PeerNode ist ein einzelner Endpunkt in einem Peerkanalnetz. Eine Anwendung kann auch über mehrere Peerknoten verfügen, die an anderen Netzen teilnehmen. So kann z.&amp;#160;B. eine Peer-to-Peer-Chatanwendung Teil von zwei separaten Netzen sein: Ein Netz zum Chatten mit Schulfreunden und ein anderes zum Chatten mit Arbeitskollegen.
