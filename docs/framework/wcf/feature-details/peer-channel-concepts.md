 [Peer-Gitter](../../../../docs/framework/wcf/feature-details/peer-meshes.md):  Beschreibt die Eigenschaften von per Peerkanal erstellten Netzen.  
  
 [Peer-Knoten](../../../../docs/framework/wcf/feature-details/peer-nodes.md):  Beschreibt Peerknoten.  
  
 [Peerkanalsicherheit](../../../../docs/framework/wcf/feature-details/peer-channel-security.md):  Beschreibt Sicherheitsszenarien und vom Peerkanal bereitgestellte Lösungen.
