## <a name="multiple-base-addresses"></a><span data-ttu-id="0fe22-109">Mehrere Basisadressen</span><span class="sxs-lookup"><span data-stu-id="0fe22-109">Multiple Base Addresses</span></span>  
 <span data-ttu-id="0fe22-110">Dieses Feature steht nur für WCF-Dienste, die unter IIS gehostet werden.</span><span class="sxs-lookup"><span data-stu-id="0fe22-110">This feature is only available to WCF services that are hosted under IIS.</span></span> <span data-ttu-id="0fe22-111">Diese Funktion ist standardmäßig nicht aktiviert.</span><span class="sxs-lookup"><span data-stu-id="0fe22-111">This feature is not enabled by default.</span></span> <span data-ttu-id="0fe22-112">Zur Aktivierung müssen Sie Hinzufügen der `multipleSiteBindingsEnabled` -Attribut auf die <`serviceHostingEnvironment`>-Element in der Web.config Datei, und legen Sie ihn auf `true`, wie im folgenden Beispiel gezeigt.</span><span class="sxs-lookup"><span data-stu-id="0fe22-112">To enable it you must add the `multipleSiteBindingsEnabled` attribute to the <`serviceHostingEnvironment`> element in your Web.config file and set it to `true`, as shown in the following example.</span></span>

  
 <span data-ttu-id="0fe22-113">Wenn Sie einen WCF-Dienst unter IIS hosten, erstellt IIS eine Basisadresse für Sie basierend auf dem URI auf das virtuelle Verzeichnis, das die Anwendung enthält.</span><span class="sxs-lookup"><span data-stu-id="0fe22-113">When hosting a WCF service under IIS, IIS creates one base address for you based on the URI to the virtual directory that contains the application.</span></span> <span data-ttu-id="0fe22-114">Sie können zusätzliche Basisadressen hinzufügen, die das gleiche Protokoll verwenden.
