## <a name="what-is-message-security"></a><span data-ttu-id="19185-105">Was ist Nachrichtensicherheit?</span><span class="sxs-lookup"><span data-stu-id="19185-105">What Is Message Security?</span></span>

<span data-ttu-id="19185-106">Durch Nachrichtensicherheit werden Nachrichten mithilfe der WS-Sicherheitsspezifikation gesichert.</span><span class="sxs-lookup"><span data-stu-id="19185-106">Message security uses the WS-Security specification to secure messages.</span></span> <span data-ttu-id="19185-107">Die WS-Sicherheitsspezifikation enthält Erweiterungen für SOAP-messaging, um sicherzustellen, dass Vertraulichkeit, Integrität und Authentifizierung auf SOAP-Nachrichtenebene (statt der Transportebene) werden.</span><span class="sxs-lookup"><span data-stu-id="19185-107">The WS-Security specification describes enhancements to SOAP messaging to ensure confidentiality, integrity, and authentication at the SOAP message level (instead of the transport level).</span></span>

<span data-ttu-id="19185-108">Der Unterschied zwischen Nachrichtensicherheit und Transportsicherheit besteht kurz gesagt darin, dass bei der Nachrichtensicherheit die Sicherheitsanmeldeinformationen und Ansprüche zusammen mit dem Nachrichtenschutz in jeder Nachricht gekapselt werden (Signierung oder Verschlüsselung).</span><span class="sxs-lookup"><span data-stu-id="19185-108">In brief, message security differs from transport security by encapsulating the security credentials and claims with every message along with any message protection (signing or encryption).</span></span> <span data-ttu-id="19185-109">Die direkte Anwendung der Sicherheit auf die Nachricht durch das Ändern des Nachrichteninhalts ermöglicht gesicherte Nachrichten, die in Bezug auf Sicherheitsaspekte unabhängig sind.</span><span class="sxs-lookup"><span data-stu-id="19185-109">Applying the security directly to the message by modifying its content allows the secured message to be self-containing with respect to the security aspects.</span></span> <span data-ttu-id="19185-110">Hierdurch werden bestimmte Szenarien unterstützt, die bei Verwendung von Transportsicherheit nicht möglich sind.</span><span class="sxs-lookup"><span data-stu-id="19185-110">This enables some scenarios that are not possible when transport security is used.</span></span>
