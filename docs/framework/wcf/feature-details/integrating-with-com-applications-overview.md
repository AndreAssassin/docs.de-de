 <span data-ttu-id="61552-109">Der Dienstmoniker wird verwendet, mit der `GetObject` Funktion, die den nicht verwalteten Entwickler einen stark typisierten, COM-spezifischen Ansatz zum Aufrufen von WCF-Webdiensten bereit.</span><span class="sxs-lookup"><span data-stu-id="61552-109">The service moniker is used with the `GetObject` function to provide the unmanaged developer with a strongly-typed, COM-specific approach for calling WCF Web services.</span></span> <span data-ttu-id="61552-110">Dies erfordert eine lokale, COM-sichtbare Definition der WCF-Web-Dienstvertrag und die Bindung, die verwendet werden soll.</span><span class="sxs-lookup"><span data-stu-id="61552-110">This requires a local, COM-visible definition of the WCF Web service contract and the binding that is to be used.</span></span> <span data-ttu-id="61552-111">Wie andere WCF-Clients muss der Dienstmoniker einen typisierten Kanal für den Dienst erstellen, obwohl diese kanalerstellung transparent für den COM-Programmierer, beim ersten Methodenaufruf auftritt.</span><span class="sxs-lookup"><span data-stu-id="61552-111">Like other WCF clients, the service moniker must construct a typed channel to the service, though this channel construction occurs transparently to the COM programmer on the first method call.</span></span>  
  
 <span data-ttu-id="61552-112">Enthält, die auch andere WCF-Clients, wenn den Moniker verwenden Geben Sie Anwendungen, die Adresse, Bindung und Vertrag für die Kommunikation mit einem Dienst.</span><span class="sxs-lookup"><span data-stu-id="61552-112">In common with other WCF clients, when using the moniker, applications specify the address, binding and contract to communicate with a service.</span></span> <span data-ttu-id="61552-113">Der Vertrag kann mithilfe der folgenden Methoden spezifiziert werden:</span><span class="sxs-lookup"><span data-stu-id="61552-113">The contract can be specified in one of the following ways:</span></span>
