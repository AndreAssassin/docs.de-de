# <a name="wcf-discovery"></a>WCF-Suche
Windows Communication Foundation (WCF) bietet Unterstützung dafür, dass Dienste erkennbar sind, zur Laufzeit auf interoperable Weise mithilfe des WS-Discovery-Protokolls verwendet wird. WCF-Dienste können ihre Verfügbarkeit im Netzwerk mit einer Multicastnachricht oder für einen suchproxyserver ankündigen. Clientanwendungen können ein Netzwerk oder einen Suchproxyserver durchsuchen, um Dienste zu ermitteln, die eine bestimmte Gruppe von Kriterien erfüllen. Die Themen in diesem Abschnitt enthalten eine Übersicht, und das Programmiermodell für diese Funktion wird ausführlich beschrieben.
