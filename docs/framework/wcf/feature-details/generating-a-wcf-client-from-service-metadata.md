- <span data-ttu-id="070c5-107">WS-MetadataExchange (MEX)-Anforderung an die angegebene Adresse</span><span class="sxs-lookup"><span data-stu-id="070c5-107">WS-MetadataExchange (MEX) request to the supplied address.</span></span>

```console

```
