2. <span data-ttu-id="5c44e-117">Klicken Sie im linken Bereich des Fensters **Konsolenstamm** auf **Zertifikate (Lokaler Computer)**.</span><span class="sxs-lookup"><span data-stu-id="5c44e-117">In the **Console Root** window's left pane, click **Certificates (Local Computer)**.</span></span>

4. <span data-ttu-id="5c44e-119">Klicken Sie auf den Ordner **Zertifikate** , um ihn zu erweitern.</span><span class="sxs-lookup"><span data-stu-id="5c44e-119">Click the **Certificates** folder to expand it.</span></span>

6. <span data-ttu-id="5c44e-122">Doppelklicken Sie auf das Zertifikat.</span><span class="sxs-lookup"><span data-stu-id="5c44e-122">Double-click the certificate.</span></span>  
  
7. <span data-ttu-id="5c44e-123">Klicken Sie im Dialogfeld **Zertifikat** auf die Registerkarte **Details** .</span><span class="sxs-lookup"><span data-stu-id="5c44e-123">In the **Certificate** dialog box, click the **Details** tab.</span></span>

9. <span data-ttu-id="5c44e-125">Kopieren Sie die hexadezimalen Zeichen aus dem Feld.</span><span class="sxs-lookup"><span data-stu-id="5c44e-125">Copy the hexadecimal characters from the box.</span></span> <span data-ttu-id="5c44e-126">Wenn dieser Fingerabdruck im Code für `X509FindType`verwendet wird, entfernen Sie die Leerzeichen zwischen den hexadezimalen Zahlen.</span><span class="sxs-lookup"><span data-stu-id="5c44e-126">If this thumbprint is used in code for the `X509FindType`, remove the spaces between the hexadecimal numbers.</span></span> <span data-ttu-id="5c44e-127">Beispielsweise muss der Fingerabdruck "a9 09 50 2d d8 2a e4 14 33 e6 f8 38 86 b0 0d 42 77 a3 2a 7b" im Code als "a909502dd82ae41433e6f83886b00d4277a32a7b" angegeben werden.</span><span class="sxs-lookup"><span data-stu-id="5c44e-127">For example, the thumbprint "a9 09 50 2d d8 2a e4 14 33 e6 f8 38 86 b0 0d 42 77 a3 2a 7b" should be specified as "a909502dd82ae41433e6f83886b00d4277a32a7b" in code.</span></span>
