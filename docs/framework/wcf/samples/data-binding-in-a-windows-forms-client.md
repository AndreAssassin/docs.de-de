 <span data-ttu-id="fbd14-109">Die Datenbindung tritt auf dem Client in der Windows Forms-Anwendung auf.</span><span class="sxs-lookup"><span data-stu-id="fbd14-109">The data binding occurs on the <span data-ttu-id="fbd14-111">Außerdem wird ein Vermittler namens `BindingSource` erstellt.</span><span class="sxs-lookup"><span data-stu-id="fbd14-111">An intermediary named `BindingSource` is also
