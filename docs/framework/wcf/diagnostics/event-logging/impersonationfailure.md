# <a name="impersonationfailure"></a>ImpersonationFailure
ID: 176  
  
 Zunehmen Fehler  
  
 Kategorie SecurityAudit  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis gibt an, dass der Identitätswechsel fehlerhaft war, wobei der Methodenname, die Clientidentität und die Aktivitäts-ID aufgelistet werden.
