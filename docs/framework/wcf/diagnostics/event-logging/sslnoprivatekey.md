# <a name="sslnoprivatekey"></a>SslNoPrivateKey
ID: 154  
  
 Zunehmen Fehler  
  
 Kategorie TransactionBridge  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis zeigt an, dass ein Identitätszertifikat mit dem bestimmten Betreffnamen und Fingerabdruck keinen privaten Schlüssel besitzt. Das Ereignis führt den Prozessnamen und die Prozess-ID auf.
