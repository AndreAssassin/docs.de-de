# <a name="transportauthenticationfailure"></a>TransportAuthenticationFailure
ID: 174  
  
 Schweregrad: Fehler  
  
 Kategorie: SecurityAudit  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis gibt an, dass die Transportauthentifizierung fehlerhaft war, wobei die Clientidentität und die Aktivitäts-ID aufgelistet werden.
