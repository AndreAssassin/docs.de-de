# <a name="coordinatorrecoverylogentrycorrupt"></a>CoordinatorRecoveryLogEntryCorrupt
ID: 139  
  
 Zunehmen Fehler  
  
 Kategorie TransactionBridge  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis gibt an, dass ein Koordinatorwiederherstellungsprotokolleintrag fehlerhaft war und nicht deserialisiert werden konnte. Aufgrund dieses Fehlers können Daten verloren gehen. Das Ereignis führt die TransaktionsID, die Wiederherstellungsdaten (Base64-codiert), die Ausnahme, den Prozessnamen und die Prozess-ID auf.
