Ruft den Instanznamen des Vorgangsleistungsindikators ab.  
  
## <a name="syntax"></a>Syntax  
  
```csharp
string GetOperationCounterInstanceName(  
  string Operation  
);  
```  
  
## <a name="parameters"></a>Parameter  
  
### <a name="operation"></a>Vorgang  
 [in] string
