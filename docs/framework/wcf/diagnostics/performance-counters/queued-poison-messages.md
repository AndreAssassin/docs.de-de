# <a name="queued-poison-messages"></a>Queued Poison Messages
Indikatorname: In der Warteschlange für nicht verarbeitbare Nachrichten.  
  
## <a name="description"></a>Beschreibung  
 Anzahl von Nachrichten, die vom Wartenschlangentransport in diesem Dienst als beschädigt gekennzeichnet werden.  
  
 Finden Sie unter [Behandlung beschädigter Nachrichten](https://go.microsoft.com/fwlink/?LinkID=96233) für Weitere Informationen über die Kennzeichnung von beschädigten Nachrichten sind.
