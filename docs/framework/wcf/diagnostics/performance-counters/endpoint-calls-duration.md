# <a name="endpoint-calls-duration"></a>Endpunkt: Aufruf Dauer
Die durchschnittliche Dauer der Aufrufe dieses Endpunkts.  Die durchschnittliche Dauer wird anhand der folgenden Formel berechnet: (N1-N0)/(D1-D0).
