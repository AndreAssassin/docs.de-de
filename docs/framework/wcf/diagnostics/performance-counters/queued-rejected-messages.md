# <a name="queued-rejected-messages"></a>In die Warteschlange gestellte abgelehnte Meldungen
Indikatorname: In der Warteschlange stehende abgelehnte Meldungen.  
  
## <a name="description"></a>Beschreibung  
 Anzahl von Meldungen, die vom Warteschlangentransport in diesem Dienst abgelehnt werden.  
  
 Finden Sie unter [Behandlung beschädigter Nachrichten](https://go.microsoft.com/fwlink/?LinkID=96233) für Weitere Informationen zur Ablehnung von Meldungen.
