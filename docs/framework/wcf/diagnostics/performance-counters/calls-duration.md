# <a name="calls-duration"></a>Dauer von Aufrufen
Name des Zählers: Dauer von Aufrufen  
  
## <a name="description"></a>Beschreibung  
 Die durchschnittliche Dauer der Aufrufe dieses Vorgangs. Die durchschnittliche Dauer wird basierend auf dieser Gleichung berechnet: (N1-N0)/(D1-D0).  
  
> [!WARNING]
> Bei Verwendung in einem asynchronen WCF-Dienst wird der Aufruf Dauer-Leistungswert immer-1 zurückgegeben.
