# <a name="3421---websocketconnectionaborted"></a>3421 - WebSocketConnectionAborted
## <a name="properties"></a>Eigenschaften  
  
|||  
|-|-|  
|ID|3421|  
|Schlüsselwörter|HTTP|  
|Ebene|Fehler|  
|Kanal|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn die WebSocket-Verbindung abgebrochen wird.  
  
## <a name="message"></a>Meldung  
 Die Verbindung mit WebSocketId: %1 wurde abgebrochen.
