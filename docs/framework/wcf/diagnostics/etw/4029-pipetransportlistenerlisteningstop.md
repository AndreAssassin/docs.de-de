# <a name="4029---pipetransportlistenerlisteningstop"></a>4029 - PipeTransportListenerListeningStop
## <a name="properties"></a>Eigenschaften  
  
|||  
|-|-|  
|ID|4029|  
|Schlüsselwörter|ActivationServices|  
|Ebene|Ausführlich|  
|Kanal|Microsoft-Windows-Application Server-Applications/Analytic|  
  
## <a name="description"></a>Beschreibung  
 Dieses Ereignis wird ausgegeben, wenn der Pipe-Transportlistener nicht mehr empfangsbereit ist.  
  
## <a name="message"></a>Meldung  
 Der Pipe-Transportlistener ist nicht mehr empfangsbereit.
