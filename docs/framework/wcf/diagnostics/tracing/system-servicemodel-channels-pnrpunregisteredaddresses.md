# <a name="systemservicemodelchannelspnrpunregisteredaddresses"></a>System.ServiceModel.Channels.PnrpUnregisteredAddresses
Die Registrierung der Adressen des lokalen Knotens wurde vom Pnrp-Resolver aufgehoben.  
  
## <a name="description"></a>Beschreibung  
 Diese Ablaufverfolgung tritt auf, nachdem der PNRP-Dienst erfolgreich Adressen des lokalen Knotens hebt die Registrierung.
