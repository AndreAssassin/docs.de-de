# <a name="debugging-on-the-client"></a>Debuggen auf dem Client
Um Benutzern das Schreiben von Clientanwendungen für den WCF-Dienst zu vereinfachen, können Sie Hinzufügen der [ \<ServiceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) -Dienstverhalten in die Konfigurationsdatei des Diensts. Dieses Verhalten kann zum Veröffentlichen von Hilfeseiten und für die Rückgabe von verwalteten Ausnahmeinformationen in den Details der SOAP-Fehler, die an den Client zurückgegeben werden, verwendet werden.
