## <a name="features-of-the-common-language-runtime"></a><span data-ttu-id="111ff-129">Features der Common Language Runtime</span><span class="sxs-lookup"><span data-stu-id="111ff-129">Features of the common language runtime</span></span>

<span data-ttu-id="111ff-130">Die Common Language Runtime verwaltet Speicher, Thread- und Codeausführung, Überprüfung der Codesicherheit, Kompilierung und andere Systemdienste.</span><span class="sxs-lookup"><span data-stu-id="111ff-130">The common language runtime manages memory, thread execution, code execution, code safety verification, compilation, and other system services.</span></span> <span data-ttu-id="111ff-131">Diese Features sind in den verwalteten Code integriert, der durch die Common Language Runtime ausgeführt wird.</span><span class="sxs-lookup"><span data-stu-id="111ff-131">These features are intrinsic to the managed code that runs on the common language runtime.</span></span>

<span data-ttu-id="111ff-132">Aus Sicherheitsgründen werden verwalteten Komponenten unterschiedliche Vertrauensstufen zugewiesen, die von mehreren Faktoren abhängen, zu denen auch der Ursprung (z.
