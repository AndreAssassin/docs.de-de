<span data-ttu-id="1b884-124">Ab .NET Framework, Version 2.0, generiert der JIT-Compiler immer Verfolgungsinformationen unabhängig vom Wert für `GenerateTrackingInfo`, aber die `AllowOptimize` Wert immer noch wirkt sich ein.</span><span class="sxs-lookup"><span data-stu-id="1b884-124">Starting with the .NET Framework version 2.0, the JIT compiler always generates tracking information regardless of the value for `GenerateTrackingInfo`;
