# <a name="datamemberfieldeditor-class"></a>DataMemberFieldEditor-Klasse

Stellt eine Benutzeroberfläche zum Bearbeiten von Eigenschaften von datengebundenen Objekten bereit, indem alle Eigenschaften des Objekts aufgelistet werden, aus denen `DataSource` die Wertquelle ausgewählt werden soll.  
  
## <a name="syntax"></a>Syntax
  
```csharp
internal class DataMemberFieldEditor : UITypeEditor
```

> [!WARNING]
> Die `DataMemberFieldEditor` Klasse ist intern und soll nicht direkt in Ihrem Code verwendet werden.
>
> Microsoft unterstützt die Verwendung dieser Klasse in einer Produktionsanwendung unter keinen Umständen.
