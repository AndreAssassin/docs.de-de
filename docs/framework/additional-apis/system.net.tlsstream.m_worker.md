# <a name="tlsstreamm_worker-field"></a>Tlsstream. m_Worker-Feld

Stellt den Status des SSL-Streams dar.

## <a name="syntax"></a>Syntax

```csharp
private SslState m_Worker;
```

## <a name="field-value"></a>Feldwert

`System.Net.Security.SslState`  
Der Status des SSL-Streams.

## <a name="remarks"></a>Hinweise

> [!WARNING]
> Das `TlsStream.m_Worker` Feld ist privat und sollte nicht direkt im Code verwendet werden.
>
> Microsoft unterstützt die Verwendung dieses Felds in einer Produktionsanwendung unter keinen Umständen.
