- [Breaking Changes in Roslyn 1.0 (VS2015) im Vergleich zum nativen C#-Compiler (VS2013 oder früher).](https://github.com/dotnet/roslyn/blob/master/docs/compilers/CSharp/Compiler%20Breaking%20Changes%20-%20VS2015.md)
