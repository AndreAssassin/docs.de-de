# <a name="compiler-error-cs1942"></a><span data-ttu-id="8a354-102">Compilerfehler CS1942</span><span class="sxs-lookup"><span data-stu-id="8a354-102">Compiler Error CS1942</span></span>

<span data-ttu-id="8a354-103">Der Typ des Ausdrucks in der Klausel „Klausel“ ist falsch.</span><span class="sxs-lookup"><span data-stu-id="8a354-103">The type of the expression in the 'clause' clause is incorrect.</span></span> <span data-ttu-id="8a354-104">Fehler beim Typrückschluss im Aufruf von „Methode“.</span><span class="sxs-lookup"><span data-stu-id="8a354-104">Type inference failed in the call to 'method'.</span></span>  
  
<span data-ttu-id="8a354-105">Dieser Fehler wird in der Regel generiert, wenn der Bereichsvariable ein falscher expliziter Typ zugewiesen wurde.</span><span class="sxs-lookup"><span data-stu-id="8a354-105">This error is typically generated when the range variable has been given an incorrect explicit type.</span></span>  

## <a name="to-correct-this-error"></a><span data-ttu-id="8a354-106">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="8a354-106">To correct this error</span></span>

1. <span data-ttu-id="8a354-107">Ist die Bereichsvariable explizit typisiert, vergewissern Sie sich, dass der Typ dem Typ der Elemente in der Auflistung, die sie durchläuft, entweder entspricht oder implizit konvertiert werden kann.</span><span class="sxs-lookup"><span data-stu-id="8a354-107">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="8a354-108">Ist der Bereichsvariable das Schlüsselwort `var` vorangestellt, entfernen Sie `var`.</span><span class="sxs-lookup"><span data-stu-id="8a354-108">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8a354-109">Beispiel</span><span class="sxs-lookup"><span data-stu-id="8a354-109">Example</span></span>

<span data-ttu-id="8a354-110">Durch den folgenden Code wird der Fehler CS1942 ausgelöst:</span><span class="sxs-lookup"><span data-stu-id="8a354-110">The following code generates CS1942:</span></span>  

```csharp
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```

<span data-ttu-id="8a354-111">CS1942 ist CS1949 zugeordnet, da die Verwendung von `var` mit einer Bereichsvariable dazu führt, dass der zugrundeliegende Vorgang `Cast<T>` fehlschlägt, weil `var` kein Typ ist.</span><span class="sxs-lookup"><span data-stu-id="8a354-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
