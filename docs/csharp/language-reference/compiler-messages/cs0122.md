# <a name="compiler-error-cs0122"></a>Compilerfehler CS0122
Der Zugriff auf „member“ ist aufgrund der Sicherheitsebene nicht möglich.  
  
 Die [Zugriffsmodifizierer](../keywords/modifiers.md) für einen Klassenmember verhindert den Zugriff auf den Member. Weitere Informationen finden Sie unter [Zugriffsmodifizierer](../../programming-guide/classes-and-structs/access-modifiers.md).  
  
 Eine Ursache (die nicht im folgenden Beispiel gezeigt wird) kann das Auslassen des Compilerflags **/out** für das Ziel einer Friend-Assembly sein. Weitere Informationen finden Sie unter [Friend-Assemblys](../../../standard/assembly/friend-assemblies.md) und unter [/out (C# Compiler Options) (/out (C#-Compileroptionen))](../compiler-options/out-compiler-option.md).  
  
## <a name="example"></a>Beispiel  
 Im folgenden Beispiel wird CS0122 generiert:  
  
```csharp  
// CS0122.cs  
public class MyClass  
{  
    // Make public to resolve CS0122  
    void MyMethod()  
    {  
    }  
}  
  
public class MyClass2  
{  
    public static int Main()  
    {  
        MyClass a = new MyClass();  
        // MyMethod is private  
        a.MyMethod();   // CS0122  
        return 0;  
   }  
}  
```
