## <a name="example"></a><span data-ttu-id="5e41d-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="5e41d-108">Example</span></span>

 <span data-ttu-id="5e41d-109">Im folgenden Beispiel wird CS0122 generiert:</span><span class="sxs-lookup"><span data-stu-id="5e41d-109">The following sample generates CS0122:</span></span>

```csharp
// CS0122.cs
public class MyClass
{
    // Make public to resolve CS0122.
    void MyMethod()
    {
    }
}

public class MyClass2
{
    public static int Main()
    {  
        var a = new MyClass();  
        // MyMethod is private.
        a.MyMethod();   // CS0122
        return 0;
   }
}
```
