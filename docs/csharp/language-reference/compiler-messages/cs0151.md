## <a name="example-of-ambiguous-conversion"></a><span data-ttu-id="f8c59-106">Beispiel für eine mehrdeutige Konvertierung</span><span class="sxs-lookup"><span data-stu-id="f8c59-106">Example of ambiguous conversion</span></span>

<span data-ttu-id="f8c59-107">Dieser Fehler kann auftreten, wenn keine Konvertierung vorgenommen wurde, oder wenn die verfügbaren impliziten Konvertierungen zu einer mehrdeutigen Situation führen.</span><span class="sxs-lookup"><span data-stu-id="f8c59-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="f8c59-108">Im folgenden Beispiel wird der Fehler CS0151 generiert:</span><span class="sxs-lookup"><span data-stu-id="f8c59-108">The following example generates CS0151:</span></span>

```csharp
public class MyClass
{
   public static implicit operator int (MyClass aa)
   {
      return 0;
   }

   public static implicit operator long (MyClass aa)
   {
      return 0;
   }

   public static void Main()
   {
      MyClass a = new MyClass();

      // Compiler cannot choose between int and long.
      switch (a)   // CS0151
      // try the following line instead
      // switch ((int)a)
      {
         case 1:
            break;
      }
   }
}
```
