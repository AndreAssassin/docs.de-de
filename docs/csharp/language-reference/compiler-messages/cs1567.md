# <a name="compiler-error-cs1567"></a><span data-ttu-id="3b351-102">Compilerfehler CS1567</span><span class="sxs-lookup"><span data-stu-id="3b351-102">Compiler Error CS1567</span></span>
<span data-ttu-id="3b351-103">Fehler beim Generieren der Win32-Ressource: „Datei“.</span><span class="sxs-lookup"><span data-stu-id="3b351-103">Error generating Win32 resource: 'file'</span></span>  
  
 <span data-ttu-id="3b351-104">Die Kompilierung hat entweder die Compileroption [/win32icon](../compiler-options/win32icon-compiler-option.md) verwendet, oder sie hat [/win32res](../compiler-options/win32res-compiler-option.md) nicht verwendet. Durch die Verwendung dieser Optionen generiert der Compiler normalerweise eine Datei mit den Ressourceninformationen.  <span data-ttu-id="3b351-105">Wenn Sie das Dateierstellungsproblem nicht beheben können, haben Sie die Möglichkeit, [/win32res](../compiler-options/win32res-compiler-option.md) zu verwenden, wobei keine Datei mit Ressourceninformationen generiert wird.</span><span class="sxs-lookup"><span data-stu-id="3b351-105">If you are unable to resolve the file-generation problem, you could use [/win32res](../compiler-options/win32res-compiler-option.md), which does not generate a file that contains resource information.</span></span>
