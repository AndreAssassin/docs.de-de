## <a name="example"></a><span data-ttu-id="caf53-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="caf53-106">Example</span></span>  
 <span data-ttu-id="caf53-107">In folgendem Beispiel wird der Fehler CS0050 generiert, da für `MyClass` kein Zugriffsmodifizierer angegeben wurde, und der Standardwert deshalb `private` lautet.</span><span class="sxs-lookup"><span data-stu-id="caf53-107">The following sample generates CS0050 because no accessibility modifier is supplied for `MyClass` and its accessibility therefore defaults to `private`.</span></span>

```csharp  
// try the following line instead  
// public class MyClass   
{  
}  
  
public class MyClass2  
{  
    public static MyClass MyMethod()   // CS0050  
    {  
        return new MyClass();  
    }  
  
    public static void Main() { }  
}  
```
