## <a name="example"></a><span data-ttu-id="5cf16-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="5cf16-108">Example</span></span>  
 <span data-ttu-id="5cf16-109">Dieses Codebeispiel besteht aus einer CPP-Datei, die in eine DLL kompiliert wird, sowie aus einer CS-Datei, die diese DLL verwendet.</span><span class="sxs-lookup"><span data-stu-id="5cf16-109">This code sample consists of a .cpp file, which compiles to a .dll, and a .cs file, which uses that .dll.</span></span> <span data-ttu-id="5cf16-110">Der folgende Code ist für die DLL-Datei bestimmt und definiert eine Eigenschaft, auf die der Code in der CS-Datei zugreift.</span><span class="sxs-lookup"><span data-stu-id="5cf16-110">The following code is for the .dll file and defines a property to be accessed by the code in the .cs file.</span></span>  
  
```cpp  
// CPP1546.cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
