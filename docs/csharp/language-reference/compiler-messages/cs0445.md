# <a name="compiler-error-cs0445"></a><span data-ttu-id="637b7-102">Compilerfehler CS0445</span><span class="sxs-lookup"><span data-stu-id="637b7-102">Compiler Error CS0445</span></span>
<span data-ttu-id="637b7-103">Das Ergebnis einer Unboxing-Konvertierung kann nicht geändert werden.</span><span class="sxs-lookup"><span data-stu-id="637b7-103">Cannot modify the result of an unboxing conversion</span></span>  
  
 <span data-ttu-id="637b7-104">Das Ergebnis einer Unboxing-Konvertierung ist eine temporäre Variable.</span><span class="sxs-lookup"><span data-stu-id="637b7-104">The result of an unboxing conversion is a temporary variable.</span></span> <span data-ttu-id="637b7-105">Der Compiler verhindert die Änderung solcher Variablen, da jede Änderung behoben werden würde, wenn die temporäre Variable verschwindet.</span><span class="sxs-lookup"><span data-stu-id="637b7-105">The compiler prevents you from modifying such variables because any modification would go away when the temporary variable goes away.</span></span> <span data-ttu-id="637b7-106">Um dieses Problem zu beheben, deklarieren Sie eine neue Werttypvariable zum Speichern des intermediären Ausdrucks, und weisen Sie das Ergebnis der Unboxing-Konvertierung dieser Variablen zu.</span><span class="sxs-lookup"><span data-stu-id="637b7-106">To fix this, declare a new value-type variable to store the intermediate expression, and assign the result of the unboxing conversion to that variable.</span></span>  
  
 <span data-ttu-id="637b7-107">Mit dem folgenden Code wird CS0455 generiert.</span><span class="sxs-lookup"><span data-stu-id="637b7-107">The following code generates CS0455.</span></span>  
  
```csharp  
// CS0445.CS  
class UnboxingTest  
{  
    public static void Main()  
    {  
        Point p;  
        p.x = 1;  
        p.y = 2;  
        object obj = p;  
        // The following line generates CS0445, because the result  
        // of unboxing obj is a temporary variable.  
        ((Point)obj).x = 2;  
  
        // The following lines resolve the error.  
  
        // Store the result of the unboxing conversion in p2.  
        Point p2;
        p2 = (Point)obj;  
        // Then you can modify the unboxed value.  
        p2.x = 2;  
    }  
}  
  
struct Point  
{  
    public int x, y;  
}  
```
