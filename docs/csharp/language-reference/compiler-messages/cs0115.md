## <a name="example"></a><span data-ttu-id="0a2ba-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="0a2ba-106">Example</span></span>  
 <span data-ttu-id="0a2ba-107">Im folgenden Beispiel wird CS0115 generiert:</span><span class="sxs-lookup"><span data-stu-id="0a2ba-107">The following sample generates CS0115.</span></span> <span data-ttu-id="0a2ba-108">Sie haben zwei Möglichkeiten, CS0115 zu beheben:</span><span class="sxs-lookup"><span data-stu-id="0a2ba-108">You can resolve CS0115 in one of two ways:</span></span>  
  
- <span data-ttu-id="0a2ba-109">Entfernen Sie das Schlüsselwort `override` aus der Methode in `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="0a2ba-109">Remove the `override` keyword from the method in `MyClass2`.</span></span>  
  
- <span data-ttu-id="0a2ba-110">Verwenden Sie `MyClass1` als Basisklasse für `MyClass2`.</span><span class="sxs-lookup"><span data-stu-id="0a2ba-110">Use `MyClass1` as a base class for `MyClass2`.</span></span>  
  
```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
