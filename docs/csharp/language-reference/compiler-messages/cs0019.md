## <a name="example"></a><span data-ttu-id="4a4bf-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="4a4bf-108">Example</span></span>

 <span data-ttu-id="4a4bf-109">In folgendem Beispiel wird der Fehler CS0019 an zwei Stellen erzeugt, weil [bool](../../../csharp/language-reference/keywords/bool.md) in C# nicht in [int](../../../csharp/language-reference/keywords/int.md) konvertiert werden kann. CS0019 wird auch erzeugt, wenn der Substraktionsoperator auf eine Zeichenfolge angewendet wird.</span><span class="sxs-lookup"><span data-stu-id="4a4bf-109">In the following example, CS0019 is generated in two places because [bool](../../../csharp/language-reference/keywords/bool.md) in C# is not convertible to [int](../../../csharp/language-reference/keywords/int.md). CS0019 also is generated when the subtraction operator is applied to a string.</span></span> <span data-ttu-id="4a4bf-110">Der Additionsoperator (+) kann mit Zeichenfolgenoperanden verwendet werden, da dieser Operator von der `String`-Klasse überladen wird, um eine Zeichenfolgenverkettung durchzuführen.</span><span class="sxs-lookup"><span data-stu-id="4a4bf-110">The addition operator (+) can be used with string operands because that operator is overloaded by the `String` class to perform string concatenation.</span></span>  

```csharp
static void Main()  
{  
    bool result = true;  
    if (result > 0) //CS0019  
    {  
        // Do something.  
    }  
  
    int i = 1;  
    // You cannot compare an integer and a boolean value.  
    if (i == true) //CS0019  
    {  
        //Do something...  
    }  
  
    // The following use of == causes no error. It is the comparison of
    // an integer and a boolean value that causes the error in the
    // previous if statement.  
    if (result == true)  
    {  
        //Do something...  
    }  
  
    string s = "Just try to subtract me.";  
    float f = 100 - s; // CS0019  
}  
```
