# <a name="compiler-error-cs1936"></a><span data-ttu-id="a5e95-102">Compilerfehler CS1936</span><span class="sxs-lookup"><span data-stu-id="a5e95-102">Compiler Error CS1936</span></span>

<span data-ttu-id="a5e95-103">Es konnte keine Implementierung des Abfragemusters für den Quelltyp „type“ gefunden werden.</span><span class="sxs-lookup"><span data-stu-id="a5e95-103">Could not find an implementation of the query pattern for source type 'type'.</span></span>  <span data-ttu-id="a5e95-104">"Methode" wurde nicht gefunden.</span><span class="sxs-lookup"><span data-stu-id="a5e95-104">'method' not found.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="a5e95-107">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="a5e95-107">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="a5e95-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="a5e95-110">Example</span></span>

<span data-ttu-id="a5e95-111">Im folgenden Beispiel wird CS1936 erzeugt:</span><span class="sxs-lookup"><span data-stu-id="a5e95-111">The following example produces CS1936:</span></span>  

```csharp
// cs1936.cs  
using System.Collections;  
using System.Linq;  
class Test  
{  
    static int Main()  
    {  
        object obj;  
        IEnumerable e = from x in obj // CS1936  
                        select x;  
        return 0;  
    }  
}  
```

<span data-ttu-id="a5e95-112">Dieser Fehler tritt normalerweise auf, wenn Sie versehentlich versuchen, ein Objekt eines Typs abzufragen, statt einer Sammlung dieser Objekte.</span><span class="sxs-lookup"><span data-stu-id="a5e95-112">This error typically occurs when you accidentally try to query an object of some type instead of a collection of those objects.</span></span>
