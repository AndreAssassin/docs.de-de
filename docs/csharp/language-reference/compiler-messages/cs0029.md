<span data-ttu-id="7e967-103">Kann den Typ „type“ nicht implizit in „type“ konvertieren</span><span class="sxs-lookup"><span data-stu-id="7e967-103">Cannot implicitly convert type 'type' to 'type'</span></span>

 <span data-ttu-id="7e967-104">Der Compiler erfordert eine explizite Konvertierung.</span><span class="sxs-lookup"><span data-stu-id="7e967-104">The compiler requires an explicit conversion.</span></span> <span data-ttu-id="7e967-105">Sie müssen z.B. einen r-Wert in den gleichen Typ wie einen l-Wert umwandeln.</span><span class="sxs-lookup"><span data-stu-id="7e967-105">For example, you may need to cast an r-value to be the same type as an l-value.</span></span> <span data-ttu-id="7e967-106">Oder Sie müssen zur Unterstützung bestimmter Operatorüberladungen Konvertierungsroutinen angeben.</span><span class="sxs-lookup"><span data-stu-id="7e967-106">Or, you must provide conversion routines to support certain operator overloads.</span></span>

 <span data-ttu-id="7e967-107">Sie müssen Konvertierungen durchführen, wenn Sie eine Variablen eines Typs einer Variablen eines andere Typs zuweisen.</span><span class="sxs-lookup"><span data-stu-id="7e967-107">Conversions must occur when assigning a variable of one type to a variable of a different type.</span></span> <span data-ttu-id="7e967-108">Wenn Sie eine Zuweisung zwischen Variablen verschiedener Typen durchführen, muss der Compiler den Typ der rechten Seite des Zuweisungsoperator in den Typ der linken Seite des Zuweisungsoperators konvertieren.</span><span class="sxs-lookup"><span data-stu-id="7e967-108">When making an assignment between variables of different types, the compiler must convert the type on the right-hand side of the assignment operator to the type on the left-hand side of the assignment operator.</span></span> <span data-ttu-id="7e967-109">Nehmen Sie z.B. den folgenden Code:</span><span class="sxs-lookup"><span data-stu-id="7e967-109">Take the following the code:</span></span>


 <span data-ttu-id="7e967-110">`i = lng;` führt eine Zuweisung durch, aber die Datentypen der Variablen auf der linken und der rechten Seite des Zuweisungsoperators stimmen nicht überein.</span><span class="sxs-lookup"><span data-stu-id="7e967-110">`i = lng;` makes an assignment, but the data types of the variables on the left and right-hand side of the assignment operator don't match.</span></span> <span data-ttu-id="7e967-111">Vor der Zuweisung konvertiert der Compiler implizit die Variable `lng`, die vom Typ „long“ ist, in den Typ „int“.  <span data-ttu-id="7e967-113">Eine einschränkende Konvertierung besteht, wenn Sie in einen Datentyp konvertieren, der weniger Speicherplatz im Arbeitsspeicher in Anspruch nimmt als der Datentyp, der konvertiert wird.</span><span class="sxs-lookup"><span data-stu-id="7e967-113">A narrowing conversion exists when converting to a data type that occupies less storage space in memory than the data type we are converting from.</span></span> <span data-ttu-id="7e967-114">Wenn Sie z.B. den Typ „long“ in den Typ „int“ konvertieren, wird dies als einschränkende Konvertierung angesehen.</span><span class="sxs-lookup"><span data-stu-id="7e967-114">For example, converting a long to an int would be considered a narrowing conversion.</span></span> <span data-ttu-id="7e967-115">Der Typ „long“ nimmt 8 Byte Arbeitsspeicher in Anspruch, während ein Typ „int“ nur 4 Byte in Anspruch nimmt.</span><span class="sxs-lookup"><span data-stu-id="7e967-115">A long occupies 8 bytes of memory while an int occupies 4 bytes.</span></span> <span data-ttu-id="7e967-116">Um zu sehen, wie es zu Datenverlust kommen kann, schauen Sie sich das folgende Beispiel an:</span><span class="sxs-lookup"><span data-stu-id="7e967-116">To see how data loss can occur, consider the following sample:</span></span>

```csharp
int i = 50;
long lng = 3147483647;

```

 <span data-ttu-id="7e967-117">Die Variable `lng` enthält nun einen Wert, der nicht in der Variable `i` gespeichert werden kann, da er zu groß ist.</span><span class="sxs-lookup"><span data-stu-id="7e967-117">The variable `lng` now contains a value that cannot be stored in the variable `i` because it is too large.</span></span> <span data-ttu-id="7e967-118">Wenn Sie nun diesen Wert in den Typ „int“ konvertieren würden, würden Sie einige Ihrer Daten verlieren, und der konvertierte Wert wäre nicht der gleiche wie vor der Konvertierung.</span><span class="sxs-lookup"><span data-stu-id="7e967-118">If we were to convert this value to an int type we would be losing some of our data and the converted value would not be the same as the value before the conversion.</span></span>

 <span data-ttu-id="7e967-119">Eine erweiternde Konvertierung ist das Gegenteil einer einschränkenden Konvertierung.</span><span class="sxs-lookup"><span data-stu-id="7e967-119">A widening conversion would be the opposite of a narrowing conversion.</span></span> <span data-ttu-id="7e967-120">Bei erweiternden Konvertierungen wird eine Konvertierung in einen Datentyp durchgeführt, der mehr Platz in Anspruch nimmt als der Typ, der konvertiert wird.</span><span class="sxs-lookup"><span data-stu-id="7e967-120">With widening conversions, we are converting to a data type that occupies more storage space in memory than the data type we are converting from.</span></span> <span data-ttu-id="7e967-121">Dies ist ein Beispiel für eine erweiternde Konvertierung:</span><span class="sxs-lookup"><span data-stu-id="7e967-121">Here is an example of a widening conversion:</span></span>

```csharp

```

 <span data-ttu-id="7e967-122">Beachten Sie den Unterschied zwischen diesem und dem ersten Codebeispiel.</span><span class="sxs-lookup"><span data-stu-id="7e967-122">Notice the difference between this code sample and the first.</span></span> <span data-ttu-id="7e967-123">Dieses Mal befindet sich die Variable `lng` auf der rechten Seite des Zuweisungsoperators, sodass Sie das Ziel der Zuweisung darstellt.</span><span class="sxs-lookup"><span data-stu-id="7e967-123">This time the variable `lng` is on the left-hand side of the assignment operator, so it is the target of our assignment.</span></span> <span data-ttu-id="7e967-124">Vor der Zuweisung muss der Compiler die Variable `i` des Typ „int“ implizit in den Typ „long“ konvertieren.</span><span class="sxs-lookup"><span data-stu-id="7e967-124">Before the assignment can be made, the compiler must implicitly convert the variable `i`, which is of type int, to type long.</span></span> <span data-ttu-id="7e967-125">Dabei handelt es sich um eine erweiternde Konvertierung, da Sie einen Typ, der 4 Byte in Anspruch nimmt (int), in einen Typ, der 8 Byte in Anspruch nimmt (long), konvertieren.</span><span class="sxs-lookup"><span data-stu-id="7e967-125">This is a widening conversion since we are converting from a type that occupies 4 bytes of memory (an int) to a type that occupies 8 bytes of memory (a long).</span></span> <span data-ttu-id="7e967-126">Implizite erweiternde Konvertierungen sind zulässig, da es dabei nicht zu Datenverlusten kommen kann.</span><span class="sxs-lookup"><span data-stu-id="7e967-126">Implicit widening conversions are allowed because there is no potential loss of data.</span></span> <span data-ttu-id="7e967-127">Jeder Wert, der in einem int-Typ gespeichert werden kann, kann auch in einem long-Typ gespeichert werden.</span><span class="sxs-lookup"><span data-stu-id="7e967-127">Any value that can be stored in an int can also be stored in a long.</span></span>

 <span data-ttu-id="7e967-128">Sie wissen, dass implizite einschränkende Konvertierungen nicht zulässig sind.
