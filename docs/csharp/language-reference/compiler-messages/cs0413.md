<span data-ttu-id="8d8dd-104">Dieser Fehler tritt auf, wenn ein generischer Typ den [as](../../language-reference/operators/type-testing-and-conversion-operators.md#as-operator)-Operator verwendet, aber keine Klassentypeinschränkung aufweist.</span><span class="sxs-lookup"><span data-stu-id="8d8dd-104">This error occurs if a generic type uses the [as](../../language-reference/operators/type-testing-and-conversion-operators.md#as-operator) operator, but that generic type does not have a class type constraint.</span></span> <span data-ttu-id="8d8dd-105">Der `as`-Operator ist nur bei Verweistypen und Nullable-Werttypen zulässig. Daher muss der Typparameter eingeschränkt werden, um zu garantieren, dass er kein Werttyp ist.</span><span class="sxs-lookup"><span data-stu-id="8d8dd-105">The `as` operator is only allowed with reference and nullable value types, so the type parameter must be constrained to guarantee that it is not a value type.</span></span> <span data-ttu-id="8d8dd-106">Um diesen Fehler zu vermeiden, verwenden Sie eine Klassentypeinschränkung oder eine Verweistypeinschränkung.</span><span class="sxs-lookup"><span data-stu-id="8d8dd-106">To avoid this error, use a class type constraint or a reference type constraint.</span></span>
  
<span data-ttu-id="8d8dd-107">Dies liegt daran, dass der `as`-Operator `null` zurückgeben kann, was kein zulässiger Wert für einen Werttyp ist. ## <a name="example"></a><span data-ttu-id="8d8dd-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="8d8dd-108">Example</span></span>

<span data-ttu-id="8d8dd-109">Im folgenden Beispiel wird CS0413 generiert:</span><span class="sxs-lookup"><span data-stu-id="8d8dd-109">The following sample generates CS0413.</span></span>  
  
```csharp  
// CS0413.cs  
// compile with: /target:library  
class A {}  
class B : A {}  
  
class CMain  
{  
   A a = null;  
   public void G<T>()  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  // CS0413  
   }  
  
   // OK  
   public void H<T>() where T : A  
   {  
      a = new A();  
      System.Console.WriteLine (a as T);  
   }  
}  
```
