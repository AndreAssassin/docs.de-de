## <a name="ref-struct-types"></a><span data-ttu-id="ec4d6-177">ref-Strukturtypen</span><span class="sxs-lookup"><span data-stu-id="ec4d6-177">Ref struct types</span></span>

<span data-ttu-id="ec4d6-178">Das Hinzufügen des `ref`-Modifizierers zu einer `struct`-Deklaration definiert, dass Instanzen dieses Typs stapelzugeordnet sein müssen.</span><span class="sxs-lookup"><span data-stu-id="ec4d6-178">Adding the `ref` modifier to a `struct` declaration defines that instances of that type must be stack allocated.</span></span> <span data-ttu-id="ec4d6-179">Das bedeutet, Instanzen dieses Typs können nie im Heap als Member einer anderen Klasse erstellt werden.</span><span class="sxs-lookup"><span data-stu-id="ec4d6-179">In other words, instances of these types can never be created on the heap as a member of another class.</span></span> <span data-ttu-id="ec4d6-180">Der primäre Beweggrund für dieses Feature waren <xref:System.Span%601> und zugehörige Strukturen.</span><span class="sxs-lookup"><span data-stu-id="ec4d6-180">The primary motivation for this feature was <xref:System.Span%601> and related structures.</span></span>

- <span data-ttu-id="ec4d6-189">Sie können lokale `ref struct`-Variablen nicht in Iteratoren deklarieren.</span><span class="sxs-lookup"><span data-stu-id="ec4d6-189">You cannot declare `ref struct` local variables in iterators.</span></span>
