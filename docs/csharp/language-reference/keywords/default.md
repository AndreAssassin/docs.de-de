# <a name="default-c-reference"></a>default (C#-Referenz)

Das `default`-Schlüsselwort kann auf zwei Arten verwendet werden:
- Die [`switch`-Anweisung](switch.md) zum Angeben der Standardbezeichnung.
- Ein [Standardwertausdruck](../../programming-guide/statements-expressions-operators/default-value-expressions.md) zum Erzeugen des Standardwerts für einen Typ.
