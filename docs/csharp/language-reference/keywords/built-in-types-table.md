## <a name="remarks"></a><span data-ttu-id="6c0cb-122">Anmerkungen</span><span class="sxs-lookup"><span data-stu-id="6c0cb-122">Remarks</span></span>

<span data-ttu-id="6c0cb-123">Mit Ausnahme von `object` und `string` werden alle in der Tabelle enthaltenen Typen als einfache Typen bezeichnet.</span><span class="sxs-lookup"><span data-stu-id="6c0cb-123">All of the types in the table, except `object` and `string`, are referred to as simple types.</span></span>  
  
<span data-ttu-id="6c0cb-124">C#-Typschlüsselwörter und ihre Aliase sind austauschbar.</span><span class="sxs-lookup"><span data-stu-id="6c0cb-124">The C# type keywords and their aliases are interchangeable.</span></span> <span data-ttu-id="6c0cb-125">So können Sie beispielsweise eine ganzzahlige Variable mit einer der beiden folgenden Deklarationen angeben:</span><span class="sxs-lookup"><span data-stu-id="6c0cb-125">For example, you can declare an integer variable by using either of the following declarations:</span></span>  

```csharp
int x = 123;
System.Int32 y = 123;
```

<span data-ttu-id="6c0cb-126">Verwenden Sie den Operator [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator), um die <xref:System.Type?displayProperty=nameWithType>-Instanz abzurufen, die den angegebenen Typ darstellt:</span><span class="sxs-lookup"><span data-stu-id="6c0cb-126">Use the [typeof](../operators/type-testing-and-conversion-operators.md#typeof-operator) operator to get the <xref:System.Type?displayProperty=nameWithType> instance that represents the specified type:</span></span>

```csharp
Type stringType = typeof(string);
Console.WriteLine(stringType.FullName);

Type doubleType = typeof(System.Double);
Console.WriteLine(doubleType.FullName);

// Output:
// System.String
// System.Double
```
