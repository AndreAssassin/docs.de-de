# <a name="null-c-reference"></a>null (C#-Referenz)

Das Schlüsselwort `null` ist ein Literal, das einen NULL-Verweis darstellt, der auf kein Objekt verweist. `null` ist der Standardwert einer Verweistypvariablen. Normale Verweistypen dürfen nicht NULL sein. C# 2.0 hat jedoch auf NULL festlegbare Werttypen eingeführt. Siehe [Nullable-Typen](../../programming-guide/nullable-types/index.md).
