# <a name="reference-tables-for-types-c-reference"></a>Referenztabellen für Typen (C#-Referenz)

Die C#-Typen sind in den folgenden Referenztabellen zusammengefasst:

|Tabelle|Beschreibung|
|---------|---------|
|[Integrierte Typen](built-in-types-table.md)|Listet integrierte C#-Typen und die entsprechenden .NET-Typen auf.|
|[Integrale Typen](integral-types-table.md)|Bietet eine Übersicht der integralen Typen.|
|[Gleitkommatypen](floating-point-types-table.md)|Bietet eine Übersicht der Gleitkommatypen.|
|[Werttypen](value-types-table.md)|Listet C#-Werttypen auf.|
|[Standardwerte](default-values-table.md)|Listet Standardwerte von C#-Werttypen auf.|
|[Implizite numerische Konvertierungen](implicit-numeric-conversions-table.md)|Bietet eine Übersicht der unterstützten impliziten Konvertierungen zwischen numerischen C#-Typen.|
|[Explizite numerische Konvertierungen](explicit-numeric-conversions-table.md)|Bietet eine Übersicht der unterstützten expliziten Konvertierungen zwischen numerischen C#-Typen.|
|[Formatierung numerischer Ergebnisse](formatting-numeric-results-table.md)|Bietet eine Übersicht der numerischen Standardformatzeichenfolgen.|
