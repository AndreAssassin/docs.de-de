# <a name="endregion-c-reference"></a>#endregion (C#-Referenz)
`#endregion` markiert das Ende eines [#region](./preprocessor-region.md)-Blocks. Zum Beispiel:  
  
```csharp
#region MyClass definition  
class MyClass   
{  
    static void Main()   
    {  
    }  
}  
#endregion  
```
