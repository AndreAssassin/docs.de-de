## <a name="example"></a><span data-ttu-id="9a066-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="9a066-106">Example</span></span>  
 <span data-ttu-id="9a066-107">In diesem Beispiel wird der Fehler CS0633 bei Verwendung von <xref:System.Diagnostics.ConditionalAttribute>veranschaulicht.</span><span class="sxs-lookup"><span data-stu-id="9a066-107">This example illustrates CS0633 using the <xref:System.Diagnostics.ConditionalAttribute>.</span></span> <span data-ttu-id="9a066-108">Im folgenden Beispiel wird CS0633 generiert.</span><span class="sxs-lookup"><span data-stu-id="9a066-108">The following sample generates CS0633.</span></span>  
  
```csharp  
// CS0633a.cs

using System.Diagnostics;  
public class Test  
{  
   [Conditional("DEB+UG")]   // CS0633  
   // try the following line instead  
   // [Conditional("DEBUG")]  
   public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="9a066-109">Beispiel</span><span class="sxs-lookup"><span data-stu-id="9a066-109">Example</span></span>  
 <span data-ttu-id="9a066-110">In diesem Beispiel wird der Fehler CS0633 bei Verwendung von <xref:System.Runtime.CompilerServices.IndexerNameAttribute>veranschaulicht.</span><span class="sxs-lookup"><span data-stu-id="9a066-110">This example illustrates CS0633 using the <xref:System.Runtime.CompilerServices.IndexerNameAttribute>.</span></span>  
  
```csharp  
// CS0633b.cs  
// compile with: /target:module  
#define DEBUG  
using System.Runtime.CompilerServices;  
public class Test  
{  
   [IndexerName("Invalid+Identifier")]   // CS0633  
   // try the following line instead  
   // [IndexerName("DEBUG")]  
   public int this[int i]
   {
      get { return i; }
   }  
}  
```
