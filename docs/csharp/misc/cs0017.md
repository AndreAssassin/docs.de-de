# <a name="compiler-error-cs0017"></a><span data-ttu-id="1aa20-102">Compilerfehler CS0017</span><span class="sxs-lookup"><span data-stu-id="1aa20-102">Compiler Error CS0017</span></span>
<span data-ttu-id="1aa20-103">Für das Programm „Ausgabedateiname“ sind mehrere Einstiegspunkte definiert.</span><span class="sxs-lookup"><span data-stu-id="1aa20-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="1aa20-104">Kompilieren Sie mit „/main“, um den Typ anzugeben, der den Einstiegspunkt enthält.</span><span class="sxs-lookup"><span data-stu-id="1aa20-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="1aa20-105">Ein Programm kann nur eine [Main](../programming-guide/main-and-command-args/index.md) -Methode besitzen.</span><span class="sxs-lookup"><span data-stu-id="1aa20-105">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="1aa20-106">Um diesen Fehler zu beheben, können Sie entweder alle Main-Methoden im Code löschen, mit Ausnahme von einem, oder Sie können die [-Main-](../language-reference/compiler-options/main-compiler-option.md) Compileroption verwenden, um anzugeben, welche Hauptmethode Sie verwenden möchten.</span><span class="sxs-lookup"><span data-stu-id="1aa20-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [-main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="1aa20-107">Im folgenden Beispiel wird CS0017 generiert:</span><span class="sxs-lookup"><span data-stu-id="1aa20-107">The following sample generates CS0017:</span></span>  
  
```csharp

{  
   static public void Main()  
   {  
   }  
}

{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
