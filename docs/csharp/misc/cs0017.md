<span data-ttu-id="38ac8-103">Für das Programm „Ausgabedateiname“ sind mehrere Einstiegspunkte definiert.</span><span class="sxs-lookup"><span data-stu-id="38ac8-103">Program 'output file name' has more than one entry point defined.</span></span> <span data-ttu-id="38ac8-104">Kompilieren Sie mit „/main“, um den Typ anzugeben, der den Einstiegspunkt enthält.</span><span class="sxs-lookup"><span data-stu-id="38ac8-104">Compile with /main to specify the type that contains the entry point.</span></span>  
  
 <span data-ttu-id="38ac8-105">Ein Programm kann nur eine [Main](../programming-guide/main-and-command-args/index.md) -Methode besitzen.</span><span class="sxs-lookup"><span data-stu-id="38ac8-105">A program can only have one [Main](../programming-guide/main-and-command-args/index.md) method.</span></span>  
  
 <span data-ttu-id="38ac8-106">Um diesen Fehler zu beheben, löschen Sie entweder alle Main-Methoden im Code (bis auf eine), oder verwenden Sie die [/main](../language-reference/compiler-options/main-compiler-option.md) -Compileroption, um die zu verwendende Main-Methode anzugeben.</span><span class="sxs-lookup"><span data-stu-id="38ac8-106">To resolve this error, you can either delete all Main methods in your code, except one, or you can use the [/main](../language-reference/compiler-options/main-compiler-option.md) compiler option to specify which Main method you want to use.</span></span>  
  
 <span data-ttu-id="38ac8-107">Im folgenden Beispiel wird CS0017 generiert:</span><span class="sxs-lookup"><span data-stu-id="38ac8-107">The following sample generates CS0017:</span></span>  
  
```csharp

{  
   static public void Main()  
   {  
   }  
}

{  
   public static void Main()   // CS0017, delete one Main or use /main  
   {  
   }  
}  
```
