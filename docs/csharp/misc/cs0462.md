<span data-ttu-id="c2d84-103">Die geerbten Member "Member1" und "Member2" haben die gleiche Signatur im Typ "Typ" und können daher nicht überschrieben werden</span><span class="sxs-lookup"><span data-stu-id="c2d84-103">The inherited members 'member1' and 'member2' have the same signature in type 'type', so they cannot be overridden</span></span>  
  
 <span data-ttu-id="c2d84-104">Dieser Fehler entsteht bei der Einführung von Generics.</span><span class="sxs-lookup"><span data-stu-id="c2d84-104">This error arises with the introduction of generics.</span></span> <span data-ttu-id="c2d84-105">In der Regel können in einer Klasse mit derselben Signatur nicht zwei Versionen einer Methode vorhanden sein.</span><span class="sxs-lookup"><span data-stu-id="c2d84-105">Normally, you cannot have two versions of a method in a class with the same signature.</span></span> <span data-ttu-id="c2d84-106">Bei Generics können Sie jedoch eine generische Methode angeben, die möglicherweise eine andere Methode dupliziert, wenn sie mit einem bestimmten Typ instanziiert wird.</span><span class="sxs-lookup"><span data-stu-id="c2d84-106">But with generics, you can specify a generic method that might duplicate another method if it is instantiated with a particular type.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c2d84-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="c2d84-107">Example</span></span>  
 <span data-ttu-id="c2d84-108">Beim Instanziieren von `C<int>` werden zwei Versionen der `F` -Methode mit derselben Signatur erstellt, sodass beim Überschreiben in der `D` -Klasse keine Entscheidung getroffen werden kann, welche Version überschrieben werden soll.</span><span class="sxs-lookup"><span data-stu-id="c2d84-108">When `C<int>` is instantiated, two versions of the method `F` are created with the same signature, so the override in class `D` cannot decide which one to apply the override to.</span></span>  
  
 <span data-ttu-id="c2d84-109">Im folgenden Beispiel wird CS0462 generiert.</span><span class="sxs-lookup"><span data-stu-id="c2d84-109">The following sample generates CS0462.</span></span>  
  
```csharp  
// CS0462.cs  
// compile with: /target:library  
class C<T>
{  
   public virtual void F(T t) {}  
   public virtual void F(int t) {}  
}  
  
class D : C<int>
{  
   public override void F(int t) {}   // CS0462  
}  
```
