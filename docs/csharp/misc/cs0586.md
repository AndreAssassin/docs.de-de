 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, weil er unerwartete Syntax nicht analysieren kann. Wenn dies nicht der Fall ist, versuchen Sie es [wenden Sie sich an uns](/visualstudio/ide/talk-to-us).
