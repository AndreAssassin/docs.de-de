# <a name="compiler-error-cs0586"></a>Compilerfehler CS0586

Interner Compilerfehler: Stufe 'stufe'

 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, weil er unerwartete Syntax nicht analysieren kann. Wenn dies nicht der Fall ist, versuchen [Sie, mit uns sprechen](/visualstudio/ide/feedback-options).

> [!NOTE]
> Dieser Compilerfehler wird in Roslyn nicht mehr verwendet.
