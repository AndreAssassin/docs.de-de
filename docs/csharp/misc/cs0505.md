 <span data-ttu-id="c4de7-107">Im folgenden Beispiel wird CS0505 generiert:</span><span class="sxs-lookup"><span data-stu-id="c4de7-107">The following sample generates CS0505:</span></span>  
  
```csharp

public class clx  
{  
   public int i;  
}
```
