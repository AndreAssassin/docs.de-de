 <span data-ttu-id="917a6-104">Eine Klassendeklaration hat versucht, eine Nichtmethode in einer Basisklasse zu überschreiben.</span><span class="sxs-lookup"><span data-stu-id="917a6-104">A class declaration attempted to override a non-method in a base class.</span></span> <span data-ttu-id="917a6-105">Bei Überschreibungen muss der Membertyp übereinstimmen.</span><span class="sxs-lookup"><span data-stu-id="917a6-105">Overrides must match the member type.</span></span> <span data-ttu-id="917a6-106">Wenn eine Methode mit demselben Namen wie für eine Methode in einer Basisklasse benötigt wird, verwenden Sie [new](../../csharp/language-reference/keywords/new.md) (und nicht [override](../../csharp/language-reference/keywords/override.md)) für die Deklaration der Methode in der Basisklasse.</span><span class="sxs-lookup"><span data-stu-id="917a6-106">If a method with the same name as a method in a base class is desired, use [new](../../csharp/language-reference/keywords/new.md) (and not [override](../../csharp/language-reference/keywords/override.md)) on the method declaration in the base class.</span></span>  
  
 <span data-ttu-id="917a6-107">Im folgenden Beispiel wird CS0505 generiert:</span><span class="sxs-lookup"><span data-stu-id="917a6-107">The following sample generates CS0505:</span></span>  
  
```csharp

public class clx  
{  
   public int i;  
}

   public override int i() { return 0; }   // CS0505  
}  
```
