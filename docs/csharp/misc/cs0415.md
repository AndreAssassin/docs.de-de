 <span data-ttu-id="b9ff0-104">Dieser Fehler tritt auf, wenn Sie ein IndexerName-Attribut für einen Indexer verwenden, der eine explizite Implementierung einer Schnittstelle darstellt.</span><span class="sxs-lookup"><span data-stu-id="b9ff0-104">This error occurs if you use an IndexerName attribute on an indexer that was an explicit implementation of an interface.</span></span> <span data-ttu-id="b9ff0-105">Dieser Fehler kann vermieden werden, indem der Schnittstellenname aus der Deklaration des Indexers entfernt wird, falls das möglich ist.</span><span class="sxs-lookup"><span data-stu-id="b9ff0-105">This error may be avoided by removing the interface name from the declaration of the indexer, if possible.</span></span> <span data-ttu-id="b9ff0-106">Weitere Informationen finden Sie bei der [IndexerNameAttribute-Klasse](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span><span class="sxs-lookup"><span data-stu-id="b9ff0-106">For more information, see the [IndexerNameAttribute Class](xref:System.Runtime.CompilerServices.IndexerNameAttribute).</span></span>  
  
 <span data-ttu-id="b9ff0-107">Im folgenden Beispiel wird CS0415 generiert:</span><span class="sxs-lookup"><span data-stu-id="b9ff0-107">The following sample generates CS0415:</span></span>  
  
```csharp
// CS0415.cs  
using System;  
using System.Runtime.CompilerServices;  
  
public interface IA  
{  
    int this[int index]  
    {  
        get;  
        set;  
    }  
}  
  
public class A : IA  
{  
    [IndexerName("Item")]  // CS0415  
    int IA.this[int index]  
    // Try this line instead:  
    // public int this[int index]  
    {  
        get { return 0; }  
        set { }  
    }  
  
    static void Main()  
    {  
    }  
}  
```
