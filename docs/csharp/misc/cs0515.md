# <a name="compiler-error-cs0515"></a>Compilerfehler CS0515
"Funktion": Zugriffsmodifizierer sind bei statischen Konstruktoren nicht zul√§ssig.  
  
 Ein statischer Konstruktor kann keinen [Zugriffsmodifizierer](../language-reference/keywords/modifiers.md)enthalten.  
  
## <a name="example"></a>Beispiel  
 Im folgenden Beispiel wird CS0515 generiert:  
  
```csharp  
// CS0515.cs  
public class Clx  
{  
    public static void Main()  
    {  
    }  
}  
  
public class Clz  
{  
    public static Clz()   // CS0515, remove public keyword  
    {  
    }  
}  
```
