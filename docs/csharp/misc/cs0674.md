 <span data-ttu-id="c0b83-105">Der C#-Compiler lässt die Verwendung von <xref:System.ParamArrayAttribute?displayProperty=nameWithType>nicht zu; verwenden Sie stattdessen [params](../../csharp/language-reference/keywords/params.md) .</span><span class="sxs-lookup"><span data-stu-id="c0b83-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../../csharp/language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="c0b83-106">Im folgenden Beispiel wird CS0674 generiert:</span><span class="sxs-lookup"><span data-stu-id="c0b83-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs

public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
