# <a name="compiler-error-cs0674"></a><span data-ttu-id="c47e8-102">Compilerfehler CS0674</span><span class="sxs-lookup"><span data-stu-id="c47e8-102">Compiler Error CS0674</span></span>
<span data-ttu-id="c47e8-103">Verwenden Sie nicht "System.ParamArrayAttribute".</span><span class="sxs-lookup"><span data-stu-id="c47e8-103">Do not use 'System.ParamArrayAttribute'.</span></span> <span data-ttu-id="c47e8-104">Verwenden Sie stattdessen das Schlüsselwort "params".</span><span class="sxs-lookup"><span data-stu-id="c47e8-104">Use the 'params' keyword instead.</span></span>  
  
 <span data-ttu-id="c47e8-105">Der C#-Compiler lässt die Verwendung von <xref:System.ParamArrayAttribute?displayProperty=nameWithType>nicht zu; verwenden Sie stattdessen [params](../language-reference/keywords/params.md) .</span><span class="sxs-lookup"><span data-stu-id="c47e8-105">The C# compiler does not allow for the use of <xref:System.ParamArrayAttribute?displayProperty=nameWithType>; use [params](../language-reference/keywords/params.md) instead.</span></span>  
  
 <span data-ttu-id="c47e8-106">Im folgenden Beispiel wird CS0674 generiert:</span><span class="sxs-lookup"><span data-stu-id="c47e8-106">The following sample generates CS0674:</span></span>  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
