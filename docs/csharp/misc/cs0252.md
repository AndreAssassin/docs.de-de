 <span data-ttu-id="acba5-104">Der Compiler führt einen Verweisvergleich aus.</span><span class="sxs-lookup"><span data-stu-id="acba5-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="acba5-105">Wenn Sie den Wert von Zeichenfolgen vergleichen möchten, wandeln Sie die linke Seite des Ausdrucks in `type`um.</span><span class="sxs-lookup"><span data-stu-id="acba5-105">If you want to compare the value of strings, cast the left side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="acba5-106">Im folgenden Beispiel wird CS0252 generiert:</span><span class="sxs-lookup"><span data-stu-id="acba5-106">The following sample generates CS0252:</span></span>  
  
```csharp  
// CS0252.cs  
// compile with: /W:2  
using System;  
  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool b = o == s;   // CS0252  
      // try the following line instead  
      // bool b = (string)o == s;  
   }  
}  
```
