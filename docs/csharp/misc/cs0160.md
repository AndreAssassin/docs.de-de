# <a name="compiler-error-cs0160"></a>Compilerfehler CS0160
Eine vorherige Catch-Klausel hat bereits alle Ausnahmen dieses oder eines übergeordneten Typs abgefangen („type“).  
  
Eine Reihe von [catch](../../csharp/language-reference/keywords/try-catch.md) -Anweisungen muss in absteigender Reihenfolge der Ableitung angegeben sein. Die meisten abgeleiteten Objekte müssen beispielsweise zuerst angezeigt werden.
  
 Weitere Informationen finden Sie unter [Ausnahmen und Ausnahmebehandlung](../../csharp/programming-guide/exceptions/index.md).  
  
 Im folgenden Beispiel wird CS0160 generiert:  
  
```csharp  
// CS0160.cs  
public class MyClass2 : System.Exception {}  
public class MyClass  
{  
   public static void Main()  
   {  
      try {}  
  
      catch(System.Exception) {}   // Second-most derived; should be second catch  
      catch(MyClass2) {}   // CS0160  Most derived; should be first catch  
   }  
}  
```
