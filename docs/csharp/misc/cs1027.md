# <a name="compiler-error-cs1027"></a>Compilerfehler CS1027
\#die Anweisung "a" wurde erwartet.  
  
 Für eine angegebene `#endif` [preprocessor directive](../language-reference/preprocessor-directives/index.md) was not found for a specified `#if` directive. Oder der Compiler hat möglicherweise eine `#endregion` -Direktive gefunden, obwohl keine entsprechende `#region` -Direktive innerhalb eines `#if` -Blocks vorhanden ist.  
  
 Im folgenden Beispiel wird CS1027 generiert:  
  
```csharp  
// CS1027.cs  
#if true   // CS1027, uncomment next line to resolve  
// #endif  
  
namespace x  
{  
   public class clx  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
