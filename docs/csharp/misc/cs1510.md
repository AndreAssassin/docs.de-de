## <a name="example"></a><span data-ttu-id="f2f60-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="f2f60-106">Example</span></span>  
 <span data-ttu-id="f2f60-107">Im folgenden Beispiel wird CS1510 generiert:</span><span class="sxs-lookup"><span data-stu-id="f2f60-107">The following sample generates CS1510:</span></span>  
  
```csharp

   public static int j = 0;  
  
   public static void M(ref int j)  
   {  
      j++;  
   }  
  
   public static void Main ()  
   {  
      M (ref 2);   // CS1510, can't pass a number as a ref parameter  
      // try the following to resolve the error  
      // M (ref j);  
   }  
}  
```
