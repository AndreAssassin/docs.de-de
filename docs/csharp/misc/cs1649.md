# <a name="compiler-error-cs1649"></a><span data-ttu-id="86a81-102">Compilerfehler CS1649</span><span class="sxs-lookup"><span data-stu-id="86a81-102">Compiler Error CS1649</span></span>
<span data-ttu-id="86a81-103">An Member des schreibgeschützten Felds "Bezeichner" kann kein Verweis (ref) und keine Ausgabe (out) übergeben werden (Ausnahme: in einem Konstruktor).</span><span class="sxs-lookup"><span data-stu-id="86a81-103">Members of readonly field 'identifier' cannot be passed ref or out (except in a constructor)</span></span>  
  
 <span data-ttu-id="86a81-104">Dieser Fehler tritt auf, wenn Sie eine Variable an eine Funktion, die ein Member eines `readonly` -Felds ist, als `ref` - oder `out` -Argument übergeben.</span><span class="sxs-lookup"><span data-stu-id="86a81-104">This error occurs if you pass a variable to a function that is a member of a `readonly` field as a `ref` or `out` argument.</span></span> <span data-ttu-id="86a81-105">Da `ref` - und `out` -Parameter von der Funktion geändert werden können, ist dies nicht zulässig.</span><span class="sxs-lookup"><span data-stu-id="86a81-105">Since `ref` and `out` parameters may be modified by the function, this is not allowed.</span></span> <span data-ttu-id="86a81-106">Um diesen Fehler zu beheben, entfernen Sie das `readonly` -Schlüsselwort für das Feld, oder übergeben Sie die Member des `readonly` -Felds nicht an die Funktion.</span><span class="sxs-lookup"><span data-stu-id="86a81-106">To resolve this error, remove the `readonly` keyword on the field, or do not pass the members of the `readonly` field to the function.</span></span> <span data-ttu-id="86a81-107">Sie können z. ## <a name="example"></a><span data-ttu-id="86a81-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="86a81-108">Example</span></span>  
 <span data-ttu-id="86a81-109">Im folgenden Beispiel wird CS1649 generiert:</span><span class="sxs-lookup"><span data-stu-id="86a81-109">The following sample generates CS1649:</span></span>  
  
```csharp  
// CS1649.cs  
public struct Inner  
    {  
        public int i;  
    }  
  
class Outer  
{  
    public readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void f(ref int iref)  
    {  
    }  
  
    static void Main()  
    {  
        Outer outer = new Outer();
        f(ref outer.inner.i);  // CS1649  
        // Try this code instead:  
        // int tmp = outer.inner.i;  
        // f(ref tmp);  
    }  
}  
```
