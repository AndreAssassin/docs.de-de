## <a name="example"></a><span data-ttu-id="21a86-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="21a86-108">Example</span></span>  
 <span data-ttu-id="21a86-109">Diese Datei wird in eine DLL-Datei kompiliert. Dabei wurde das `DefaultMember` -Attribut auskommentiert, um den Fehler zu generieren.</span><span class="sxs-lookup"><span data-stu-id="21a86-109">This file compiles to a .dll file—with the `DefaultMember` attribute commented out—in order to generate the error.</span></span>  
  
```cpp  
// CPP0021.cpp  
// compile with: /clr /LD  
using namespace System::Reflection;  
// Uncomment the following line to resolve  
//[DefaultMember("myItem")]  
public ref class MyClassMC  
{  
        public:  
        property int myItem[int]  
        {  
            int get(int i){  return 5; }  
            void set(int i, int value) {}  
        }  
};  
```
