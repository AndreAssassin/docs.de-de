# <a name="compiler-error-cs1906"></a>Compilerfehler CS1906
Ungültige Option "Option"; Ressourcensichtbarkeit muss entweder "public" oder "private" sein.  
  
 Dieser Fehler zeigt an, dass eine ungültige [/Resource (Ressourcen Datei in Ausgabe einbetten)](../language-reference/compiler-options/resource-compiler-option.md) oder die Befehlszeilenoption [-linkresource (Link zur .NET Framework Ressource)](../language-reference/compiler-options/linkresource-compiler-option.md) vorliegt. Überprüfen Sie die Syntax der **/resource** - oder **/linkresource** -Befehlszeilenoption, und stellen Sie sicher, dass der verwendete Zugriffsmodifizierer entweder **public** - oder `private`ist.
