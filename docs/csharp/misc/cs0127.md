# <a name="compiler-error-cs0127"></a><span data-ttu-id="34e47-102">Compilerfehler CS0127</span><span class="sxs-lookup"><span data-stu-id="34e47-102">Compiler Error CS0127</span></span>
<span data-ttu-id="34e47-103">Da 'Funktion' 'void' zurückgibt, darf auf ein Rückgabeschlüsselwort kein Objektausdruck folgen.</span><span class="sxs-lookup"><span data-stu-id="34e47-103">Since 'function' returns void, a return keyword must not be followed by an object expression</span></span>  
  
 <span data-ttu-id="34e47-104">Eine Methode mit dem Rückgabetyp [void](../language-reference/keywords/void.md) kann keinen Wert zurückgeben.</span><span class="sxs-lookup"><span data-stu-id="34e47-104">A method with a [void](../language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="34e47-105">Weitere Informationen finden Sie unter [Methoden](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="34e47-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="34e47-106">Im folgenden Beispiel wird CS0127 generiert:</span><span class="sxs-lookup"><span data-stu-id="34e47-106">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
