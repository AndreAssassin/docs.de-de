 <span data-ttu-id="8c5de-104">Eine Klasse, die die Eigenschaft einer Schnittstelle explizit implementiert, muss alle Accessoren implementieren, die von der Schnittstelle definiert werden.</span><span class="sxs-lookup"><span data-stu-id="8c5de-104">A class that explicitly implements an interface's property must implement all the accessors that the interface defines.</span></span>

## <a name="example"></a><span data-ttu-id="8c5de-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="8c5de-106">Example</span></span>  
 <span data-ttu-id="8c5de-107">Im folgenden Beispiel wird CS0551 generiert:</span><span class="sxs-lookup"><span data-stu-id="8c5de-107">The following sample generates CS0551.</span></span>  
  
```csharp  
// CS0551.cs  
// compile with: /target:library  
interface ii  
{  
   int i  
   {  
      get;  
      set;  
   }  
}  
  
public class a : ii  
{  
   int ii.i { set {} }   // CS0551  
  
   // OK  
   int ii.i
   {  
      set {}  
      get { return 0; }  
   }  
}  
```
