# <a name="compiler-error-cs0758"></a><span data-ttu-id="25cf9-102">Compilerfehler CS0758</span><span class="sxs-lookup"><span data-stu-id="25cf9-102">Compiler Error CS0758</span></span>
<span data-ttu-id="25cf9-103">Beide partiellen Methodendeklarationen müssen einen params-Parameter verwenden, oder keine von beiden darf einen params-Parameter verwenden.</span><span class="sxs-lookup"><span data-stu-id="25cf9-103">Both partial method declarations must use a params parameter or neither may use a params parameter</span></span>  
  
 <span data-ttu-id="25cf9-104">Wenn in einem Teil einer partiellen Methode ein `params` -Parameter angegeben ist, muss dieser Parameter auch im anderen Teil enthalten sein.</span><span class="sxs-lookup"><span data-stu-id="25cf9-104">If one part of a partial method specifies a `params` parameter, the other part must specify one also.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="25cf9-105">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="25cf9-105">To correct this error</span></span>  
  
1. <span data-ttu-id="25cf9-106">Fügen Sie einem Teil der Methode den `params` -Modifizierer hinzu, oder entfernen Sie in aus dem anderen Teil.</span><span class="sxs-lookup"><span data-stu-id="25cf9-106">Either add the `params` modifier in one part of the method, or remove it in the other.</span></span>  
  
## <a name="example"></a><span data-ttu-id="25cf9-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="25cf9-107">Example</span></span>  
 <span data-ttu-id="25cf9-108">Durch den folgenden Code wird der Fehler CS0758 ausgelöst:</span><span class="sxs-lookup"><span data-stu-id="25cf9-108">The following code generates CS0758:</span></span>

        }  
  
    }
