# <a name="compiler-error-cs0761"></a><span data-ttu-id="23287-102">Compilerfehler CS0761</span><span class="sxs-lookup"><span data-stu-id="23287-102">Compiler Error CS0761</span></span>

 <span data-ttu-id="23287-104">Weist eine partielle Methode eine Implementierung auf, muss die generische Typeinschränkung mit der Einschränkung identisch sein, die für die Methodensignatur definiert wurde.</span><span class="sxs-lookup"><span data-stu-id="23287-104">If a partial method has an implementation, the generic type constraint must be identical to the constraint defined on the method signature.</span></span>
