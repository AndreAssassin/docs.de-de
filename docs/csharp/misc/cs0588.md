# <a name="compiler-error-cs0588"></a>Compilerfehler CS0588

Interner Compilerfehler: Stufe 'LEX'

 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, da unerwartete Syntax nicht analysiert werden kann. Wenn dies nicht der Fall ist, wenden Sie [sich an uns](/visualstudio/ide/talk-to-us).
 
> [!NOTE]
> Dieser Compilerfehler wird in Roslyn nicht mehr verwendet.
