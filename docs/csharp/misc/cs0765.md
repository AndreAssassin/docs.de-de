# <a name="compiler-error-cs0765"></a><span data-ttu-id="95a70-102">Compilerfehler CS0765</span><span class="sxs-lookup"><span data-stu-id="95a70-102">Compiler Error CS0765</span></span>
<span data-ttu-id="95a70-103">In Ausdrucksbaumstrukturen dürfen weder partielle Methoden mit nur einer definierenden Deklaration noch entfernte bedingte Methoden verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="95a70-103">Partial methods with only a defining declaration or removed conditional methods cannot be used in expression trees</span></span>  
  
 <span data-ttu-id="95a70-104">Obwohl es sich bei einem Aufruf einer entfernten partiellen Methode um einen Ausdruck handelt, stellt er keinen in einem Ausdrucksbaum akzeptablen Ausdruck dar.</span><span class="sxs-lookup"><span data-stu-id="95a70-104">Although a call to a removed partial method is an expression, it is not an acceptable expression in an expression tree.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="95a70-105">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="95a70-105">To correct this error</span></span>  
  
1. <span data-ttu-id="95a70-106">Fügen Sie eine implementierende Deklaration für die partielle Methode hinzu, oder entfernen Sie den Code, der den Ausschluss der bedingten Methode bei der Kompilation verhindert.</span><span class="sxs-lookup"><span data-stu-id="95a70-106">Add an implementing declaration for the partial method, or remove the code that is causing the conditional method to be excluded from compilation.</span></span>  
  
## <a name="example"></a><span data-ttu-id="95a70-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="95a70-107">Example</span></span>  
 <span data-ttu-id="95a70-108">Mit dem folgenden Code wird CS0765 an zwei Positionen generiert:</span><span class="sxs-lookup"><span data-stu-id="95a70-108">The following code generates CS0765 in two locations:</span></span>  
  
```csharp  
// cs0765.cs  
using System;  
using System.Collections;  
using System.Collections.Generic;  
using System.Diagnostics;  
using System.Linq;  
using System.Linq.Expressions;  
  
public delegate void dele();  
  
public class ConClass  
{  
    [Conditional("CONDITION")]  
    public static void TestMethod() { }  
}  
  
public partial class PartClass : IEnumerable  
{  
    List<object> list = new List<object>();  
  
    partial void Add(int x);  
  
    public IEnumerator GetEnumerator()  
    {  
        for (int i = 0; i < list.Count; i++)  
            yield return list[i];  
    }  
  
    static void Main()  
    {  
        Expression<Func<PartClass>> testExpr1 = () => new PartClass { 1, 2 }; // CS0765  
        Expression<dele> testExpr2 = () => ConClass.TestMethod(); // CS0765  
    }  
}  
```
