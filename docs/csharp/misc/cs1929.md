# <a name="compiler-error-cs1929"></a><span data-ttu-id="21ec4-102">Compilerfehler CS1929</span><span class="sxs-lookup"><span data-stu-id="21ec4-102">Compiler Error CS1929</span></span>
<span data-ttu-id="21ec4-103">Instanzenargument: Konvertierung von 'TypA' in 'TypB' nicht möglich.</span><span class="sxs-lookup"><span data-stu-id="21ec4-103">Instance argument: cannot convert from 'typeA' to 'typeB'.</span></span>  
  
 <span data-ttu-id="21ec4-104">Dieser Fehler wird generiert, wenn Sie versuchen, eine Erweiterungsmethode über eine Klasse aufzurufen, die sie nicht erweitert.</span><span class="sxs-lookup"><span data-stu-id="21ec4-104">This error is generated when you try to invoke an extension method from a class that it does not extend.</span></span> <span data-ttu-id="21ec4-105">Im hier gezeigten Beispiel wird die Erweiterungsmethode für die abgeleitete Klasse `A`definiert, jedoch nicht für die Basisklasse `B`.</span><span class="sxs-lookup"><span data-stu-id="21ec4-105">In the example shown here, the extension method is defined for the derived class `A`, but not for the base class `B`.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="21ec4-106">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="21ec4-106">To correct this error</span></span>  
  
1. <span data-ttu-id="21ec4-107">Erstellen Sie eine neue Erweiterungsmethode für den Typ an der Stelle, wo Sie sie aufrufen müssen. Verschieben Sie den Aufruf andernfalls in ein Objekt des Typs, der von der vorhandenen Methode erweitert wird.</span><span class="sxs-lookup"><span data-stu-id="21ec4-107">Create a new extension method for the type where you have to invoke it, or else move the call into an object of the type that the existing method extends.</span></span>  
  
## <a name="example"></a><span data-ttu-id="21ec4-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="21ec4-108">Example</span></span>  
 <span data-ttu-id="21ec4-109">Durch den folgenden Code werden die Fehler CS1928 und CS1929 ausgelöst:</span><span class="sxs-lookup"><span data-stu-id="21ec4-109">The following code generates CS1928 and CS1929:</span></span>  
  
```csharp  
// cs1929.cs  
using System.Linq;  
    using System.Collections;  
  
    static class Ext  
    {  
        public static void ExtMethod(this A a)  
        {  
        }  
    }  
  
    class A : B  
    {  
    }  
  
    class B  
    {  
        static void Main()  
        {  
            B b = new B();  
            b.ExtMethod(); // CS1929  
        }  
    }  
```
