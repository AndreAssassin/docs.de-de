# <a name="compiler-warning-level-4-cs0109"></a>Compilerwarnung (Stufe 4) CS0109
Der Member "Member" blendet einen vererbten Member nicht aus. Das New-Schlüsselwort ist nicht erforderlich  
  
 Eine Klassendeklaration enthält das [New](../../csharp/language-reference/keywords/new-modifier.md) -Schlüsselwort, obwohl die Deklaration eine bestehende Deklaration in einer Basisklasse nicht überschreibt. Sie können das **New** -Schlüsselwort löschen.  
  
 Im folgenden Beispiel wird CS0109 generiert:  
  
```csharp  
// CS0109.cs  
// compile with: /W:4  
namespace x  
{  
   public class a  
   {  
      public int i;  
   }  
  
   public class b : a  
   {  
      public new int i;  
      public new int j;   // CS0109  
      public static void Main()  
      {  
      }  
   }  
}  
```
