# <a name="compiler-error-cs0535"></a><span data-ttu-id="dd041-102">Compilerfehler CS0535</span><span class="sxs-lookup"><span data-stu-id="dd041-102">Compiler Error CS0535</span></span>
<span data-ttu-id="dd041-103">'Klasse' implementiert den Schnittstellenmember 'Member' nicht.</span><span class="sxs-lookup"><span data-stu-id="dd041-103">'class' does not implement interface member 'member'</span></span>  
  
 <span data-ttu-id="dd041-104">Eine [Klasse](../language-reference/keywords/class.md) , die von einer [Schnittstelle](../language-reference/keywords/interface.md)abgeleitet wird, aber die Klasse hat mindestens ein Member der Schnittstelle nicht implementiert.</span><span class="sxs-lookup"><span data-stu-id="dd041-104">A [class](../language-reference/keywords/class.md) derived from an [interface](../language-reference/keywords/interface.md), but the class did not implement one or more of the interface's members.</span></span> <span data-ttu-id="dd041-105">Eine Klasse muss alle Member der Schnittstellen implementieren, von denen sie abgeleitet wird, oder ansonsten als `abstract`deklariert werden.</span><span class="sxs-lookup"><span data-stu-id="dd041-105">A class must implement all members of interfaces from which it derives or else be declared `abstract`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="dd041-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="dd041-106">Example</span></span>  
 <span data-ttu-id="dd041-107">Im folgenden Beispiel wird CS0535 generiert:</span><span class="sxs-lookup"><span data-stu-id="dd041-107">The following sample generates CS0535.</span></span>  
  
```csharp  
// CS0535.cs  
public interface A  
{  
   void F();  
}  
  
public class B : A {}   // CS0535 A::F is not implemented  
  
// OK  
public class C : A {  
   public void F() {}  
   public static void Main() {}  
}  
```
