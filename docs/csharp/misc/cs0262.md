# <a name="compiler-error-cs0262"></a>Compilerfehler CS0262
Partielle Deklarationen von "Typ" haben Zugriffsmodifizierer, die miteinander in Konflikt stehen.  
  
 Dieser Fehler tritt auf, wenn ein partieller Typ 端ber inkonsistente Modifizierer wie "public", "private", "protected", "internal" oder "abstract" verf端gt. Diese Modifizierer m端ssen in allen partiellen Deklarationen f端r diesen Typ einheitlich sein. Weitere Informationen finden Sie unter [Partielle Klassen und Methoden](../../csharp/programming-guide/classes-and-structs/partial-classes-and-methods.md).  
  
## <a name="example"></a>Beispiel  
 Im folgenden Beispiel wird CS0262 generiert:  
  
```csharp  
// CS0262.cs  
class A  
{  
    public partial class C  // CS0262  
    {  
    }  
    private partial class C  
    {  
    }  
}  
```
