# <a name="compiler-error-cs1621"></a><span data-ttu-id="52ce9-102">Compilerfehler CS1621</span><span class="sxs-lookup"><span data-stu-id="52ce9-102">Compiler Error CS1621</span></span>
<span data-ttu-id="52ce9-103">Die yield-Anweisung kann nicht in einer anonymen Methode oder einem Lambda-Ausdruck verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="52ce9-103">The yield statement cannot be used inside an anonymous method or lambda expression</span></span>  
  
<span data-ttu-id="52ce9-104">Die [Yield](../language-reference/keywords/yield.md) -Anweisung kann nicht in einer [anonymen Methode](../language-reference/operators/delegate-operator.md) oder einem [Lambda-Ausdruck](../programming-guide/statements-expressions-operators/lambda-expressions.md)verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="52ce9-104">You cannot use the [yield](../language-reference/keywords/yield.md) statement in an [anonymous method](../language-reference/operators/delegate-operator.md) or a [lambda expression](../programming-guide/statements-expressions-operators/lambda-expressions.md).</span></span>
  
<span data-ttu-id="52ce9-105">Im folgenden Beispiel wird CS1621 generiert:</span><span class="sxs-lookup"><span data-stu-id="52ce9-105">The following sample generates CS1621:</span></span>
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
