# <a name="compiler-error-cs1621"></a>Compilerfehler CS1621
Die yield-Anweisung kann nicht in einer anonymen Methode oder einem Lambda-Ausdruck verwendet werden.  
  
Die [Yield](../../csharp/language-reference/keywords/yield.md) -Anweisung kann nicht in einer [anonymen Methode](../language-reference/operators/delegate-operator.md) oder einem [Lambda-Ausdruck](../../csharp/programming-guide/statements-expressions-operators/lambda-expressions.md)verwendet werden.
  
Im folgenden Beispiel wird CS1621 generiert:
  
```csharp  
// CS1621.cs  
  
using System.Collections;  
  
delegate object MyDelegate();  
  
class C : IEnumerable  
{  
    public IEnumerator GetEnumerator()  
    {  
        MyDelegate d = delegate  
        {  
            yield return this; // CS1621  
            return this;  
        };  
        d();  
        // Try this instead:  
        // MyDelegate d = delegate { return this; };  
        // yield return d();  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
