# <a name="compiler-error-cs1530"></a>Compilerfehler CS1530
Das Schlüsselwort 'new' ist für Elemente, die in einem Namespace definiert sind, nicht zulässig.  
  
 Das Schlüsselwort [new](../../csharp/language-reference/keywords/new-modifier.md) muss für Konstrukte in einem [Namespace](../../csharp/language-reference/keywords/namespace.md)nicht angegeben werden.  
  
 Im folgenden Beispiel wird CS1530 generiert:  
  
```csharp  
// CS1530.cs  
namespace a  
{  
   new class i   // CS1530  
   {  
   }  
  
   // try the following instead  
   class ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
