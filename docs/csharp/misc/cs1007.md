# <a name="compiler-error-cs1007"></a>Compilerfehler CS1007
Der Eigenschaftenaccessor ist bereits definiert.  
  
 Wenn Sie eine [Eigenschaft](../programming-guide/classes-and-structs/using-properties.md)deklarieren, m√ºssen Sie auch ihre Accessormethoden deklarieren. Eine Eigenschaft kann jedoch nicht mehr als eine `get` -Accessormethode oder nicht mehr als eine `set` -Accessormethode aufweisen.  
  
## <a name="example"></a>Beispiel  
 Im folgenden Beispiel wird CS1007 generiert:  
  
```csharp  
// CS1007.cs  
public class clx  
{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
