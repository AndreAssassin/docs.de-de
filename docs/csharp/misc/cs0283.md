## <a name="example"></a><span data-ttu-id="d21cb-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="d21cb-106">Example</span></span>  
 <span data-ttu-id="d21cb-107">Im folgenden Beispiel wird der Fehler CS0283 generiert.</span><span class="sxs-lookup"><span data-stu-id="d21cb-107">The following example generates CS0283.</span></span>

```csharp  
{  
    // To resolve the error but retain the "const-ness",

    }  
}  
```
