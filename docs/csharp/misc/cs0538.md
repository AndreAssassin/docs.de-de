# <a name="compiler-error-cs0538"></a><span data-ttu-id="12712-102">Compilerfehler CS0538</span><span class="sxs-lookup"><span data-stu-id="12712-102">Compiler Error CS0538</span></span>
<span data-ttu-id="12712-103">"Name" in der expliziten Schnittstellendeklaration ist keine Schnittstelle.</span><span class="sxs-lookup"><span data-stu-id="12712-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="12712-104">Es wurde versucht, explizit eine [Schnittstelle](../../csharp/language-reference/keywords/interface.md)zu deklarieren, die nicht angegeben ist.</span><span class="sxs-lookup"><span data-stu-id="12712-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="12712-105">Im folgenden Beispiel wird CS0538 generiert:</span><span class="sxs-lookup"><span data-stu-id="12712-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
