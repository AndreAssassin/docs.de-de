# <a name="compiler-error-cs0743"></a><span data-ttu-id="8ad27-102">Compilerfehler CS0743</span><span class="sxs-lookup"><span data-stu-id="8ad27-102">Compiler Error CS0743</span></span>
<span data-ttu-id="8ad27-103">Kontextbezogenes Schlüsselwort "on" erwartet.</span><span class="sxs-lookup"><span data-stu-id="8ad27-103">Expected contextual keyword 'on'</span></span>  
  
 <span data-ttu-id="8ad27-104">Das Muster für eine `join` -Klausel ist `join`...`in`...`on`...`equals`, wie im folgenden Beispiel gezeigt:</span><span class="sxs-lookup"><span data-stu-id="8ad27-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="8ad27-105">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="8ad27-105">To correct this error</span></span>  
  
1. <span data-ttu-id="8ad27-106">Fügen Sie der `on` -Klausel das `join` Schlüsselwort hinzu.</span><span class="sxs-lookup"><span data-stu-id="8ad27-106">Add the `on` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8ad27-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="8ad27-107">Example</span></span>  
 <span data-ttu-id="8ad27-108">Mit dem folgenden Code wird CS0743 generiert:</span><span class="sxs-lookup"><span data-stu-id="8ad27-108">The following code generates CS0743:</span></span>  
  
```csharp  
// cs0743.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3, 4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 x equals y // CS0743  
                select x;  
        return 1;  
    }  
}  
```
