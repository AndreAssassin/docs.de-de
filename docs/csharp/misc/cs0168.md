# <a name="compiler-warning-level-3-cs0168"></a>Compilerwarnung (Stufe 3) CS0168
Die Variable "Var" ist deklariert, aber nie verwendet.

Der Compiler gibt eine Warnung der Stufe 3 aus, wenn Sie eine Variable deklarieren, aber nicht verwenden.
  
Im folgende Beispiel wird eine Warnung CS0168 generiert:  
  
```csharp
// CS0168.cs  
// compile with: /W:3  
public class clx
{
    public int i;
}

public class clz
{
    public static void Main() {
        clx a; // CS0168, the variable 'a' is declared but never used
        // try the following line instead
        // clx a = new clx();  // this does not generate a warning because the clx constructor must execute.
    }
}
```
