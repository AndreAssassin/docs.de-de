 <span data-ttu-id="6dd88-106">Im folgenden Beispiel wird CS0642 generiert:</span><span class="sxs-lookup"><span data-stu-id="6dd88-106">The following sample generates CS0642:</span></span>  
  
```csharp

{  
   public static void Main()  
   {  
      int i;  
  
      for (i = 0; i < 10; i += 1);   // CS0642 semicolon intentional?

         System.Console.WriteLine (i);  
      }  
   }  
}  
```
