## <a name="example"></a><span data-ttu-id="ab143-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="ab143-107">Example</span></span>

   public class B { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="ab143-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="ab143-108">Example</span></span>  
 <span data-ttu-id="ab143-109">Die zweite Datei erstellt eine DLL, die eine `C` -Klasse definiert, die von der `B` -Klasse abgeleitet wird, welche im vorherigen Beispiel generiert wurde.</span><span class="sxs-lookup"><span data-stu-id="ab143-109">The second file creates a DLL that defines a class `C` that is derived from the class `B` that was created in the previous example.</span></span>

// compile with: /target:library /reference:CS0011_1.dll  
// post-build command: del /f CS0011_1.dll

```  
  
## <a name="example"></a><span data-ttu-id="ab143-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="ab143-110">Example</span></span>
