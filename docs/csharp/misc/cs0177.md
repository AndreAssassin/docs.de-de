 <span data-ttu-id="bf3cd-104">Einem Parameter, der mit dem [out](../language-reference/keywords/out-parameter-modifier.md) -Schlüsselwort gekennzeichnet ist, wurde im Methodenkörper kein Wert zugewiesen.</span><span class="sxs-lookup"><span data-stu-id="bf3cd-104">A parameter marked with the [out](../language-reference/keywords/out-parameter-modifier.md) keyword was not assigned a value in the method body.</span></span> <span data-ttu-id="bf3cd-105">Weitere Informationen finden Sie unter [übergeben von Parametern](../programming-guide/classes-and-structs/passing-parameters.md) .</span><span class="sxs-lookup"><span data-stu-id="bf3cd-105">For more information, see [Passing Parameters](../programming-guide/classes-and-structs/passing-parameters.md)</span></span>
