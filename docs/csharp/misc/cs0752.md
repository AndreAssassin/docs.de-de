 <span data-ttu-id="5b558-104">Eine partielle Methode darf keinen [out](../language-reference/keywords/out-parameter-modifier.md) -Parameter enthalten.</span><span class="sxs-lookup"><span data-stu-id="5b558-104">A partial method cannot have an [out](../language-reference/keywords/out-parameter-modifier.md) parameter.</span></span> <span data-ttu-id="5b558-105">Out-Parameter sind nicht zulässig, da es beim Entfernen der partiellen Methode durch den Compiler keine Garantie gibt, dass der out-Parameter jemals zugewiesen wird.</span><span class="sxs-lookup"><span data-stu-id="5b558-105">Out parameters are not allowed because if the partial method is removed by the compiler then there is no guarantee that the out parameter is ever assigned.</span></span>
