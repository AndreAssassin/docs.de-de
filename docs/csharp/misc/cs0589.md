# <a name="compiler-error-cs0589"></a>Compilerfehler CS0589

Interner Compilerfehler: Stufe 'PARSE'

 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, weil er unerwartete Syntax nicht analysieren kann. Wenn dies nicht der Fall ist, wenden Sie [sich an uns](/visualstudio/ide/talk-to-us).
 
> [!NOTE]
> Dieser Compilerfehler wird in Roslyn nicht mehr verwendet.
