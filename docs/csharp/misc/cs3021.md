# <a name="compiler-warning-level-2-cs3021"></a><span data-ttu-id="c8731-102">Compilerwarnung (Stufe 2) CS3021</span><span class="sxs-lookup"><span data-stu-id="c8731-102">Compiler Warning (level 2) CS3021</span></span>
<span data-ttu-id="c8731-103">"Typ" erfordert kein CLSCompliant-Attribut, da die Assembly kein CLSCompliant-Attribut aufweist.</span><span class="sxs-lookup"><span data-stu-id="c8731-103">'type' does not need a CLSCompliant attribute because the assembly does not have a CLSCompliant attribute</span></span>  
  
 <span data-ttu-id="c8731-104">Diese Warnung tritt auf `[CLSCompliant(false)]` , wenn für eine Klasse in einer Assembly, die kein clscompliance-Attribut auf Assemblyebene hat, auf true festgelegt ist `[assembly: CLSCompliant(true)]`(d. h. die Zeile).</span><span class="sxs-lookup"><span data-stu-id="c8731-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="c8731-105">Da sich die Assembly nicht selbst als CLS-kompatibel deklariert, besteht keine Notwendigkeit, dass sich ein Element in der Assembly selbst als nicht kompatibel deklariert, da angenommen wird, dass keine Kompatibilität besteht.</span><span class="sxs-lookup"><span data-stu-id="c8731-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="c8731-106">Weitere Informationen zur CLS-Kompatibilität finden Sie unter [Sprachunabhängigkeit und sprachunabhängige Komponenten](../../standard/language-independence.md).</span><span class="sxs-lookup"><span data-stu-id="c8731-106">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>
  
 <span data-ttu-id="c8731-107">Damit diese Warnung nicht mehr angezeigt wird, entfernen Sie das Attribut, oder fügen Sie das Attribut auf Assemblyebene hinzu.</span><span class="sxs-lookup"><span data-stu-id="c8731-107">To get rid of this warning, remove the attribute or add the assembly level attribute.</span></span>  
  
## <a name="example"></a><span data-ttu-id="c8731-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="c8731-108">Example</span></span>  
 <span data-ttu-id="c8731-109">Im folgenden Beispiel wird CS3021 generiert.</span><span class="sxs-lookup"><span data-stu-id="c8731-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
