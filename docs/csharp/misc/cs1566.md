# <a name="compiler-error-cs1566"></a>Compilerfehler CS1566
Fehler beim Lesen der Ressourcendatei 'datei' – 'ursache'  
  
 Der Compiler hatte ein Problem mit dem an die Compileroption [/resource](../language-reference/compiler-options/resource-compiler-option.md) übergebenen Dateinamen.
