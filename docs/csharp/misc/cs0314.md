## <a name="example"></a><span data-ttu-id="f2b54-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="f2b54-108">Example</span></span>  
 <span data-ttu-id="f2b54-109">Mit dem folgenden Code wird CS0314 generiert:</span><span class="sxs-lookup"><span data-stu-id="f2b54-109">The following code generates CS0314:</span></span>  
  
```csharp  
// cs0314.cs  
// Compile with: /target:library  
public class ClassConstraint { }  
  
public class A<T> where T : ClassConstraint  
{ }  
  
public class B<T> : A<T> //CS0314  
{ }  
  
// Try using this instead.  
public class C<T> : A<T> where T : ClassConstraint  
{ }  
```
