# <a name="compiler-error-cs2012"></a>Compilerfehler CS2012
"Datei" kann nicht zum Schreiben geöffnet werden  
  
 Beim Verwenden der [-bugreport: `file`-](../language-reference/compiler-options/bugreport-compiler-option.md) Compileroption konnte die Datei nicht zum Schreiben geöffnet werden. Vergewissern Sie sich, dass Sie einen gültigen Dateinamen angegeben haben und die Datei nicht schreibgeschützt ist.
