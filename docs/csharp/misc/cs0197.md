<span data-ttu-id="3d54c-103">Das Übergeben von 'Argument' als Verweis oder Ausgabe oder das Übernehmen der Adresse kann zu einer Laufzeitausnahme führen, da es sich hierbei um ein Feld einer Marshal-by-reference-Klasse handelt.</span><span class="sxs-lookup"><span data-stu-id="3d54c-103">Passing 'argument' as ref or out or taking its address may cause a runtime exception because it is a field of a marshal-by-reference class</span></span>  
  
 <span data-ttu-id="3d54c-104">Jede Klasse, die direkt oder indirekt von <xref:System.MarshalByRefObject> abgeleitet wird, ist eine Marshal-by-reference-Klasse.</span><span class="sxs-lookup"><span data-stu-id="3d54c-104">Any class that derives, directly or indirectly, from <xref:System.MarshalByRefObject> is a marshal-by-reference class.</span></span> <span data-ttu-id="3d54c-105">Eine solche Klasse kann über Prozess- und Computergrenzen hinweg als Verweis gemarshallt werden.</span><span class="sxs-lookup"><span data-stu-id="3d54c-105">Such a class can be marshaled by reference across process and machine boundaries.</span></span> <span data-ttu-id="3d54c-106">Folglich können Instanzen dieser Klasse Proxys für Remoteobjekte sein.</span><span class="sxs-lookup"><span data-stu-id="3d54c-106">Thus, instances of this class could be proxies to remote objects.</span></span> <span data-ttu-id="3d54c-107">Sie können ein Feld eines Proxyobjekts nicht als [ref](../language-reference/keywords/ref.md) oder [out](../language-reference/keywords/out-parameter-modifier.md)übergeben. So, you cannot pass fields of such a class as `ref` or `out`, unless the instance is [this](../language-reference/keywords/this.md), which can not be a proxy object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3d54c-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="3d54c-108">Example</span></span>  
 <span data-ttu-id="3d54c-109">Im folgenden Beispiel wird CS0197 generiert.</span><span class="sxs-lookup"><span data-stu-id="3d54c-109">The following sample generates CS0197.</span></span>  
  
```csharp  
// CS0197.cs  
// compile with: /W:1  
class X : System.MarshalByRefObject  
{  
   public int i;  
}  
  
class M  
{  
   public int i;  
   static void AddSeventeen(ref int i)  
   {  
      i += 17;  
   }  
  
   static void Main()  
   {  
      X x = new X();  
      x.i = 12;  
      AddSeventeen(ref x.i);   // CS0197  
  
      // OK  
      M m = new M();  
      m.i = 12;  
      AddSeventeen(ref m.i);  
   }  
}  
```
