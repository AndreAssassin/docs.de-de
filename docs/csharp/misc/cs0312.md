## <a name="example"></a><span data-ttu-id="d15de-111">Beispiel</span><span class="sxs-lookup"><span data-stu-id="d15de-111">Example</span></span>  
 <span data-ttu-id="d15de-112">Mit dem folgenden Code wird CS0312 generiert:</span><span class="sxs-lookup"><span data-stu-id="d15de-112">The following code generates CS0312:</span></span>  
  
```csharp  
// cs0312.cs  
class Program  
{  
    static void MTyVar<T, U>() where T : U { }  
  
    static int Main()  
    {  
        MTyVar<int?, int>(); // CS0312  
        return 1;  
    }  
}  
```
