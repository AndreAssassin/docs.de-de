# <a name="compiler-error-cs1021"></a><span data-ttu-id="6b554-102">Compilerfehler CS1021</span><span class="sxs-lookup"><span data-stu-id="6b554-102">Compiler Error CS1021</span></span>

<span data-ttu-id="6b554-103">Die integrale Konstante ist zu groß.</span><span class="sxs-lookup"><span data-stu-id="6b554-103">Integral constant is too large</span></span>  
  
<span data-ttu-id="6b554-104">Ein Wert, der durch ein Ganzzahlliteral dargestellt wird, ist größer als <xref:System.UInt64.MaxValue?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="6b554-104">A value represented by an integer literal is greater than <xref:System.UInt64.MaxValue?displayProperty=nameWithType>.</span></span>  
  
<span data-ttu-id="6b554-105">Im folgenden Beispiel wird CS1021 generiert:</span><span class="sxs-lookup"><span data-stu-id="6b554-105">The following sample generates CS1021:</span></span>  

```csharp
// CS1021.cs  
class Program
{
    static void Main(string[] args)
    {
        int a = 18_446_744_073_709_552_000;
    }
}  
```

<span data-ttu-id="6b554-106">Der folgende Code generiert auch CS1021:</span><span class="sxs-lookup"><span data-stu-id="6b554-106">The following code also generates CS1021:</span></span>

```csharp
using System.Numerics;

class Program
{
    static void Main(string[] args)
    {
        var a = new BigInteger(18_446_744_073_709_552_000);
    }
}
```
