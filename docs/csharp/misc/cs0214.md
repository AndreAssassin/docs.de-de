# <a name="compiler-error-cs0214"></a><span data-ttu-id="41e25-102">Compilerfehler CS0214</span><span class="sxs-lookup"><span data-stu-id="41e25-102">Compiler Error CS0214</span></span>
<span data-ttu-id="41e25-103">Zeiger und Puffer fester Größe können nur in einem unsicheren Kontext verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="41e25-103">Pointers and fixed size buffers may only be used in an unsafe context</span></span>  
  
 <span data-ttu-id="41e25-104">Zeiger können nur mit dem [unsafe](../language-reference/keywords/unsafe.md) -Schlüsselwort verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="41e25-104">Pointers can only be used with the [unsafe](../language-reference/keywords/unsafe.md) keyword.</span></span> <span data-ttu-id="41e25-105">Weitere Informationen finden Sie unter [Unsicherer Code und Zeiger](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="41e25-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
 <span data-ttu-id="41e25-106">Im folgenden Beispiel wird CS0214 generiert:</span><span class="sxs-lookup"><span data-stu-id="41e25-106">The following sample generates CS0214:</span></span>  
  
```csharp  
// CS0214.cs  
// compile with: /target:library /unsafe  
public struct S  
{  
   public int a;  
}  
  
public class MyClass  
{  
   public static void Test()  
   {  
      S s = new S();  
      S * s2 = &s;    // CS0214  
      s2->a = 3;      // CS0214  
      s.a = 0;  
   }  
  
   // OK  
   unsafe public static void Test2()  
   {  
      S s = new S();  
      S * s2 = &s;  
      s2->a = 3;  
      s.a = 0;  
   }  
}  
```
