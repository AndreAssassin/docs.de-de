<span data-ttu-id="2c9b8-103">Attribute können nicht für Typargumente verwendet werden, sondern nur für Typparameter.</span><span class="sxs-lookup"><span data-stu-id="2c9b8-103">Attributes cannot be used on type arguments, only on type parameters</span></span>  
  
 <span data-ttu-id="2c9b8-104">Dieser Fehler tritt auf, wenn Sie ein Attribut auf ein Typargument anwenden, das in einer Aufrufanweisung auftritt.</span><span class="sxs-lookup"><span data-stu-id="2c9b8-104">This error occurs when you apply an attribute to a type argument that occurs in an invocation statement.</span></span> <span data-ttu-id="2c9b8-105">Es ist zulässig, ein Attribut auf einen Typparameter in einer Klassen- oder Methodendeklarationsanweisung wie der folgenden anzuwenden:</span><span class="sxs-lookup"><span data-stu-id="2c9b8-105">It is acceptable to apply an attribute to a type parameter in a class or method declaration statement such as the following:</span></span>

 <span data-ttu-id="2c9b8-106">Dieser Fehler wird durch die folgende Codezeile generiert.</span><span class="sxs-lookup"><span data-stu-id="2c9b8-106">The following line of code will generate this error.</span></span> <span data-ttu-id="2c9b8-107">Es wird davon ausgegangen, dass die in der vorherigen Codezeile definierte Klasse `C`eine statische Methode namens `MyStaticMethod`enthält.</span><span class="sxs-lookup"><span data-stu-id="2c9b8-107">It is assumed that the class `C`, defined in the previous line of code, has a static method called `MyStaticMethod`.</span></span>

