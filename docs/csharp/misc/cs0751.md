# <a name="compiler-error-cs0751"></a><span data-ttu-id="e5335-102">Compilerfehler CS0751</span><span class="sxs-lookup"><span data-stu-id="e5335-102">Compiler Error CS0751</span></span>
<span data-ttu-id="e5335-103">Eine partielle Methode muss in einer partiellen Klasse oder Struktur deklariert sein.</span><span class="sxs-lookup"><span data-stu-id="e5335-103">A partial method must be declared in a partial class or partial struct</span></span>  
  
 <span data-ttu-id="e5335-104">Es ist nicht möglich, eine [partielle](../../csharp/language-reference/keywords/partial-method.md) Methode zu deklarieren, sofern diese nicht in einer partiellen Klasse oder Struktur gekapselt ist.</span><span class="sxs-lookup"><span data-stu-id="e5335-104">It is not possible to declare a [partial](../../csharp/language-reference/keywords/partial-method.md) method unless it is encapsulated inside a partial class or partial struct.</span></span>
