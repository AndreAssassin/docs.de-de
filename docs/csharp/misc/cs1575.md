# <a name="compiler-error-cs1575"></a><span data-ttu-id="bcffa-102">Compilerfehler CS1575</span><span class="sxs-lookup"><span data-stu-id="bcffa-102">Compiler Error CS1575</span></span>
<span data-ttu-id="bcffa-103">Ein stackalloc-Ausdruck erfordert [] nach Typ</span><span class="sxs-lookup"><span data-stu-id="bcffa-103">A stackalloc expression requires [] after type</span></span>  
  
 <span data-ttu-id="bcffa-104">Die Größe der angeforderten Zuordnung, mit [stackalloc](../language-reference/operators/stackalloc.md), muss in eckigen Klammern angegeben werden.</span><span class="sxs-lookup"><span data-stu-id="bcffa-104">The size of the requested allocation, with [stackalloc](../language-reference/operators/stackalloc.md), must be specified in square brackets.</span></span>  
  
 <span data-ttu-id="bcffa-105">Im folgenden Beispiel wird CS1575 generiert:</span><span class="sxs-lookup"><span data-stu-id="bcffa-105">The following sample generates CS1575:</span></span>  
  
```csharp  
// CS1575.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int *p = stackalloc int (30);   // CS1575  
      // try the following line instead  
      // int *p = stackalloc int [30];  
   }  
}  
```
