# <a name="compiler-error-cs1020"></a>Compilerfehler CS1020
Überladbarer binärer Operator erwartet.  
  
 Es wurde versucht, eine operatorüberladung definieren, aber der Operator war keine Überladbarer binärer Operator, der zwei Parameter übernimmt. Finden Sie in der Liste der Überladbare Operatoren, die [Überladbare Operatoren](../../csharp/language-reference/operators/operator-overloading.md#overloadable-operators) Teil der [operatorüberladung](../../csharp/language-reference/operators/operator-overloading.md) Artikel.
  
 Im folgenden Beispiel wird CS1020 generiert:  
  
```csharp  
// CS1020.cs  
public class iii  
{  
   public static int operator ++(iii aa, int bb)   // CS1020, change ++ to +  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
