 <span data-ttu-id="62d9e-106">Im folgenden Beispiel wird CS0157 generiert:</span><span class="sxs-lookup"><span data-stu-id="62d9e-106">The following sample generates CS0157:</span></span>

```csharp  
{

   {  
   }

      public static void Main()  
      {

         {  
         }

         {  
            return;   // CS0157, cannot leave finally clause  
         }  
      }  
   }  
}  
```
