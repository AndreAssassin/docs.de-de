# <a name="compiler-error-cs0082"></a>Compilerfehler CS0082
Der Typ "Typ" reserviert bereits einen Member namens "Name" mit den gleichen Parametertypen.  
  
 Eigenschaften zum Zeitpunkt der Kompilierung werden in Methoden mit `get_` und/oder `set_` vor dem Bezeichner Ã¼bersetzt. Wenn Sie eine eigene Methode definieren, die mit dem Methodennamen in Konflikt steht, wird ein Fehler generiert.  
  
## <a name="example"></a>Beispiel  
 Im folgenden Beispiel wird der Fehler CS0082 generiert:  
  
```csharp  
//cs0082.cs  
class MyClass  
{  
  
    //property  
    public int MyProp  
    {  
        get //CS0082  
        {  
            return 1;  
        }  
    }  
  
    //conflicting Get  
    public int get_MyProp()  
    {  
        return 2;  
    }  
  
    public static int Main()  
    {  
        return 1;  
    }  
}  
```
