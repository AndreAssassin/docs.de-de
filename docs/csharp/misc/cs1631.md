# <a name="compiler-error-cs1631"></a><span data-ttu-id="c0099-102">Compilerfehler CS1631</span><span class="sxs-lookup"><span data-stu-id="c0099-102">Compiler Error CS1631</span></span>
<span data-ttu-id="c0099-103">Mit "yield" kann im Text einer catch-Klausel kein Wert zurückgegeben werden.</span><span class="sxs-lookup"><span data-stu-id="c0099-103">Cannot yield a value in the body of a catch clause</span></span>  
  
 <span data-ttu-id="c0099-104">Die yield-Anweisung ist im Text einer catch-Klausel nicht zulässig.</span><span class="sxs-lookup"><span data-stu-id="c0099-104">The yield statement is not allowed from within the body of a catch clause.</span></span> <span data-ttu-id="c0099-105">Verschieben Sie die yield-Anweisung außerhalb der catch-Klausel, um diesen Fehler zu vermeiden.</span><span class="sxs-lookup"><span data-stu-id="c0099-105">To avoid this error, move the yield statement outside the body of the catch clause.</span></span>  
  
 <span data-ttu-id="c0099-106">Im folgenden Beispiel wird CS1631 generiert:</span><span class="sxs-lookup"><span data-stu-id="c0099-106">The following sample generates CS1631:</span></span>  
  
```csharp  
// CS1631.cs  
using System;  
using System.Collections;  
  
public class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()   
   {  
      try  
      {  
      }  
      catch(Exception e)  
      {  
        yield return this;  // CS1631  
      }  
   }    
  
   public static void Main()   
   {  
   }  
}  
```
