## <a name="example"></a><span data-ttu-id="0b4ca-105">Beispiel</span><span class="sxs-lookup"><span data-stu-id="0b4ca-105">Example</span></span>  
 <span data-ttu-id="0b4ca-106">Im folgenden Beispiel wird CS0655 generiert:</span><span class="sxs-lookup"><span data-stu-id="0b4ca-106">The following sample generates CS0655:</span></span>  
  
```csharp  
// CS0655.cs

{  
    // decimal is not valid attribute parameter type  
    public decimal d = 0;

}

// Try the following line instead:  
// [My(e = 0)]

{  
    public static void Main()  
    {  
    }  
}  
```
