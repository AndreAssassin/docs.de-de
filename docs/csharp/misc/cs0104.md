 <span data-ttu-id="8e265-104">Das Programm enthält [using](../language-reference/keywords/using.md) -Anweisungen für zwei Namespaces, und im Code wird auf einen Namen verwiesen, der in beiden Namespaces enthalten ist.</span><span class="sxs-lookup"><span data-stu-id="8e265-104">Your program contains [using](../language-reference/keywords/using.md) directives for two namespaces and your code references a name that appears in both namespaces.</span></span>
