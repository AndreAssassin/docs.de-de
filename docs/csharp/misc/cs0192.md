# <a name="compiler-error-cs0192"></a><span data-ttu-id="8b687-102">Compilerfehler CS0192</span><span class="sxs-lookup"><span data-stu-id="8b687-102">Compiler Error CS0192</span></span>

 <span data-ttu-id="8b687-104">Ein Feld (Variable), das mit dem [readonly](../../csharp/language-reference/keywords/readonly.md) -Schlüsselwort markiert ist, kann nur innerhalb eines Konstruktors an einen [ref](../../csharp/language-reference/keywords/ref.md) oder [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) -Parameter übergeben werden.</span><span class="sxs-lookup"><span data-stu-id="8b687-104">A field (variable) marked with the [readonly](../../csharp/language-reference/keywords/readonly.md) keyword cannot be passed either to a [ref](../../csharp/language-reference/keywords/ref.md) or [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) parameter except inside a constructor.</span></span> <span data-ttu-id="8b687-105">Weitere Informationen finden Sie unter [Felder](../../csharp/programming-guide/classes-and-structs/fields.md).</span><span class="sxs-lookup"><span data-stu-id="8b687-105">For more information, see [Fields](../../csharp/programming-guide/classes-and-structs/fields.md).</span></span>  
  
 <span data-ttu-id="8b687-106">CS0192 wird auch ausgegeben, wenn das `readonly` -Feld [static](../../csharp/language-reference/keywords/static.md) ist und der Konstruktor nicht als `static`markiert ist.</span><span class="sxs-lookup"><span data-stu-id="8b687-106">CS0192 also results if the `readonly` field is [static](../../csharp/language-reference/keywords/static.md) and the constructor is not marked `static`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="8b687-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="8b687-107">Example</span></span>  
 <span data-ttu-id="8b687-108">Im folgenden Beispiel wird CS0192 generiert:</span><span class="sxs-lookup"><span data-stu-id="8b687-108">The following sample generates CS0192.</span></span>  
  
```csharp 
// CS0192.cs  
class MyClass  
{  
    public readonly int TestInt = 6;  
    static void TestMethod(ref int testInt)  
    {  
        testInt = 0;  
    }  
  
    MyClass()  
    {  
        TestMethod(ref TestInt);   // OK  
    }  
  
    public void PassReadOnlyRef()  
    {  
        TestMethod(ref TestInt);   // CS0192  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
