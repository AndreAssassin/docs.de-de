# <a name="compiler-error-cs0193"></a><span data-ttu-id="928f5-102">Compilerfehler CS0193</span><span class="sxs-lookup"><span data-stu-id="928f5-102">Compiler Error CS0193</span></span>

<span data-ttu-id="928f5-103">Der \* or->-Operator muss auf einen Zeiger angewendet werden.</span><span class="sxs-lookup"><span data-stu-id="928f5-103">The \* or -> operator must be applied to a pointer</span></span>  
  
<span data-ttu-id="928f5-104">Der \* Operator oder-> wurde mit einem nicht Zeigertyp verwendet.</span><span class="sxs-lookup"><span data-stu-id="928f5-104">The \* or -> operator was used with a nonpointer type.</span></span> <span data-ttu-id="928f5-105">Weitere Informationen finden Sie unter [Zeigertypen](../programming-guide/unsafe-code-pointers/pointer-types.md).</span><span class="sxs-lookup"><span data-stu-id="928f5-105">For more information, see [Pointer types](../programming-guide/unsafe-code-pointers/pointer-types.md).</span></span>  
  
 <span data-ttu-id="928f5-106">Im folgenden Beispiel wird CS0193 generiert:</span><span class="sxs-lookup"><span data-stu-id="928f5-106">The following sample generates CS0193:</span></span>  
  
```csharp  
// CS0193.cs  
using System;  
  
public struct Age  
{  
   public int AgeYears;  
   public int AgeMonths;  
   public int AgeDays;  
}  
  
public class MyClass  
{  
   public static void SetAge(ref Age anAge, int years, int months, int days)  
   {  
      anAge->Months = 3;   // CS0193, anAge is not a pointer  
      // try the following line instead  
      // anAge.AgeMonths = 3;  
   }  
  
   public static void Main()  
   {  
      Age MyAge = new Age();  
      Console.WriteLine(MyAge.AgeMonths);  
      SetAge(ref MyAge, 22, 4, 15);  
      Console.WriteLine(MyAge.AgeMonths);  
   }  
}  
```
