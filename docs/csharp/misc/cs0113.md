# <a name="compiler-error-cs0113"></a>Compilerfehler CS0113
Ein Member "Funktion", der als "override" markiert ist, kann nicht als "new" oder "virtual" markiert werden.  
  
 Die zum Markieren einer Methode verwendeten Schlüsselwörter [new](../../csharp/language-reference/keywords/new-modifier.md) und [override](../../csharp/language-reference/keywords/override.md) schließen sich gegenseitig aus.  
  
 Im folgenden Beispiel wird CS0113 generiert:  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
