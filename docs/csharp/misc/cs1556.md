# <a name="compiler-error-cs1556"></a>Compilerfehler CS1556
"Konstrukt", angegeben für die Main-Methode, muss eine gültige Klasse oder Struktur sein.  
  
 An die [/main](../language-reference/compiler-options/main-compiler-option.md) -Compileroption wurde ein Bezeichner übergeben, der kein Klassenname war.
