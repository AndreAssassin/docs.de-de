## <a name="example"></a><span data-ttu-id="11e38-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="11e38-108">Example</span></span>  
 <span data-ttu-id="11e38-109">Der folgende Code generiert CS0738, da die Klassenmethode `void` und der Schnittstellenmember mit dem gleichen Namen `int`zurückgibt:</span><span class="sxs-lookup"><span data-stu-id="11e38-109">The following code generates CS0738 because the class method returns `void` and the interface member of the same name returns `int`:</span></span>

{  
    int TestMethod();  
}
