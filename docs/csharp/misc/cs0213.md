 <span data-ttu-id="1c04b-104">Eine lokale Variable in einer [unsafe](../language-reference/keywords/unsafe.md) -Methode oder einem Parameter ist bereits "fixed" (auf dem Stapel), sodass Sie die Adresse keiner dieser beiden Variablen in einem [fixed](../language-reference/keywords/fixed-statement.md) -Ausdruck verwenden können.</span><span class="sxs-lookup"><span data-stu-id="1c04b-104">A local variable in an [unsafe](../language-reference/keywords/unsafe.md) method or a parameter is already fixed (on the stack), so you cannot take the address of either of these two variables in a [fixed](../language-reference/keywords/fixed-statement.md) expression.</span></span> <span data-ttu-id="1c04b-105">Weitere Informationen finden Sie unter [Unsicherer Code und Zeiger](../programming-guide/unsafe-code-pointers/index.md).</span><span class="sxs-lookup"><span data-stu-id="1c04b-105">For more information, see [Unsafe Code and Pointers](../programming-guide/unsafe-code-pointers/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="1c04b-106">Beispiel</span><span class="sxs-lookup"><span data-stu-id="1c04b-106">Example</span></span>  
 <span data-ttu-id="1c04b-107">Im folgenden Beispiel wird CS0213 generiert.</span><span class="sxs-lookup"><span data-stu-id="1c04b-107">The following sample generates CS0213.</span></span>  
  
```csharp  
// CS0213.cs  
// compile with: /unsafe  
public class MyClass  
{  
   unsafe public static void Main()  
   {  
      int i = 45;  
      fixed (int *j = &i) { }  // CS0213  
      // try the following line instead  
      // int* j = &i;  
  
      int[] a = new int[] {1,2,3};  
      fixed (int *b = a)  
      {  
         fixed (int *c = b) { }  // CS0213  
         // try the following line instead  
         // int *c = b;  
      }  
   }  
}  
```
