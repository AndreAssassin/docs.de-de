# <a name="compiler-error-cs1512"></a><span data-ttu-id="56c88-102">Compilerfehler CS1512</span><span class="sxs-lookup"><span data-stu-id="56c88-102">Compiler Error CS1512</span></span>
<span data-ttu-id="56c88-103">Das base-Schlüsselwort ist im aktuellen Kontext nicht verfügbar.</span><span class="sxs-lookup"><span data-stu-id="56c88-103">Keyword 'base' is not available in the current context</span></span>  
  
 <span data-ttu-id="56c88-104">Das [base](../../csharp/language-reference/keywords/base.md) -Schlüsselwort wurde außerhalb einer Methode, Eigenschaft oder eines Konstruktors verwendet.</span><span class="sxs-lookup"><span data-stu-id="56c88-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>  
  
 <span data-ttu-id="56c88-105">Im folgenden Beispiel wird der Fehler CS1512 generiert:</span><span class="sxs-lookup"><span data-stu-id="56c88-105">The following example generates CS1512:</span></span>  
  
```csharp  
// CS1512.cs  
using System;  
  
class Base {}  
  
class CMyClass : Base  
{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
