# <a name="compiler-error-cs1503"></a>Compilerfehler CS1503
Argument "Zahl" kann nicht von TypA in TypB konvertiert werden.  
  
 Der Typ eines Arguments in einer Methode entspricht nicht dem Typ, der bei der Instanziierung der Klasse übergeben wurde. Dieser Fehler tritt in der Regel zusammen mit CS1502 auf. Nähere Informationen zum Beheben dieses Fehlers finden Sie unter [CS1502](../../csharp/language-reference/compiler-messages/cs1502.md) .
