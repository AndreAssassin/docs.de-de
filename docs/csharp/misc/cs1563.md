# <a name="compiler-error-cs1563"></a>Compilerfehler CS1563
Die Ausgabe "Ausgabedatei" hat keine Quelldateien.  
  
 Die Compileroption [/out](../language-reference/compiler-options/out-compiler-option.md) wurde angegeben, ohne dass danach Quellcodedateien folgen. Sie sollten **/out**entweder nicht verwenden oder nach **/out**die Quellcodedateien angeben.
