## <a name="example"></a><span data-ttu-id="af6cb-109">Beispiel</span><span class="sxs-lookup"><span data-stu-id="af6cb-109">Example</span></span>  
 <span data-ttu-id="af6cb-110">Durch den folgenden Code wird der Fehler CS1947 ausgelöst:</span><span class="sxs-lookup"><span data-stu-id="af6cb-110">The following code generates CS1947:</span></span>

class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };
