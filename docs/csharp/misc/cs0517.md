# <a name="compiler-error-cs0517"></a>Compilerfehler CS0517
"Klasse" hat keine Basisklasse und kann keinen Basiskonstruktor aufrufen.  
  
 CS0517 tritt nur dann auf, wenn die Common Language Runtime von .NET Framework den Quellcode für die Objektklasse kompiliert, die die einzige Klasse ist, für die keine Basisklasse vorhanden ist.
