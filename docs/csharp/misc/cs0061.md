 <span data-ttu-id="7835a-104">Ein [public](../language-reference/keywords/public.md) -Konstrukt muss ein Objekt zurückgeben, auf das öffentlich zugegriffen werden kann.</span><span class="sxs-lookup"><span data-stu-id="7835a-104">A [public](../language-reference/keywords/public.md) construct must return a publicly accessible object.</span></span>

 <span data-ttu-id="7835a-107">Das folgende Beispiel generiert CS0061:</span><span class="sxs-lookup"><span data-stu-id="7835a-107">The following sample generates CS0061.</span></span>  
  
```csharp  
// CS0061.cs

// OK

internal interface C {}  
internal interface CC : C {}  
```
