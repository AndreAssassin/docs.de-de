# <a name="compiler-error-cs0619"></a><span data-ttu-id="4e947-102">Compilerfehler CS0619</span><span class="sxs-lookup"><span data-stu-id="4e947-102">Compiler Error CS0619</span></span>

 <span data-ttu-id="4e947-104">Ein Klassenmember wurde mit dem [veraltet] markiert ((.. /.. / csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) Attribut ein Fehler ausgegeben wird, wenn der Verweis auf den Klassenmember.</span><span class="sxs-lookup"><span data-stu-id="4e947-104">A class member was marked with the [Obsolete]((../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="4e947-105">Im folgenden Beispiel wird der Fehler CS0619 generiert:</span><span class="sxs-lookup"><span data-stu-id="4e947-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
