# <a name="compiler-error-cs0619"></a><span data-ttu-id="93cfa-102">Compilerfehler CS0619</span><span class="sxs-lookup"><span data-stu-id="93cfa-102">Compiler Error CS0619</span></span>
<span data-ttu-id="93cfa-103">Member ist veraltet: "Text"</span><span class="sxs-lookup"><span data-stu-id="93cfa-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="93cfa-104">Ein Klassenmember wurde mit dem [veralteten] (() markiert. /Programming-Guide/Concepts/Attributes/Common-Attributes.MD # Obsolete)-Attribut, sodass beim Verweis auf den Klassenmember ein Fehler ausgegeben wird.</span><span class="sxs-lookup"><span data-stu-id="93cfa-104">A class member was marked with the [Obsolete]((../programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="93cfa-105">Im folgenden Beispiel wird der Fehler CS0619 generiert:</span><span class="sxs-lookup"><span data-stu-id="93cfa-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
