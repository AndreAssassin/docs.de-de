 <span data-ttu-id="56e2f-104">Der Fehler tritt auf, wenn Sie einen Ausdruck schreiben, der den Standardwert einer Variablen des generischen Typs enthält, die ein Verweistyp (z. {  
```csharp  
    public static void GenericClass<T>(T t1) where T : class

```
