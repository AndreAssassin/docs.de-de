## <a name="example"></a>Beispiel  
 Im folgenden Beispiel wird CS1723 generiert.

```csharp  
// CS1723.cs  
// compile with: /t:library /doc:filename.XML  
///<summary>A generic list class.</summary>  
///<see cref="T" />   // CS1723  
// To resolve comment the previous line.  
public class List<T>
{  
}  
```
