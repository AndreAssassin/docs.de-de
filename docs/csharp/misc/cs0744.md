<span data-ttu-id="4f5db-103">Kontextbezogenes Schlüsselwort "equals" erwartet.</span><span class="sxs-lookup"><span data-stu-id="4f5db-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="4f5db-104">Das Muster für eine `join` -Klausel ist `join`...`in`...`on`...`equals`, wie im folgenden Beispiel gezeigt:</span><span class="sxs-lookup"><span data-stu-id="4f5db-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

            join y in array2 on x equals y

## <a name="example"></a><span data-ttu-id="4f5db-107">Beispiel</span><span class="sxs-lookup"><span data-stu-id="4f5db-107">Example</span></span>  
 <span data-ttu-id="4f5db-108">Mit dem folgenden Code wird CS0744 generiert:</span><span class="sxs-lookup"><span data-stu-id="4f5db-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
