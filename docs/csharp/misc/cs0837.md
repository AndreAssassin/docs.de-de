# <a name="compiler-error-cs0837"></a><span data-ttu-id="7bb23-102">Compilerfehler CS0837</span><span class="sxs-lookup"><span data-stu-id="7bb23-102">Compiler Error CS0837</span></span>
<span data-ttu-id="7bb23-103">Der erste Operand des "is"- oder "as"-Operators darf kein Lambda-Ausdruck und keine anonyme Methode sein.</span><span class="sxs-lookup"><span data-stu-id="7bb23-103">The first operand of an "is" or "as" operator may not be a lambda expression or anonymous method.</span></span>  
  
 <span data-ttu-id="7bb23-104">Lambda-Ausdrücke und anonyme Methoden dürfen nicht auf der linken Seite von [is](../language-reference/operators/type-testing-and-cast.md#is-operator) oder [as](../language-reference/operators/type-testing-and-cast.md#as-operator)verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="7bb23-104">Lambda expressions and anonymous methods may not be used on the left side of [is](../language-reference/operators/type-testing-and-cast.md#is-operator) or [as](../language-reference/operators/type-testing-and-cast.md#as-operator).</span></span>
  
## <a name="to-correct-this-error"></a><span data-ttu-id="7bb23-105">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="7bb23-105">To correct this error</span></span>  
  
- <span data-ttu-id="7bb23-106">Wenn der Fehler bei einem `is` -Operator auftritt, müssen Sie bedenken, dass `is` einen Wert und einen Typ annimmt und Ihnen mitteilt, ob der Wert mittels Verweis-, Boxing- oder Unboxing-Konvertierung in diesen Typ umgewandelt werden kann.</span><span class="sxs-lookup"><span data-stu-id="7bb23-106">If the error involves the `is` operator, remember that `is` takes a value and a type and tells you whether the value can be made into that type by a reference, boxing, or unboxing conversion.</span></span> <span data-ttu-id="7bb23-107">Da Lambda-Ausdrücke keine Werte sind und keine Verweis-, Boxing- oder Unboxing-Konvertierungen aufweisen, können Lambda-Ausdrücke nicht mit `is`verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="7bb23-107">Because lambdas are not values and have no reference, boxing, or unboxing conversions, lambdas are not candidates for `is`.</span></span>  
  
- <span data-ttu-id="7bb23-108">Wenn im Code `as`falsch verwendet wird, ist die beste Lösung wahrscheinlich die Änderung in eine Umwandlung.</span><span class="sxs-lookup"><span data-stu-id="7bb23-108">If the code misuses `as`, the correction is probably to change it to a cast.</span></span>  
  
## <a name="example"></a><span data-ttu-id="7bb23-109">Beispiel</span><span class="sxs-lookup"><span data-stu-id="7bb23-109">Example</span></span>

<span data-ttu-id="7bb23-110">Im folgenden Beispiel wird CS0837 generiert:</span><span class="sxs-lookup"><span data-stu-id="7bb23-110">The following example generates CS0837:</span></span>  
  
```csharp  
// cs0837.cs  
namespace TestNamespace  
{  
    public delegate void Del();  
  
    class Test  
    {  
        static int Main()  
        {  
            bool b1 = (() => { }) is Del;   // CS0837  
            bool b2 = delegate() { } is Del;// CS0837  
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837  
            return 1;  
        }  
    }  
}  
```
