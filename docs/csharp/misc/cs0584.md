# <a name="compiler-error-cs0584"></a>Compilerfehler CS0584
Interner Compilerfehler: Stufe 'Stufe' Symbol 'Symbol'  
  
 Versuchen Sie festzustellen, ob der Compiler einen Fehler verursacht, da unerwartete Syntax nicht analysiert werden kann. Wenn dies nicht der Fall ist, versuchen Sie es [wenden Sie sich an uns](/visualstudio/ide/talk-to-us).
