# <a name="compiler-error-cs1676"></a><span data-ttu-id="a5e80-102">Compilerfehler CS1676</span><span class="sxs-lookup"><span data-stu-id="a5e80-102">Compiler Error CS1676</span></span>
<span data-ttu-id="a5e80-103">Der Parameter 'number' muss mit dem Schlüsselwort 'keyword' deklariert werden</span><span class="sxs-lookup"><span data-stu-id="a5e80-103">Parameter 'number' must be declared with the 'keyword' keyword</span></span>  
  
 <span data-ttu-id="a5e80-104">Dieser Fehler tritt auf, wenn der Parametertypmodifizierer in einer anonymen Methode nicht mit dem übereinstimmt, der in der Deklaration des Delegaten verwendet wird, in den Sie die Methode umwandeln.</span><span class="sxs-lookup"><span data-stu-id="a5e80-104">This error occurs when the parameter type modifier in an anonymous method is different from that used in the declaration of the delegate you are casting the method to.</span></span>  
  
 <span data-ttu-id="a5e80-105">Im folgenden Beispiel wird CS1676 generiert:</span><span class="sxs-lookup"><span data-stu-id="a5e80-105">The following sample generates CS1676:</span></span>  
  
```csharp  
// CS1676.cs  
delegate void E(ref int i);  
class Errors   
{  
   static void Main()  
   {  
      E e = delegate(out int i) { };   // CS1676  
      // To resolve, use the following line instead:  
      // E e = delegate(ref int i) { };  
   }  
}  
```
