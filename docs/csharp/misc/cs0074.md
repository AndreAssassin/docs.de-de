# <a name="compiler-error-cs0074"></a>Compilerfehler CS0074
'ereignis': Ein abstraktes Ereignis kann keinen Initialisierer haben.  
  
 Wenn ein [Ereignis](../../csharp/language-reference/keywords/event.md) als **abstrakt**markiert ist, kann es nicht initialisiert werden. Weitere Informationen finden Sie unter [Ereignisse](../../csharp/programming-guide/events/index.md)definiert sind.  
  
 Im folgenden Beispiel wird CS0074 generiert:  
  
```csharp  
// CS0074.cs  
delegate void D();  
  
abstract class Test  
{  
   public abstract event D e = null;   // CS0074  
   // try the following line instead  
   // public abstract event D e;  
  
   public static void Main()  
   {  
   }  
}  
```
