# <a name="compiler-error-cs0568"></a>Compilerfehler CS0568
Strukturen können keine expliziten parameterlosen Konstruktoren enthalten.  
  
 Jede [Struktur](../../csharp/language-reference/keywords/struct.md) besitzt bereits einen parameterlosen Konstruktor, der das Objekt, das 0 (null) initialisiert. Aus diesem Grund müssen die Konstruktoren, die Sie für eine Struktur erstellen, einen oder mehrere Parameter annehmen können.  
  
 Im folgenden Beispiel wird CS0568 generiert:  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
