# <a name="compiler-error-cs0112"></a><span data-ttu-id="2189a-102">Compilerfehler CS0112</span><span class="sxs-lookup"><span data-stu-id="2189a-102">Compiler Error CS0112</span></span>
<span data-ttu-id="2189a-103">Ein statischer Member "Funktion" kann nicht als "override", "virtual" oder "abstract" markiert werden.</span><span class="sxs-lookup"><span data-stu-id="2189a-103">A static member 'function' cannot be marked as override, virtual or abstract</span></span>  
  
 <span data-ttu-id="2189a-104">Eine Methodendeklaration, die das `override`-, **virtual**- oder **abstract** -Schlüsselwort verwendet, darf nicht auch das **static** -Schlüsselwort verwenden.</span><span class="sxs-lookup"><span data-stu-id="2189a-104">Any method declaration that uses the `override`, **virtual**, or **abstract** keyword cannot also use the **static** keyword.</span></span>  
  
 <span data-ttu-id="2189a-105">Weitere Informationen finden Sie unter [Methoden](../programming-guide/classes-and-structs/methods.md).</span><span class="sxs-lookup"><span data-stu-id="2189a-105">For more information, see [Methods](../programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="2189a-106">Im folgenden Beispiel wird CS0112 generiert:</span><span class="sxs-lookup"><span data-stu-id="2189a-106">The following sample generates CS0112:</span></span>  
  
```csharp  
// CS0112.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
   public class MyClass2 : MyClass  
   {  
      override public static void Foo()   // CS0112, remove static keyword  
      {  
      }  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
