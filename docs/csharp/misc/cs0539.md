<span data-ttu-id="8c128-103">"Member" in der expliziten Schnittstellendeklaration ist kein Member der Schnittstelle.</span><span class="sxs-lookup"><span data-stu-id="8c128-103">'member' in explicit interface declaration is not a member of interface</span></span>  
  
 <span data-ttu-id="8c128-104">Es wurde versucht, explizit ein [Schnittstellen](../../csharp/language-reference/keywords/interface.md) -Member zu deklarieren, das nicht vorhanden ist.</span><span class="sxs-lookup"><span data-stu-id="8c128-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md) member that does not exist.</span></span> <span data-ttu-id="8c128-105">Sie sollten die Deklaration entweder löschen oder so ändern, dass sie auf einen gültigen Schnittstellenmember verweist.</span><span class="sxs-lookup"><span data-stu-id="8c128-105">You should either delete the declaration or change it so that it refers to a valid interface member.</span></span>  
  
 <span data-ttu-id="8c128-106">Im folgenden Beispiel wird CS0539 generiert:</span><span class="sxs-lookup"><span data-stu-id="8c128-106">The following sample generates CS0539:</span></span>

```csharp  
   {  
      void m();  
   }  
  
   public class clx : I  
   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
