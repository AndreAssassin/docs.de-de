## <a name="example"></a><span data-ttu-id="cde44-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="cde44-108">Example</span></span>  
 <span data-ttu-id="cde44-109">Im folgenden Beispiel wird CS1666 generiert:</span><span class="sxs-lookup"><span data-stu-id="cde44-109">The following sample generates CS1666.</span></span>  
  
```csharp  
// CS1666.cs  
// compile with: /unsafe /target:library  
unsafe struct S  
{  
   public fixed int buffer[1];  
}  
  
unsafe class Test  
{  
   S field = new S();  
  
   private bool example1()  
   {  
      return (field.buffer[0] == 0);   // CS1666 error  
   }  
  
   private bool example2()  
   {  
      // OK  
      fixed (S* p = &field)  
      {  
         return (p->buffer[0] == 0);  
      }  
   }  
  
   private bool example3()  
   {  
      S local = new S();  
      return (local.buffer[0] == 0);
   }
}  
```
