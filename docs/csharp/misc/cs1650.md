# <a name="compiler-error-cs1650"></a><span data-ttu-id="cf204-102">Compilerfehler CS1650</span><span class="sxs-lookup"><span data-stu-id="cf204-102">Compiler Error CS1650</span></span>
<span data-ttu-id="cf204-103">Feldern des statischen schreibgeschützten Felds 'Bezeichner' kann nichts zugewiesen werden (Ausnahme: in einem statischen Konstruktor oder einem Variableninitialisierer).</span><span class="sxs-lookup"><span data-stu-id="cf204-103">Fields of static readonly field 'identifier' cannot be assigned to (except in a static constructor or a variable initializer)</span></span>  
  
 <span data-ttu-id="cf204-104">Dieser Fehler tritt auf, wenn versucht wird, ein Member eines schreibgeschützten und statischen Felds zu ändern, wo Änderungen nicht zulässig sind.</span><span class="sxs-lookup"><span data-stu-id="cf204-104">This error occurs when you attempt to modify a member of a field which is readonly and static where it is not allowed to be modified.</span></span> <span data-ttu-id="cf204-105">Um diesen Fehler zu beheben, beschränken Sie Zuweisungen an schreibgeschützte Felder auf den Konstruktor oder Variableninitialisierer, oder entfernen Sie das `readonly` -Schlüsselwort aus der Deklaration des Felds.</span><span class="sxs-lookup"><span data-stu-id="cf204-105">To resolve this error, limit assignments to readonly fields to the constructor or variable initializer, or remove the `readonly` keyword from the declaration of the field.</span></span>  
  
```csharp  
// CS1650.cs  
public struct Inner  
{  
    public int i;  
}  
  
class Outer  
{  
    public static readonly Inner inner = new Inner();  
}  
  
class D  
{  
    static void Main()  
    {  
        Outer.inner.i = 1;  // CS1650  
    }  
}  
```
