# <a name="compiler-error-cs0243"></a>Compilerfehler CS0243

Das Conditional-Attribut ist für 'Methode' nicht gültig, da es eine Überschreibungsmethode ist.  
  
 Das [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) -Attribut für für Methoden nicht zulässig, die mit dem [override](../../csharp/language-reference/keywords/override.md) -Schlüsselwort gekennzeichnet sind. Weitere Informationen finden Sie unter [wann Override "und" neue Schlüsselwörter](../../csharp/programming-guide/classes-and-structs/knowing-when-to-use-override-and-new-keywords.md).  
  
 Der Compiler bindet niemals Überschreibungsmethoden. Er bindet nur die Basismethode und die Common Language Runtime ruft dann die Überschreibung entsprechend auf.  
  
 Im folgenden Beispiel wird CS0243 generiert:  

```csharp
// CS0243.cs  
// compile with: /target:library  
public class MyClass  
{  
   public virtual void M() {}  
}  
  
public class MyClass2 : MyClass  
{  
   [System.Diagnostics.ConditionalAttribute("MySymbol")]   // CS0243  
   // remove Conditional attribute or remove override keyword  
   public override void M() {}  
}  
```
