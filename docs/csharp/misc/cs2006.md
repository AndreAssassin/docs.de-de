# <a name="compiler-error-cs2006"></a>Compilerfehler CS2006
Befehlszeilensyntaxfehler: Fehlender 'Text', 'Option'-Option  
  
 Die Syntax für *Option* erfordert zusätzlichen Text. Informationen finden Sie unter [Compileroptionen](../../csharp/language-reference/compiler-options/index.md).
