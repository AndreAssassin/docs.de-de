# <a name="compiler-error-cs0037"></a>Compilerfehler CS0037
NULL kann nicht in 'typ' konvertiert werden, da dieser Werttyp nicht auf NULL festgelegt werden kann.  
  
 Der Compiler kann einem Werttyp nicht NULL zuweisen; NULL kann nur [Verweistypen](../language-reference/keywords/reference-types.md) oder Typen zugewiesen werden, die NULL-Werte zulassen. [Struktur](../language-reference/keywords/struct.md) ist ein Werttyp. Weitere Informationen finden Sie unter [Nullable-Typen](../programming-guide/nullable-types/index.md).  
  
 Im folgenden Beispiel wird CS0037 generiert:  
  
```csharp  
// CS0037.cs  
public struct s  
{  
}  
  
class a  
{  
   public static void Main()  
   {  
      int i = null;   // CS0037  
      s ss = null;    // CS0037  
   }  
}  
```
