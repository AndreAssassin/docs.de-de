# <a name="compiler-error-cs0035"></a><span data-ttu-id="363db-102">Compilerfehler CS0035</span><span class="sxs-lookup"><span data-stu-id="363db-102">Compiler Error CS0035</span></span>
<span data-ttu-id="363db-103">Der Operator „operator“ ist bei einem Operanden vom Typ „type“ mehrdeutig.</span><span class="sxs-lookup"><span data-stu-id="363db-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
<span data-ttu-id="363db-104">Der Compiler unterstützt verschiedene Konvertierungen und kann nicht entscheiden, welche er vor dem Anwenden des Operators auswählen soll.</span><span class="sxs-lookup"><span data-stu-id="363db-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span>
  
<span data-ttu-id="363db-105">Im folgenden Beispiel wird CS0035 generiert:</span><span class="sxs-lookup"><span data-stu-id="363db-105">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
