# <a name="compiler-error-cs2022"></a>Compilerfehler CS2022
Die Optionen /out und /target müssen vor den Quelldateinamen stehen.  
  
 Die Compileroptionen [/out (Ausgabedateinamen festlegen)](../language-reference/compiler-options/out-compiler-option.md) und [/target (Ausgabedateiformat festlegen)](../language-reference/compiler-options/target-compiler-option.md) müssen, wenn sie in der Befehlszeile angegeben werden, den Quellcodedateien vorangestellt sein.
