# <a name="compiler-error-cs0220"></a><span data-ttu-id="07f72-102">Compilerfehler CS0220</span><span class="sxs-lookup"><span data-stu-id="07f72-102">Compiler Error CS0220</span></span>
<span data-ttu-id="07f72-103">Vorgangsüberlauf zur Kompilierzeit im aktivierten Modus.</span><span class="sxs-lookup"><span data-stu-id="07f72-103">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="07f72-104">Ein Vorgang wurde von [checked](../../csharp/language-reference/keywords/checked.md)erkannt, wobei es sich um die Standardeinstellung handelt, und es ist ein Datenverlust aufgetreten.</span><span class="sxs-lookup"><span data-stu-id="07f72-104">An operation was detected by [checked](../../csharp/language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="07f72-105">Korrigieren Sie die Eingaben für die Zuweisung, oder verwenden Sie [unchecked](../../csharp/language-reference/keywords/unchecked.md) , um diesen Fehler zu beheben.</span><span class="sxs-lookup"><span data-stu-id="07f72-105">Either correct the inputs to the assignment or use [unchecked](../../csharp/language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="07f72-106">Weitere Informationen finden Sie unter [Checked und Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span><span class="sxs-lookup"><span data-stu-id="07f72-106">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="07f72-107">Im folgenden Beispiel wird CS0220 generiert:</span><span class="sxs-lookup"><span data-stu-id="07f72-107">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
