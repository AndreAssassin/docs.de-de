# <a name="compiler-error-cs0172"></a><span data-ttu-id="870af-102">Compilerfehler CS0172</span><span class="sxs-lookup"><span data-stu-id="870af-102">Compiler Error CS0172</span></span>
<span data-ttu-id="870af-103">Der Typ des bedingten Ausdrucks kann nicht bestimmt werden, da "Typ1" und "Typ2" implizit ineinander konvertiert werden.</span><span class="sxs-lookup"><span data-stu-id="870af-103">Type of conditional expression cannot be determined because 'type1' and 'type2' implicitly convert to one another</span></span>  
  
 <span data-ttu-id="870af-104">In einer bedingten Anweisung muss es möglich sein, die Typen auf beiden Seiten des `:` -Operators zu konvertieren.</span><span class="sxs-lookup"><span data-stu-id="870af-104">In a conditional statement, you must be able to convert the types on either side of the `:` operator.</span></span> <span data-ttu-id="870af-105">Darüber hinaus dürfen keine Routinen für eine gegenseitige Konvertierung vorhanden sein; Sie benötigen nur eine Konvertierung.</span><span class="sxs-lookup"><span data-stu-id="870af-105">Also, there cannot be mutual conversion routines; you only need one conversion.</span></span> <span data-ttu-id="870af-106">Weitere Informationen finden Sie unter [Konvertierungsoperatoren](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span><span class="sxs-lookup"><span data-stu-id="870af-106">For more information, see [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="870af-107">Im folgenden Beispiel wird CS0172 generiert:</span><span class="sxs-lookup"><span data-stu-id="870af-107">The following sample generates CS0172:</span></span>  
  
```csharp  
// CS0172.cs  
public class Square  
{  
   public class Circle  
   {  
      public static implicit operator Circle(Square aa)  
      {  
         return null;  
      }  
  
      public static implicit operator Square(Circle aa)  
      // using explicit resolves this error  
      // public static explicit operator Square(Circle aa)  
      {  
         return null;  
      }  
   }  
  
   public static void Main()  
   {  
      Circle aa = new Circle();  
      Square ii = new Square();  
      object o = (1 == 1) ? aa : ii;   // CS0172  
      // the following cast would resolve this error  
      // (1 == 1) ? aa : (Circle)ii;  
   }  
}  
```
