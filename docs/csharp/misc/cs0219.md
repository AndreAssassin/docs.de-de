<span data-ttu-id="515fd-103">Die Variable 'variable' ist zugewiesen, ihr Wert wird aber nie verwendet</span><span class="sxs-lookup"><span data-stu-id="515fd-103">The variable 'variable' is assigned but its value is never used</span></span>

 <span data-ttu-id="515fd-104">Der Compiler gibt eine Warnung der Stufe 3 aus, wenn Sie eine Variable deklarieren und zuweisen, aber nicht verwenden.</span><span class="sxs-lookup"><span data-stu-id="515fd-104">The compiler issues a level-three warning, when you declare and assign a variable, but do not use it.</span></span>

 > [!NOTE]
 > <span data-ttu-id="515fd-105">Der Compiler generiert diese Warnung nur, wenn der Variablenwert eine Kompilierungszeitkonstante ist.</span><span class="sxs-lookup"><span data-stu-id="515fd-105">The compiler generates this warning only when the variable value is a compile-time constant.</span></span> <span data-ttu-id="515fd-106">Das Zuweisen eines nicht konstanten Ausdrucks oder Methodenergebnisses zu einer lokalen Variablen erleichtert die Beobachtung dieser Ausdrücke im Debugger.</span><span class="sxs-lookup"><span data-stu-id="515fd-106">Assigning a non-constant expression or method result to a local variable makes it easier to observe those expressions in the debugger.</span></span> <span data-ttu-id="515fd-107">Außerdem kann das Ergebnis erreichbar sein und die Garbage Collection verhindert werden, während diese Variable erreichbar ist.</span><span class="sxs-lookup"><span data-stu-id="515fd-107">It also makes the result reachable, preventing garbage collection while that variable is reachable.</span></span>

 <span data-ttu-id="515fd-108">Das folgende Beispiel zeigt die Fälle, in denen die Warnung generiert wird:</span><span class="sxs-lookup"><span data-stu-id="515fd-108">The following sample shows the cases when and when not the warning is generated:</span></span>

```csharp
// CS0219.cs
// compile with: /W:3
public class MyClass
{
   public static void Main()
   {
      var interpolated = "Interpolated";
      var a = 0; // CS0219
      int b = GetZero(); // Doesn't generate a warning.
      var c = "Regular string"; // CS0219
      var d = $"Constant interpolated string"; // Doesn't generate a warning.
      var e = $"{interpolated} string"; // Doesn't generate a warning.
   }

   private static int GetZero()
   {
       return 0;
   }
}  
```
