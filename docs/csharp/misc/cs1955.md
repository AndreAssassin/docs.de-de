## <a name="example"></a><span data-ttu-id="b132f-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="b132f-108">Example</span></span>

<span data-ttu-id="b132f-109">Der folgende Code generiert CS1955, da der Code versucht, ein Feld und eine Eigenschaft mithilfe des [Aufrufausdrucks `()` ](../language-reference/operators/member-access-operators.md#invocation-expression-)aufzurufen.</span><span class="sxs-lookup"><span data-stu-id="b132f-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation expression `()`](../language-reference/operators/member-access-operators.md#invocation-expression-).</span></span> <span data-ttu-id="b132f-110">Sie können kein Feld oder eine Eigenschaft aufrufen.</span><span class="sxs-lookup"><span data-stu-id="b132f-110">You cannot call a field or a property.</span></span> <span data-ttu-id="b132f-111">Verwenden Sie den [Memberzugriffsausdruck, `.` ](../language-reference/operators/member-access-operators.md#member-access-expression-) um auf den gespeicherten Wert zuzugreifen.</span><span class="sxs-lookup"><span data-stu-id="b132f-111">Use the [member access expression `.`](../language-reference/operators/member-access-operators.md#member-access-expression-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
