## <a name="example"></a><span data-ttu-id="c61bf-108">Beispiel</span><span class="sxs-lookup"><span data-stu-id="c61bf-108">Example</span></span>

<span data-ttu-id="c61bf-109">Der folgende Code generiert Fehler CS1955 ausgelöst, da der Code versucht, ein Feld und eine Eigenschaft aufrufen, indem Sie mit der [Aufrufoperator `()` ](../language-reference/operators/member-access-operators.md#invocation-operator-).</span><span class="sxs-lookup"><span data-stu-id="c61bf-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation operator `()`](../language-reference/operators/member-access-operators.md#invocation-operator-).</span></span> <span data-ttu-id="c61bf-110">Sie können keinem Feld oder eine Eigenschaft aufrufen.</span><span class="sxs-lookup"><span data-stu-id="c61bf-110">You cannot call a field or a property.</span></span> <span data-ttu-id="c61bf-111">Verwenden der [Memberzugriffsoperator `.` ](../language-reference/operators/member-access-operators.md#member-access-operator-) Zugriff auf den Wert gespeichert.</span><span class="sxs-lookup"><span data-stu-id="c61bf-111">Use the [member access operator `.`](../language-reference/operators/member-access-operators.md#member-access-operator-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
