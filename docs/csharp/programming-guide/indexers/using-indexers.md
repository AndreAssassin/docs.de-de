# <a name="using-indexers-c-programming-guide"></a><span data-ttu-id="80eb0-102">Verwenden von Indexern (C#-Programmierhandbuch)</span><span class="sxs-lookup"><span data-stu-id="80eb0-102">Using indexers (C# Programming Guide)</span></span>

<span data-ttu-id="80eb0-103">Indexer sind ein syntaktisches Hilfsmittel, um Ihnen die Erstellung einer [Klasse](../../language-reference/keywords/class.md), einer [Struktur](../../language-reference/keywords/struct.md), oder einer [Schnittstelle](../../language-reference/keywords/interface.md) zu ermöglichen, auf die Clientanwendungen wie auf ein Array zugreifen können.</span><span class="sxs-lookup"><span data-stu-id="80eb0-103">Indexers are a syntactic convenience that enable you to create a [class](../../language-reference/keywords/class.md), [struct](../../language-reference/keywords/struct.md), or [interface](../../language-reference/keywords/interface.md) that client applications can access just as an array.</span></span> <span data-ttu-id="80eb0-104">Indexer werden am häufigsten in Typen implementiert, deren Hauptzweck darin besteht, eine interne Auflistung oder ein Array zu kapseln.</span><span class="sxs-lookup"><span data-stu-id="80eb0-104">Indexers are most frequently implemented in types whose primary purpose is to encapsulate an internal collection or array.</span></span> <span data-ttu-id="80eb0-105">Ein Beispiel: Sie verfügen über eine Klasse namens `TempRecord` zur Darstellung der Temperatur in Fahrenheit zu 10 verschiedenen Zeitpunkten während eines 24-stündigen Zeitraums.</span><span class="sxs-lookup"><span data-stu-id="80eb0-105">For example, suppose you have a class `TempRecord` that represents the temperature in Fahrenheit as recorded at 10 different times during a 24 hour period.</span></span> <span data-ttu-id="80eb0-106">Die Klasse enthält das Array `temps` des Typs `float[]`, um die Temperaturwerte zu speichern.</span><span class="sxs-lookup"><span data-stu-id="80eb0-106">The class contains an array `temps` of type `float[]` to store the temperature values.</span></span> <span data-ttu-id="80eb0-107">Durch die Implementierung eines Indexers in dieser Klasse können Clients auf die Temperaturen in einer `TempRecord`-Instanz als `float temp = tr[4]` zugreifen, statt als `float temp = tr.temps[4]`.</span><span class="sxs-lookup"><span data-stu-id="80eb0-107">By implementing an indexer in this class, clients can access the temperatures in a `TempRecord` instance as `float temp = tr[4]` instead of as `float temp = tr.temps[4]`.</span></span> <span data-ttu-id="80eb0-108">Die Angabe des Indexers vereinfacht nicht nur die Syntax für Clientanwendungen; die Klasse und Ihr Zweck können außerdem intuitiver von anderen Entwicklern verstanden werden.</span><span class="sxs-lookup"><span data-stu-id="80eb0-108">The indexer notation not only simplifies the syntax for client applications; it also makes the class and its purpose more intuitive for other developers to understand.</span></span>  
  
<span data-ttu-id="80eb0-109">Um einen Indexer in einer Klasse oder Struktur zu deklarieren, verwenden Sie das [this](../../language-reference/keywords/this.md) -Schlüsselwort, wie im folgenden Beispiel gezeigt:</span><span class="sxs-lookup"><span data-stu-id="80eb0-109">To declare an indexer on a class or struct, use the [this](../../language-reference/keywords/this.md) keyword, as the following example shows:</span></span>

## <a name="remarks"></a><span data-ttu-id="80eb0-110">Hinweise</span><span class="sxs-lookup"><span data-stu-id="80eb0-110">Remarks</span></span>

<span data-ttu-id="80eb0-111">Der Typ eines Indexers und der Typ seiner Parameter müssen zumindest dieselben Zugriffsmöglichkeiten bieten wie der Indexer selbst.</span><span class="sxs-lookup"><span data-stu-id="80eb0-111">The type of an indexer and the type of its parameters must be at least as accessible as the indexer itself.</span></span> <span data-ttu-id="80eb0-112">Weitere Informationen zu den Zugriffsebenen finden Sie unter [Zugriffsmodifizierer](../../language-reference/keywords/access-modifiers.md).</span><span class="sxs-lookup"><span data-stu-id="80eb0-112">For more information about accessibility levels, see [Access Modifiers](../../language-reference/keywords/access-modifiers.md).</span></span>

 <span data-ttu-id="80eb0-114">Die Signatur eines Indexers besteht aus der Anzahl und den Typen seiner formalen Parameter.</span><span class="sxs-lookup"><span data-stu-id="80eb0-114">The signature of an indexer consists of the number and types of its formal parameters.</span></span> <span data-ttu-id="80eb0-115">Sie umfasst weder den Indexertyp noch die Namen der formalen Parameter.</span><span class="sxs-lookup"><span data-stu-id="80eb0-115">It doesn't include the indexer type or the names of the formal parameters.</span></span> <span data-ttu-id="80eb0-116">Wenn Sie mehrere Indexer in derselben Klasse deklarieren, müssen sie verschiedene Signaturen aufweisen.</span><span class="sxs-lookup"><span data-stu-id="80eb0-116">If you declare more than one indexer in the same class, they must have different signatures.</span></span>  
  
 <span data-ttu-id="80eb0-117">Ein Indexerwert wird nicht als Variable klassifiziert.  <span data-ttu-id="80eb0-118">Um für den Indexer einen Namen anzugeben, den andere Sprachen verwenden können, verwenden Sie <xref:System.Runtime.CompilerServices.IndexerNameAttribute?displayProperty=nameWithType>, wie im folgenden Beispiel gezeigt:</span><span class="sxs-lookup"><span data-stu-id="80eb0-118">To provide the indexer with a name that other languages can use, use <xref:System.Runtime.CompilerServices.IndexerNameAttribute?displayProperty=nameWithType>, as the following example shows:</span></span>

```csharp
{
    // get and set accessors  
}  
```

<span data-ttu-id="80eb0-119">Dieser Indexer hat den Namen `TheItem`.</span><span class="sxs-lookup"><span data-stu-id="80eb0-119">This indexer will have the name `TheItem`.</span></span> <span data-ttu-id="80eb0-120">Wird kein Namensattribut bereitgestellt, wird `Item` zum Standardnamen.</span><span class="sxs-lookup"><span data-stu-id="80eb0-120">Not providing the name attribute would make `Item` the default name.</span></span>
