 <span data-ttu-id="cea66-109">Eine Instanz der Klasse ist ein [Verweistyp](../../language-reference/keywords/reference-types.md), kein Werttyp.</span><span class="sxs-lookup"><span <span data-ttu-id="cea66-111">Die aufgerufene Methode erhält also eine Kopie der Adresse der Instanz, während die aufrufende Methode die ursprüngliche Adresse der Instanz beibehält.</span><span instance.</span></span> <span data-ttu-id="cea66-112">Die Klasseninstanz in der aufrufenden Methode verfügt über eine Adresse, der Parameter in der aufgerufenen Methode verfügt über eine Kopie der method changes a class member, the original class instance in the calling method also changes.</span></span>
