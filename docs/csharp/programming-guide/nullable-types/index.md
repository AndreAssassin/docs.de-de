<span data-ttu-id="40bc3-104">Werttypen, die Nullwerte zulassen, sind Instanzen der <xref:System.Nullable%601?displayProperty=nameWithType>-Struktur.</span><span class="sxs-lookup"><span data-stu-id="40bc3-104">Nullable value types are instances of the <xref:System.Nullable%601?displayProperty=nameWithType> structure.</span></span> <span data-ttu-id="40bc3-105">Werttypen, die Nullwerte zulassen, können alle Werte eines zugrunde liegenden `T`-Typs und einen zusätzlichen [NULL](../../language-reference/keywords/null.md)-Wert darstellen.</span><span class="sxs-lookup"><span data-stu-id="40bc3-105">Nullable value types can represent all the values of an underlying type `T`, and an additional [null](../../language-reference/keywords/null.md) value.</span></span> <span data-ttu-id="40bc3-106">Der zugrunde liegende Typ `T` kann jeder [Werttyp](../../language-reference/keywords/value-types.md) sein, der keine NULL-Werte zulässt.</span><span class="sxs-lookup"><span data-stu-id="40bc3-106">The underlying type `T` can be any non-nullable [value type](../../language-reference/keywords/value-types.md).</span></span> <span data-ttu-id="40bc3-107">`T` kann kein Referenztyp sein.</span><span class="sxs-lookup"><span data-stu-id="40bc3-107">`T` cannot be a reference type.</span></span>

> [!NOTE]
> <span data-ttu-id="40bc3-108">C# 8.0 führt das Feature der Nullverweistypen ein.</span><span class="sxs-lookup"><span data-stu-id="40bc3-108">C# 8.0 introduces the nullable reference types feature.</span></span> <span data-ttu-id="40bc3-109">Weitere Informationen finden Sie unter [Nullverweistypen](../../nullable-references.md).</span><span class="sxs-lookup"><span data-stu-id="40bc3-109">For more information, see [Nullable reference types](../../nullable-references.md).</span></span> <span data-ttu-id="40bc3-110">Die Werttypen, die Nullwerte zulassen, sind ab C# 2 verfügbar.</span><span class="sxs-lookup"><span data-stu-id="40bc3-110">The nullable value types are available starting with C# 2.</span></span>

<span data-ttu-id="40bc3-111">Sie können beispielsweise `null` oder jeden anderen Integerwert von <xref:System.Int32.MinValue?displayProperty=nameWithType> bis <xref:System.Int32.MaxValue?displayProperty=nameWithType> einem `Nullable<int>`-, [TRUE](../../language-reference/keywords/true-literal.md)- und [FALSE](../../language-reference/keywords/false-literal.md)-Wert zuweisen, oder `null` zu `Nullable<bool>`.</span><span class="sxs-lookup"><span data-stu-id="40bc3-111">For example, you can assign `null` or any integer value from <xref:System.Int32.MinValue?displayProperty=nameWithType> to <xref:System.Int32.MaxValue?displayProperty=nameWithType> to a `Nullable<int>` and [true](../../language-reference/keywords/true-literal.md), [false](../../language-reference/keywords/false-literal.md), or `null` to a `Nullable<bool>`.</span></span>

<span data-ttu-id="40bc3-112">Sie verwenden einen Werttyp, der Nullwerte zulässt, wenn Sie den nicht definierten Wert eines zugrunde liegenden Typs darstellen müssen.</span><span class="sxs-lookup"><span data-stu-id="40bc3-112">You use a nullable value type when you need to represent the undefined value of an underlying type.</span></span> <span data-ttu-id="40bc3-113">Eine boolesche Variable darf nur zwei Werte besitzen: `true` und `false`.</span><span class="sxs-lookup"><span data-stu-id="40bc3-113">A Boolean variable can have only two values: `true` and `false`.</span></span> <span data-ttu-id="40bc3-114">Es gibt keinen „nicht definierten“ Wert.</span><span class="sxs-lookup"><span data-stu-id="40bc3-114">There is no "undefined" value.</span></span> <span data-ttu-id="40bc3-115">In den meisten Programmierungsanwendungen – allen voran Datenbankinteraktionen – kann ein Variablenwert in einem nicht definierten Zustand vorkommen oder fehlen.</span><span class="sxs-lookup"><span data-stu-id="40bc3-115">In many programming applications, most notably database interactions, a variable value can be undefined or missing.</span></span> <span data-ttu-id="40bc3-116">Ein Feld in einer Datenbank kann z.B. die Werte TRUE und FALSE enthalten; ebenso kann es aber auch gar keinen Wert enthalten.</span><span class="sxs-lookup"><span data-stu-id="40bc3-116">For example, a field in a database may contain the values true or false, or it may contain no value at all.</span></span> <span data-ttu-id="40bc3-117">In diesem Fall verwenden Sie einen `Nullable<bool>`-Typ.</span><span class="sxs-lookup"><span data-stu-id="40bc3-117">You use a `Nullable<bool>` type in that case.</span></span>

<span data-ttu-id="40bc3-118">Werttypen, die Nullwerte zulassen, weisen die folgenden Eigenschaften auf:</span><span class="sxs-lookup"><span data-stu-id="40bc3-118">Nullable value types have the following characteristics:</span></span>

- <span data-ttu-id="40bc3-119">Werttypen, die Nullwerte zulassen, stellen Werttypvariablen dar, denen der `null`-Wert zugewiesen werden kann.</span><span class="sxs-lookup"><span data-stu-id="40bc3-119">Nullable value types represent value-type variables that can be assigned the `null` value.</span></span>

- <span data-ttu-id="40bc3-120">Die Syntax `T?` ist eine Kurzform für `Nullable<T>`.</span><span class="sxs-lookup"><span data-stu-id="40bc3-120">The syntax `T?` is shorthand for `Nullable<T>`.</span></span> <span data-ttu-id="40bc3-121">Die beiden Formen sind austauschbar.</span><span class="sxs-lookup"><span data-stu-id="40bc3-121">The two forms are interchangeable.</span></span>

- <span data-ttu-id="40bc3-122">Sie weisen einem Werttyp, der Nullwerte zulässt, einen Wert genauso zu, wie Sie es für einen zugrunde liegenden Werttyp tun würden: `int? x = 10;` oder `double? d = 4.108;`.</span><span class="sxs-lookup"><span data-stu-id="40bc3-122">Assign a value to a nullable value type just as you would for an underlying value type: `int? x = 10;` or `double? d = 4.108;`.</span></span> <span data-ttu-id="40bc3-123">Sie können ebenfalls den `null`-Wert zuweisen: `int? x = null;`.</span><span class="sxs-lookup"><span data-stu-id="40bc3-123">You also can assign the `null` value: `int? x = null;`.</span></span>

- <span data-ttu-id="40bc3-124">Verwenden Sie die schreibgeschützten Eigenschaften <xref:System.Nullable%601.HasValue%2A?displayProperty=nameWithType> und <xref:System.Nullable%601.Value%2A?displayProperty=nameWithType>, um auf NULL zu prüfen und den Wert abzurufen, wie im folgenden Beispiel gezeigt: `if (x.HasValue) y = x.Value;`</span><span class="sxs-lookup"><span data-stu-id="40bc3-124">Use the <xref:System.Nullable%601.HasValue%2A?displayProperty=nameWithType> and <xref:System.Nullable%601.Value%2A?displayProperty=nameWithType> readonly properties to test for null and retrieve the value, as shown in the following example: `if (x.HasValue) y = x.Value;`</span></span>

  - <span data-ttu-id="40bc3-125">Die Eigenschaft <xref:System.Nullable%601.HasValue%2A> gibt `true` zurück, wenn die Variable einen Wert enthält, oder sie gibt `false` zurück, wenn die Variable `null` ist.</span><span class="sxs-lookup"><span data-stu-id="40bc3-125">The <xref:System.Nullable%601.HasValue%2A> property returns `true` if the variable contains a value, or `false` if it's `null`.</span></span>

  - <span data-ttu-id="40bc3-126">Die Eigenschaft <xref:System.Nullable%601.Value%2A> gibt einen Wert zurück, sofern <xref:System.Nullable%601.HasValue%2A> `true` zurückgibt.</span><span class="sxs-lookup"><span data-stu-id="40bc3-126">The <xref:System.Nullable%601.Value%2A> property returns a value if <xref:System.Nullable%601.HasValue%2A> returns `true`.</span></span> <span data-ttu-id="40bc3-127">Andernfalls wird eine <xref:System.InvalidOperationException> ausgelöst.</span><span class="sxs-lookup"><span data-stu-id="40bc3-127">Otherwise, an <xref:System.InvalidOperationException> is thrown.</span></span>

- <span data-ttu-id="40bc3-128">Sie können wie im folgenden Beispiel dargestellt auch die Operatoren `==` und `!=` mit einem Werttyp, der Nullwerte zulässt, verwenden: `if (x != null) y = x.Value;`.</span><span class="sxs-lookup"><span data-stu-id="40bc3-128">You can also use the `==` and `!=` operators with a nullable value type, as shown in the following example: `if (x != null) y = x.Value;`.</span></span> <span data-ttu-id="40bc3-129">Wenn `a` und `b` jeweils NULL sind, ergibt `a == b` `true`.</span><span class="sxs-lookup"><span data-stu-id="40bc3-129">If `a` and `b` are both null, `a == b` evaluates to `true`.</span></span>

- <span data-ttu-id="40bc3-130">Ab C# 7.0 können Sie den [Musterabgleich](../../pattern-matching.md#the-is-type-pattern-expression) verwenden, um einen Wert eines Nullable-Typs zu untersuchen und abzurufen: `if (x is int valueOfX) y = valueOfX;`.</span><span class="sxs-lookup"><span data-stu-id="40bc3-130">Beginning with C# 7.0, you can use [pattern matching](../../pattern-matching.md#the-is-type-pattern-expression) to both examine and get a value of a nullable type: `if (x is int valueOfX) y = valueOfX;`.</span></span>

- <span data-ttu-id="40bc3-131">Der Standardwert von `T?` ist eine Instanz, deren <xref:System.Nullable%601.HasValue%2A>-Eigenschaft `false` zurückgibt.</span><span class="sxs-lookup"><span data-stu-id="40bc3-131">The default value of `T?` is an instance whose <xref:System.Nullable%601.HasValue%2A> property returns `false`.</span></span>

- <span data-ttu-id="40bc3-132">Verwenden Sie die <xref:System.Nullable%601.GetValueOrDefault>-Methode, um entweder den zugewiesenen Wert oder den [Standardwert](../../language-reference/keywords/default-values-table.md) für den zugrunde liegenden Werttyp zurückzugeben, wenn der Wert des Nullable-Typs `null` ist.</span><span class="sxs-lookup"><span data-stu-id="40bc3-132">Use the <xref:System.Nullable%601.GetValueOrDefault> method to return either the assigned value, or the [default](../../language-reference/keywords/default-values-table.md) value of the underlying value type if the value of the nullable type is `null`.</span></span>

- <span data-ttu-id="40bc3-133">Verwenden Sie die <xref:System.Nullable%601.GetValueOrDefault(%600)>-Methode, um entweder den zugewiesenen Wert oder den bereitgestellten Standardwert zurückzugeben, wenn der Wert des Nullable-Typs `null` ist.</span><span class="sxs-lookup"><span data-stu-id="40bc3-133">Use the <xref:System.Nullable%601.GetValueOrDefault(%600)> method to return either the assigned value, or the provided default value if the value of the nullable type is `null`.</span></span>

- <span data-ttu-id="40bc3-134">Verwenden Sie den [NULL-Sammeloperator](../../language-reference/operators/null-coalescing-operator.md) `??`, um einer Variable eines zugrunde liegenden Werttyps einen Wert basierend auf einem Wert zuzuweisen, der Nullwerte zulässt: `int? x = null; int y = x ?? -1;`.</span><span class="sxs-lookup"><span data-stu-id="40bc3-134">Use the [null-coalescing operator](../../language-reference/operators/null-coalescing-operator.md), `??`, to assign a value to a variable of an underlying value type based on a nullable-type value: `int? x = null; int y = x ?? -1;`.</span></span> <span data-ttu-id="40bc3-135">Da `x` `null` ist, ist in diesem Beispiel der Ergebniswert von `y` `-1`.</span><span class="sxs-lookup"><span data-stu-id="40bc3-135">In the example, since `x` is `null`, the result value of `y` is `-1`.</span></span>

- <span data-ttu-id="40bc3-136">Wenn eine benutzerdefinierte Konvertierung zwischen zwei Werttypen definiert ist, kann die gleiche Konvertierung auch mit den entsprechenden Nullable-Typen verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="40bc3-136">If a user-defined conversion is defined between two value types, the same conversion can also be used with the corresponding nullable types.</span></span>

- <span data-ttu-id="40bc3-137">Geschachtelte Werttypen, die Nullwerte zulassen, sind nicht zulässig.</span><span class="sxs-lookup"><span data-stu-id="40bc3-137">Nested nullable value types are not allowed.</span></span> <span data-ttu-id="40bc3-138">Die folgende Zeile wird nicht kompiliert: `Nullable<Nullable<int>> n;`</span><span class="sxs-lookup"><span data-stu-id="40bc3-138">The following line doesn't compile: `Nullable<Nullable<int>> n;`</span></span>
