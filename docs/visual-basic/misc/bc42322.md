# <a name="runtime-errors-might-occur-when-converting-type1-to-type2"></a>Können Laufzeitfehler auftreten, bei der Konvertierung "\<Typ1 >', '\<Typ2 >'.

Beim Versuch, einen Schnittstellentyp in einen Klassentyp zu konvertieren, kann zur Laufzeit ein Fehler auftreten.

**Fehler-ID:** BC42322
