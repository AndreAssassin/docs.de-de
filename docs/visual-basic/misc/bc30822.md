## <a name="to-correct-this-error"></a><span data-ttu-id="5b639-107">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="5b639-107">To correct this error</span></span>

    ```  
    Sub TestDBNull()  
        Dim t As DataTable  
        ' Assume the DataGrid is bound to a DataTable.  
        t = CType(DataGrid1.DataSource, DataTable)  
        Dim r As DataRow  
        r = t.Rows(datagrid1.CurrentCell.RowNumber)  
        r.BeginEdit  
        r(1) = System.DBNull.Value ' Assign DBNull to the record.  
        r.EndEdit  
        r.AcceptChanges  
        If r.IsNull(1) Then  
            MsgBox("")  
        End If  
    End Sub  
    ```  
  
2. <span data-ttu-id="5b639-110">Verwenden Sie die [nichts](../../visual-basic/language-reference/nothing.md) -Schlüsselwort für Zuweisungen und Vergleiche, wenn Sie Objektvariablen verwenden.</span><span class="sxs-lookup"><span data-stu-id="5b639-110">Use the [Nothing](../../visual-basic/language-reference/nothing.md) keyword for assignments and comparisons when you use object variables.</span></span> <span data-ttu-id="5b639-111">Dies wird im folgenden Beispiel veranschaulicht:</span><span class="sxs-lookup"><span data-stu-id="5b639-111">The following example demonstrates this.</span></span>

    ```  
        ' cls is Nothing if it has not been assigned using the New keyword.  
        If (cls Is Nothing) Then  
            MsgBox("cls is Nothing")  
        End If  
        cls = Nothing ' Assign Nothing to the class variable cls.  
    End Sub  
    ```
