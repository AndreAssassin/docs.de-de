<span data-ttu-id="02720-103">Die meisten Ausdrücke können in Ausdrucksbaumstrukturen konvertiert werden, mehrdimensionale Arrays jedoch nicht.</span><span class="sxs-lookup"><span data-stu-id="02720-103">Most expressions can be converted to expression trees, but multi-dimensional arrays cannot be.</span></span> <span data-ttu-id="02720-104">Beispielsweise verursacht der folgende Code diesen Fehler:</span><span class="sxs-lookup"><span data-stu-id="02720-104">For example, the following code causes this error:</span></span>

```vb  
    Sub Main()  
        '' A multi-dimensional array cannot be converted.  
        'Dim expTree As Expressions.Expression(Of Func(Of Object)) = _  
        '    Function() New Integer(1, 1) {{1, 2}, {2, 3}}  
  
        ' A one-dimensional array can be converted.  
        Dim expTree2 As Expressions.Expression(Of Func(Of Object)) = _

    End Sub  
End Module  
```
