# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>"\<-mitgliedmitgliedsname >" wurde bereits von "\<Proze, >" deklariert, die für diesen \<Containername generiert wurde >

Es wurde ein Member deklariert, der den gleichen Namen wie ein Member hat, der vom Common Language Runtime deklariert wird.

**Fehler-ID:** BC30733

## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler

- Ändern Sie den Namen des Members, um den Konflikt zu vermeiden.
