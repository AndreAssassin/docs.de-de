# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="df81f-102">If-Operanden dürfen keine benannten Argumente sein</span><span class="sxs-lookup"><span data-stu-id="df81f-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="df81f-103">Die Verwendung von benannten Argumenten in den Operanden des `If` -Operators ist ungültig.</span><span class="sxs-lookup"><span data-stu-id="df81f-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="df81f-104">Dieser Fehler wird beispielsweise durch den folgenden Code verursacht:</span><span class="sxs-lookup"><span data-stu-id="df81f-104">The following example causes this error:</span></span>  
  
```vb

' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="df81f-105">Dies stellt einen Unterschied zur `IIf` -Funktion dar, die benannte Argumente zulässt, wie im folgenden Code zu sehen:</span><span class="sxs-lookup"><span data-stu-id="df81f-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```vb

```
