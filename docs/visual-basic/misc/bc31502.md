# <a name="resource-name-resourcename-cannot-be-used-more-than-once"></a>Der Ressourcen Name "\<resourcename >" kann nicht mehrmals verwendet werden.
Der Name für eine angegebene Ressource in einer DLL oder ausführbaren Datei wurde mehrfach verwendet.  
  
 **Fehler-ID:** BC31502  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Geben Sie Ressourcen eindeutige Namen.
