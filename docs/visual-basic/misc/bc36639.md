# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="afff8-102">'ByRef' Parameter\<' Parametername>' kann nicht in einem Lambda-Ausdruck verwendet werden</span><span class="sxs-lookup"><span data-stu-id="afff8-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="afff8-103">Ein innerhalb einer `Sub` oder Funktion deklarierter Lambdaausdruck kann keine `ByRef` -Parameter dieser `Sub` oder Funktion verwenden.</span><span class="sxs-lookup"><span data-stu-id="afff8-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="afff8-104">Beispielsweise verursacht der folgende Code diesen Fehler, da der `ByRef` -Parameter `n` im Lambdaausdruck verwendet wird.</span><span class="sxs-lookup"><span data-stu-id="afff8-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb  
'' Not valid.
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="afff8-105">**Fehler-ID:** BC36639</span><span class="sxs-lookup"><span data-stu-id="afff8-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="afff8-106">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="afff8-106">To correct this error</span></span>  
  
- <span data-ttu-id="afff8-107">Weisen Sie den `ByRef` -Parameter einer lokalen Variablen zu, und verwenden Sie die lokale Variable im Lambdaausdruck, wie im folgenden Code dargestellt:</span><span class="sxs-lookup"><span data-stu-id="afff8-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>

    ```vb  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
