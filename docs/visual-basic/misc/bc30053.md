# <a name="arrays-cannot-be-declared-with-new"></a><span data-ttu-id="7d0d4-102">Arrays können nicht mit 'New' deklariert werden</span><span class="sxs-lookup"><span data-stu-id="7d0d4-102">Arrays cannot be declared with 'New'</span></span>
<span data-ttu-id="7d0d4-103">Das Schlüsselwort `New` kann nur im Initialisierungsteil einer Arraydeklaration auftreten.</span><span class="sxs-lookup"><span data-stu-id="7d0d4-103">The `New` keyword can appear only in the initialization part of an array declaration.</span></span> <span data-ttu-id="7d0d4-104">Dies bedeutet, dass sich `New` auf der rechten Seite des Gleichheitszeichens (`=`) befinden muss, damit ein neuer Arraytyp erstellt werden kann, der der Arrayvariablen zugewiesen werden kann.</span><span class="sxs-lookup"><span data-stu-id="7d0d4-104">This means `New` must be on the right side of the equal sign (`=`) so it can create a new array type to be assigned to the array variable.</span></span>  
  
 <span data-ttu-id="7d0d4-105">Das abgekürzte Verfahren für die Klasseninitialisierung ist für Arrays nicht verfügbar.</span><span class="sxs-lookup"><span data-stu-id="7d0d4-105">The shortcut for class initialization is not available for arrays.</span></span> <span data-ttu-id="7d0d4-106">Die folgenden beiden Codezeilen sind gültig und gleichwertig, weil sie ein Objekt aus einer Klasse initialisieren.</span><span class="sxs-lookup"><span data-stu-id="7d0d4-106">The following two code lines are both valid and are equivalent because they initialize an object from a class.</span></span>  
  
```  
Dim formA as Form = New Form  
Dim formA as New Form  
```  
  
 <span data-ttu-id="7d0d4-107">Allerdings kann die Arrayinitialisierung nicht das gleiche abgekürzte Verfahren wie die Klasseninitialisierung verwenden.</span><span class="sxs-lookup"><span data-stu-id="7d0d4-107">However, array initialization cannot use the same shortcut as class initialization.</span></span>  
  
 <span data-ttu-id="7d0d4-108">Beachten Sie, dass die `New` -Klausel für ein Array runde Klammern ( `()`) und geschweifte Klammern ( `{}`) enthalten muss.</span><span class="sxs-lookup"><span data-stu-id="7d0d4-108">Note that the `New` clause for an array must contain both parentheses, `()`, and braces, `{}`.</span></span> <span data-ttu-id="7d0d4-109">Die runden Klammern geben an, dass der neue Typ ein Array ist, und die geschweiften Klammern stellen die Initialisierungswerte bereit.</span><span class="sxs-lookup"><span data-stu-id="7d0d4-109">The parentheses specify that the new type is an array, and the braces supply the initialization values.</span></span> <span data-ttu-id="7d0d4-110">Der Compiler benötigt die geschweiften Klammern selbst dann, wenn sie leer sind – also auch dann, wenn Sie keinen der Arraywerte initialisieren.</span><span class="sxs-lookup"><span data-stu-id="7d0d4-110">The compiler requires the braces even if they are empty, that is, even if you are not initializing any of the array values.</span></span>
