# <a name="friend-access-was-denied-to-assembly-if-you-want-the-compiler-to-consider-friend-types-in-assembly-add-the-public-key-information-in-the-project-settings-or-through-a-command-line-switch-to-the-compiler-for-the-project-project"></a>Der Friend-Zugriff auf \<assembly > wurde verweigert. Wenn der Compiler Friend-Typen in \<assembly > in Erwägung gezogen soll, fügen Sie die Informationen des öffentlichen Schlüssels in den Projekteinstellungen oder über einen Befehls Zeilenschalter zum Compiler für das Projekt \<project > hinzu.

**Fehler-ID:** BC31536
