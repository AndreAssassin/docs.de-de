# <a name="finally-statements-are-not-valid-in-the-immediate-window"></a>Finally-Anweisungen sind im Direktfenster nicht gültig.
`Finally` -Anweisungen können nur im Quellcode verwendet werden.  
  
 **Fehler-ID:** BC30716  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Entfernen Sie `Finally` -Anweisungen aus Debugcode.
