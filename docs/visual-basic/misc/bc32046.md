# <a name="new-cannot-be-used-on-a-type-parameter-that-does-not-have-a-new-constraint"></a><span data-ttu-id="4588b-102">„New“ kann nicht für einen Typparameter verwendet werden, der keine New-Einschränkung aufweist.</span><span class="sxs-lookup"><span data-stu-id="4588b-102">'New' cannot be used on a type parameter that does not have a 'New' constraint</span></span>
<span data-ttu-id="4588b-103">Eine deklarationsanweisung verwendet eine [neuer Operator](../../visual-basic/language-reference/operators/new-operator.md) Klausel für einen Typparameter als Typ erstellt werden, und der Typparameter deklariert wird, ohne eine `New` Einschränkung.</span><span class="sxs-lookup"><span data-stu-id="4588b-103">A declaration statement uses a [New Operator](../../visual-basic/language-reference/operators/new-operator.md) clause specifying a type parameter as the type to be created, and the type parameter is declared without a `New` constraint.</span></span>  
  
 <span data-ttu-id="4588b-104">Eine *Einschränkung* für einen Typparameter ist eine Anforderung an jedes Typargument, das beim Erstellen des generischen Typs an diesen Typparameter übergeben wird.</span><span class="sxs-lookup"><span data-stu-id="4588b-104">A *constraint* on a type parameter imposes a requirement on any type argument passed to that type parameter when the generic type is created.</span></span> <span data-ttu-id="4588b-105">Die `New` -Einschränkung gibt an, dass das Typargument einen parameterlosen Konstruktor verfügbar machen muss, auf den der erstellende Code zugreifen kann.</span><span class="sxs-lookup"><span data-stu-id="4588b-105">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="4588b-106">Dadurch wird eine `New` -Klausel in einer Deklarationsanweisung und das Erstellen einer Instanz dieses Typs ermöglicht.</span><span class="sxs-lookup"><span data-stu-id="4588b-106">This is what allows a `New` clause in a declaration statement to create an instance of that type.</span></span>  
  
 <span data-ttu-id="4588b-107">**Fehler-ID:** BC32046</span><span class="sxs-lookup"><span data-stu-id="4588b-107">**Error ID:** BC32046</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4588b-108">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="4588b-108">To correct this error</span></span>  
  
- <span data-ttu-id="4588b-109">Wenn Sie für das Typargument das Verfügbarmachen eines parameterlosen Konstruktors, auf den der Code zugreifen kann, verbindlich festlegen können, fügen Sie der Deklaration des Typparameters die `New` -Einschränkung hinzu.</span><span class="sxs-lookup"><span data-stu-id="4588b-109">If you can require the type argument to expose an accessible parameterless constructor, add the `New` constraint to the declaration of the type parameter.</span></span>  
  
- <span data-ttu-id="4588b-110">Wenn Sie für das Typargument das Verfügbarmachen eines parameterlosen Konstruktors, auf den der Code zugreifen kann, nicht verbindlich festlegen können, entfernen Sie die `New` -Klausel aus der Deklarationsanweisung.</span><span class="sxs-lookup"><span data-stu-id="4588b-110">If you cannot require the type argument to expose an accessible parameterless constructor, remove the `New` clause from the declaration statement.</span></span> <span data-ttu-id="4588b-111">Sie können nicht gewährleisten, dass alle an den Typparameter übergebenen Typargumente die Erstellung einer Instanz zugelassen.</span><span class="sxs-lookup"><span data-stu-id="4588b-111">You cannot guarantee that any type argument passed to that type parameter permits creation of an instance.</span></span>
