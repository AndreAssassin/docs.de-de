# <a name="membername1-and-membername2-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>'\<membername1>' und '\<membername2>' kann nicht gegenseitig überladen, da sie nur von den Typen der optionalen Parameter unterscheiden
Überladene Member müssen für mindestens einen erforderlichen Parameter unterschiedliche Datentypen aufweisen.  
  
 **Fehler-ID:** BC30696  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Definieren Sie mindestens einen erforderlichen Parameter als anderen Datentyp.
