## <a name="example"></a><span data-ttu-id="74494-109">Beispiel</span><span class="sxs-lookup"><span data-stu-id="74494-109">Example</span></span>  
 <span data-ttu-id="74494-110">Die folgende Klasse enthält eine Standardeigenschaft `defaultProp`, die ein ganzzahliges Argument erfordert.</span><span class="sxs-lookup"><span data-stu-id="74494-110">The following class contains a default property, `defaultProp`, that requires an integer argument.</span></span>  
  
```vb  
Public Class SomeStrings  
    Private myStrings() As String  
    Sub New(ByVal size As Integer)  
        ReDim myStrings(size)  
    End Sub  
    Default Property defaultProp(ByVal index As Integer) As String  
        Get  
            Return myStrings(index)  
        End Get  
        Set(ByVal Value As String)  
            myStrings(index) = Value  
        End Set  
    End Property  
End Class  
```

