# <a name="delegate-constructor-for-type-typename-necessary-for-the-implementation-of-delegates-cannot-be-found"></a>Konstruktor für Typ '\<Typname >', erforderlich für die Implementierung von Delegaten, wurde nicht gefunden
Die Assembly, die den Typ enthält, ist möglicherweise beschädigt oder ungültig.  
  
 **Fehler-ID:** BC31074  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1.  Stellen Sie sicher, dass die Assembly mit diesem Typ gültig ist.  
  
2.  Kompilieren Sie das Programm erneut, um festzustellen, ob der Fehler erneut auftritt.  
  
3.  Wenn der Fehler weiterhin auftritt, speichern Sie Ihre Arbeit, und starten Sie Visual Studio.
