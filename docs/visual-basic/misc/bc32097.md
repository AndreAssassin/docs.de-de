<span data-ttu-id="dc929-103">Ein Typparameter wird als Operand für den [IsNot Operator](../../visual-basic/language-reference/operators/isnot-operator.md) verwendet, wenn der Typparameter ohne das [Class-Schlüsselwort (Visual Basic)](../../visual-basic/language-reference/statements/class-statement.md) oder einen bestimmten Klassennamen in der Einschränkungsliste definiert wird.</span><span class="sxs-lookup"><span data-stu-id="dc929-103">A type parameter is used as an operand for the [IsNot Operator](../../visual-basic/language-reference/operators/isnot-operator.md) when the type parameter is defined without either the [Class (Visual Basic)](../../visual-basic/language-reference/statements/class-statement.md) keyword or a specific class name in its constraint list.</span></span>
