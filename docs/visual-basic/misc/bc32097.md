# <a name="isnot-operand-of-type-typeparametername-can-be-compared-only-to-nothing-because-typeparametername-is-a-type-parameter-with-no-class-constraint"></a><span data-ttu-id="81fd9-102">"IsNot"-Operand vom Typ "\<Typparametername >" verglichen werden können, nur mit "Nothing", da "\<Typparametername >" ein Typparameter ohne klasseneinschränkung ist</span><span class="sxs-lookup"><span data-stu-id="81fd9-102">'IsNot' operand of type '\<typeparametername>' can be compared only to 'Nothing' because '\<typeparametername>' is a type parameter with no class constraint</span></span>
<span data-ttu-id="81fd9-103">Ein Typparameter wird als Operand für den [IsNot Operator](../../visual-basic/language-reference/operators/isnot-operator.md) verwendet, wenn der Typparameter ohne das [Class-Schlüsselwort (Visual Basic)](../../visual-basic/language-reference/statements/class-statement.md) oder einen bestimmten Klassennamen in der Einschränkungsliste definiert wird.</span><span class="sxs-lookup"><span data-stu-id="81fd9-103">A type parameter is used as an operand for the [IsNot Operator](../../visual-basic/language-reference/operators/isnot-operator.md) when the type parameter is defined without either the [Class (Visual Basic)](../../visual-basic/language-reference/statements/class-statement.md) keyword or a specific class name in its constraint list.</span></span>  
  
 <span data-ttu-id="81fd9-104">`IsNot` vergleicht zwei Verweistypen, um zu bestimmen, ob sie auf unterschiedliche Objektinstanzen im Arbeitsspeicher verweisen.</span><span class="sxs-lookup"><span data-stu-id="81fd9-104">`IsNot` compares two reference types to determine whether they point to different object instances in memory.</span></span> <span data-ttu-id="81fd9-105">Ein Operand, der kein Verweistyp ist, es sei denn, der andere Operand ist, wird nicht akzeptiert [nichts](../../visual-basic/language-reference/nothing.md).</span><span class="sxs-lookup"><span data-stu-id="81fd9-105">It cannot take an operand that is not a reference type unless the other operand is [Nothing](../../visual-basic/language-reference/nothing.md).</span></span>  
  
 <span data-ttu-id="81fd9-106">**Fehler-ID:** BC32097</span><span class="sxs-lookup"><span data-stu-id="81fd9-106">**Error ID:** BC32097</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="81fd9-107">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="81fd9-107">To correct this error</span></span>  
  
- <span data-ttu-id="81fd9-108">Wenn Sie festlegen können, dass das für diesen Typparameter angegebene Typargument immer ein Verweistyp sein muss, fügen Sie entweder das `Class` -Schlüsselwort oder einen bestimmten Klassennamen zur Einschränkungsliste für den Typparameter hinzu.</span><span class="sxs-lookup"><span data-stu-id="81fd9-108">If you can require that the type argument supplied to this type parameter always be a reference type, add either the `Class` keyword or a specific class name to the constraint list for the type parameter.</span></span>  
  
- <span data-ttu-id="81fd9-109">Wenn Sie nicht festlegen können, dass das für diesen Typparameter angegebene Typargument immer ein Verweistyp sein muss, entfernen Sie es aus dem `IsNot` -Ausdruck.</span><span class="sxs-lookup"><span data-stu-id="81fd9-109">If you cannot require that the type argument supplied to this type parameter always be a reference type, remove it from the `IsNot` expression.</span></span> <span data-ttu-id="81fd9-110">Sie können es nicht mithilfe des `IsNot` -Operators mit anderen Verweistypen vergleichen.</span><span class="sxs-lookup"><span data-stu-id="81fd9-110">You cannot compare it to other reference types with the `IsNot` operator.</span></span>
