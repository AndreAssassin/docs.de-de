 `Public Class testClass(Of t1 As {t2, Class}, t2 As Structure)`  
  
 <span data-ttu-id="3709a-105">Die indirekte Einschränkung `Structure` und die direkte Einschränkung `Class` verursachen einen Konflikt für Typparameter `t1`, weil die `Structure` -Einschränkung erfordert, dass das entsprechende Typargument ein Werttyp ist, während `Class` erfordert, dass es ein Verweistyp ist.</span><span class="sxs-lookup"><span data-stu-id="3709a-105">The indirect constraint `Structure` and the direct constraint `Class` cause a conflict for type parameter `t1`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>
