# <a name="local-variable-variablename-is-already-declared-in-the-current-block"></a>Lokale Variable '\<Variablenname >' ist bereits im aktuellen Block deklariert
Zwei Variablen mit dem gleichen Namen sind im selben Block (Prozedur- oder Kontrollblock wie `If...Then`) deklariert.  
  
 **Fehler-ID:** BC30288
