# <a name="mathfunction1-is-not-declared"></a>'\<mathfunction1 >' ist nicht deklariert
'\<mathfunction1 >' ist nicht deklariert. Funktion wurde in der Klasse "System.Math" verschoben und heißt jetzt '\<mathfunction2 >'.  
  
 Einige systeminterne Funktionen in Visual Basic in früheren Versionen wurden an verschoben wurden die <xref:System.Math?displayProperty=nameWithType> Namespace. Dadurch wird ihre Funktionalität allgemeiner verfügbar für alle Programmiersprachen.  
  
 **Fehler-ID:** BC30819  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Verwenden Sie die in <xref:System.Math?displayProperty=nameWithType>deklarierten Methoden.
