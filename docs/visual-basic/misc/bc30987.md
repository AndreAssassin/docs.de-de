<span data-ttu-id="6a010-103">Um eine Instanz eines benannten oder anonymen Typs mithilfe eines Objektinitialisierers zu deklarieren, müssen Sie die Liste der Felder oder Eigenschaften und deren Anfangswerte in Klammern einschließen ({und}).</span><span class="sxs-lookup"><span data-stu-id="6a010-103">To declare an instance of a named or anonymous type by using an object initializer, you must enclose the list of fields or properties and their initial values in braces ({ and }).</span></span>  
  
```

Dim emp = New Employee() With {.Name = "Rob Young", .ID = 55555}
```  
