# <a name="name-is-declared-in-project-projectname1-which-is-not-referenced-by-project-projectname2"></a>"\<name >" ist im Projekt "\<projectname1 >" deklariert, auf das von Projekt "\<projectname2 >" nicht verwiesen wird.
Ein Programmierelement ist in einem Projekt definiert, auf das im aktuellen Projekt kein Verweis vorhanden ist.  
  
 **Fehler-ID:** BC32004  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Fügen Sie im aktuellen Projekt einen Verweis auf das Projekt hinzu, das das gewünschte Element enthält.
