# <a name="type-argument-inference-failed-for-type-parameter-typeparametername-of-genericproceduresignature"></a><span data-ttu-id="bd5e2-102">Fehler beim Ableiten des Typarguments für den Typparameter "\<typeparametername >" von "\<genericproceduresignature >".</span><span class="sxs-lookup"><span data-stu-id="bd5e2-102">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'</span></span>

<span data-ttu-id="bd5e2-103">**Fehler-ID:** BC32051</span><span class="sxs-lookup"><span data-stu-id="bd5e2-103">**Error ID:** BC32051</span></span>

<span data-ttu-id="bd5e2-104">Fehler beim Ableiten des Typarguments für den Typparameter "\<typeparametername >" von "\<genericproceduresignature >".</span><span class="sxs-lookup"><span data-stu-id="bd5e2-104">Type argument inference failed for type parameter '\<typeparametername>' of '\<genericproceduresignature>'.</span></span> <span data-ttu-id="bd5e2-105">Das Typargument konnte nicht aus dem an den Parameter "\<parametername >" übergebenen Argument abgeleitet werden.</span><span class="sxs-lookup"><span data-stu-id="bd5e2-105">Type argument could not be inferred from the argument passed to parameter '\<parametername>'.</span></span>

 <span data-ttu-id="bd5e2-106">Eine generische Prozedur wird ohne Angabe von Typargumenten aufgerufen, und der Compiler kann den an einen der Parameter zu übergebenden Typ nicht ableiten.</span><span class="sxs-lookup"><span data-stu-id="bd5e2-106">A generic procedure is called without supplying any type arguments, and the compiler cannot infer the type to pass to one of the parameters.</span></span>

 <span data-ttu-id="bd5e2-107">Wenn Sie eine generische Prozedur aufrufen, geben Sie in der Regel für jeden Typparameter, der durch die generische Prozedur definiert wird, ein Typargument an.</span><span class="sxs-lookup"><span data-stu-id="bd5e2-107">Normally, when you call a generic procedure, you supply a type argument for each type parameter that the generic procedure defines.</span></span> <span data-ttu-id="bd5e2-108">Wenn Sie keine Typargumente angeben, versucht der Compiler, die an die Typparameter zu übergebenden Typen abzuleiten.</span><span class="sxs-lookup"><span data-stu-id="bd5e2-108">If you do not supply any type arguments, then the compiler attempts to infer the types to be passed to the type parameters.</span></span> <span data-ttu-id="bd5e2-109">Wenn durch den Kontext des Aufrufs Datentypinformationen für einen Typparameter bereitgestellt werden, die einen Konflikt verursachen, schlägt der Typrückschluss fehl.</span><span class="sxs-lookup"><span data-stu-id="bd5e2-109">If the context of the call provides conflicting data type information for a type parameter, then the type inference fails.</span></span>

 <span data-ttu-id="bd5e2-110">Dieser Fehler kann durch folgenden Code generiert werden:</span><span class="sxs-lookup"><span data-stu-id="bd5e2-110">The following code can generate this error:</span></span>

```vb
Public Sub DoSomething(Of T)(arg1 As T(), arg2 As T)
End Sub

Public Sub Main()
    DoSomething(6, 42)
End Sub
```  
  
 <span data-ttu-id="bd5e2-111">Im vorhergehenden Beispiel leitet der Compiler den Typ `Integer` für `t` basierend auf dem an `arg2`übergebenen Wert 42 ab.</span><span class="sxs-lookup"><span data-stu-id="bd5e2-111">In the preceding example, the compiler infers type `Integer` for `t` based on the value of 42 passed to `arg2`.</span></span> <span data-ttu-id="bd5e2-112">Dieser Rückschluss würde jedoch voraussetzen, dass `arg1` vom Typ `Integer()`ist, also ein Array von `Integer`. Der an `arg1` übergebene Wert 6 stimmt aber nicht mit diesem Typ überein.</span><span class="sxs-lookup"><span data-stu-id="bd5e2-112">However, that inference would require `arg1` to be of type `Integer()`, that is, an array of `Integer`, and the value 6 passed to `arg1` does not match that type.</span></span>

## <a name="to-correct-this-error"></a><span data-ttu-id="bd5e2-113">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="bd5e2-113">To correct this error</span></span>

- <span data-ttu-id="bd5e2-114">Geben Sie Typargumente für die generische Prozedur an, damit der Compiler sie nicht ableiten muss.</span><span class="sxs-lookup"><span data-stu-id="bd5e2-114">Supply type arguments to the generic procedure, so that the compiler does not have to infer them.</span></span>

- <span data-ttu-id="bd5e2-115">Geben Sie normale Argumente mit Typen an, die mit denjenigen der Typargumente übereinstimmen.</span><span class="sxs-lookup"><span data-stu-id="bd5e2-115">Supply normal arguments with types that match those of the type arguments.</span></span>
