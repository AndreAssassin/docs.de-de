# <a name="catch-cannot-appear-after-finally-within-a-try-statement"></a>"Catch" kann nicht hinter "Finally" in einer Try-Anweisung stehen
Eine `Catch` -Anweisung wird im Code hinter dem `Finally` -Schl√ºsselwort angezeigt, das einen `Try` -Anweisungsblock beendet. `Catch` muss innerhalb eines `Try...Catch...Finally` -Anweisungsblocks angezeigt werden.  
  
 **Fehler-ID:** BC30379  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler
