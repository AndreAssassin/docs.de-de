# <a name="aggregate-function-name-cannot-be-used-with-a-type-character"></a>Der Name der Aggregatfunktion darf nicht mit einem Typzeichen verwendet werden.
Ein Typzeichen befindet sich am Ende des Namens einer Aggregatfunktion.  
  
 **Fehler-ID:** BC36617  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1. Entfernen Sie das Typzeichen aus dem Namen der Aggregatfunktion.
