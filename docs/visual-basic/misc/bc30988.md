B. jeweils, dass `client` eine Instanz der `Customer` -Klasse ist.</span><span class="sxs-lookup"><span data-stu-id="b87ee-104">For example, the following statements each declare `client` to be an instance of the `Customer` class.</span></span> <span data-ttu-id="b87ee-105">Der Typname `Customer` folgt auf `New`.</span><span class="sxs-lookup"><span data-stu-id="b87ee-105">The type name `Customer` follows `New`.</span></span>

```
