Dies setzt wiederum voraus, dass diese Member definiert sind.</span><span class="sxs-lookup"><span data-stu-id="bbfe3-107">Inheritance involves the ability to reuse members of the base class, which in turn requires that these members be defined.</span></span> <span data-ttu-id="bbfe3-108">Daher können Visual Basic keinen Code wie im folgenden Beispiel kompilieren.</span><span class="sxs-lookup"><span data-stu-id="bbfe3-108">Therefore, Visual Basic cannot compile code such as the following example.</span></span>

```vb  
    Inherits innerClass

    End Class

```
