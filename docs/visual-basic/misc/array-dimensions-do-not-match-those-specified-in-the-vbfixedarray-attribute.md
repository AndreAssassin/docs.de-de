# <a name="array-dimensions-do-not-match-those-specified-in-the-vbfixedarray-attribute"></a>Die Arraydimensionen stimmen nicht mit den im "VBFixedArray"-Attribut angegebenen Dimensionen 端berein
Die Dimensionen des Arrays stimmen nicht mit den Dimensionen 端berein, die Sie im `VBFixedArray` -Attribut angegeben haben.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Legen Sie die Arraydimensionen in der `Dim` -Anweisung des Arrays so fest, dass sie mit den durch das `VBFixedArray` -Attribut angegebenen Dimensionen 端bereinstimmen.
