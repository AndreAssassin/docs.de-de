 <span data-ttu-id="dd7ec-105">Die indirekten Einschränkungen `Structure` und `Class` verursachen einen Konflikt für Typparameter `t1`, weil die `Structure` -Einschränkung erfordert, dass das entsprechende Typargument ein Werttyp ist, während `Class` erfordert, dass es ein Verweistyp ist.</span><span class="sxs-lookup"><span data-stu-id="dd7ec-105">The indirect constraints `Structure` and `Class` cause a conflict for type parameter `t1`, because the `Structure` constraint requires that the corresponding type argument be a value type, while `Class` requires that it be a reference type.</span></span>
