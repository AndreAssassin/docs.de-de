# <a name="parameter-parametername-in-extension-method-methodname-defined-in-typename-already-has-a-matching-argument"></a>Parameter '\<Parametername >' in der Erweiterungsmethode'\<Methodname > 'definiert, die'\<Typename > 'hat bereits ein entsprechendes Argument
Ein Argument wird innerhalb des gleichen Prozeduraufrufs mehrfach anhand des Namens übergeben.  
  
 **Fehler-ID:** BC36584  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Überprüfen Sie, ob einer der Argumentnamen falsch geschrieben ist.  
  
- Entfernen Sie das doppelt vorkommende Argument aus der Argumentliste.
