<span data-ttu-id="5428b-103">Die Datentypen der Typparameter in der Methode '\<Methodenname >' können nicht von diesen Argumenten abgeleitet werden, da mehrere Typen möglich sind.</span><span class="sxs-lookup"><span data-stu-id="5428b-103">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because more than one type is possible.</span></span> <span data-ttu-id="5428b-104">Sie können diesen Fehler möglicherweise beheben, indem Sie die Datentypen explizit angeben.</span><span class="sxs-lookup"><span data-stu-id="5428b-104">Specifying the data type(s) explicitly might correct this error.</span></span>

<span data-ttu-id="5428b-105">In einem Aufruf einer generischen Prozedur wurde versucht, über den Typrückschluss den Typ oder die Typen des bzw. der Typparameter zu bestimmen.</span><span class="sxs-lookup"><span data-stu-id="5428b-105">An attempt has been made to use type inference to determine the type or types of the type parameter or parameters in a call to a generic procedure.</span></span> <span data-ttu-id="5428b-106">Der Compiler findet mehrere mögliche Datentypen für einen oder mehrere Typparameter und meldet diesen Fehler.</span><span class="sxs-lookup"><span data-stu-id="5428b-106">The compiler finds more than one possible data type for one or more of the type parameters, and it reports this error.</span></span>

> [!NOTE]
> <span data-ttu-id="5428b-107">Wenn die Angabe von Argumenten keine Option ist (z. B. für Abfrageoperatoren in Abfrageausdrücken), wird nur der erste Satz der Fehlermeldung angezeigt.</span><span class="sxs-lookup"><span data-stu-id="5428b-107">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>

<span data-ttu-id="5428b-108">Im folgenden Code wird der Fehler veranschaulicht.</span><span class="sxs-lookup"><span data-stu-id="5428b-108">The following code demonstrates the error.</span></span>

```vb

    Sub Main()
        '' Not valid.
        'targetMethod(1, "2")
    End Sub

    Sub targetMethod(Of T)(ByVal p1 As T, ByVal p2 As T)
    End Sub

```
