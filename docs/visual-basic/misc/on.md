# <a name="on-visual-basic"></a>On (Visual Basic)
Führt eine Antwort auf einen Laufzeitfehler ein oder aktiviert eine Compileroption.  
  
 In-Abfragen identifiziert Schlüssel für die Auflistungen, die verknüpft werden.
