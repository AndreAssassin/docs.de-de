# <a name="end-removehandler-must-be-the-first-statement-on-a-line"></a>„End RemoveHandler“ muss die erste Anweisung der Zeile sein.
Einem Doppelpunkt (:) als Trennzeichen folgt eine `End RemoveHandler` -Anweisung. `End RemoveHandler` muss die einzige Anweisung der jeweiligen Quellzeile sein.  
  
 **Fehler-ID:** BC31119  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Platzieren Sie jede Anweisung in einer eigenen Zeile.
