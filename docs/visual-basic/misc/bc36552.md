## <a name="example"></a><span data-ttu-id="50325-110">Beispiel</span><span class="sxs-lookup"><span data-stu-id="50325-110">Example</span></span>  
 <span data-ttu-id="50325-111">Der erste Parameter im folgenden Beispiel gibt an, dass die `Print` -Methode den `String` -Datentyp erweitert.</span><span class="sxs-lookup"><span data-stu-id="50325-111">The first parameter in the following example indicates that the `Print` method extends the `String` data type.</span></span>  
  
```vb  
<Extension()> _  
Public Sub Print (ByVal str As String)  
    Console.WriteLine(str)  
End Sub  
```  
  
 <span data-ttu-id="50325-112">Wenn die Erweiterungsmethode wie folgt aufgerufen wird, wird der Parameter `str` in der Methode an `greeting`gebunden, d. h. die Instanz von `String` , die `Print`aufruft.</span><span class="sxs-lookup"><span data-stu-id="50325-112">When the extension method is called as follows, parameter `str` in the method is bound to `greeting`, the instance of `String` that calls `Print`.</span></span> <span data-ttu-id="50325-113">Der Compiler verwendet `greeting` als das Argument der Erweiterungsmethode `Print`.</span><span class="sxs-lookup"><span data-stu-id="50325-113">The compiler will use `greeting` as the argument to extension method `Print`.</span></span>

