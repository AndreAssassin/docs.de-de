- <span data-ttu-id="7e7ac-107">**Datentyp des Elements.**</span><span class="sxs-lookup"><span data-stu-id="7e7ac-107">**Element Data Type.**</span></span> <span data-ttu-id="7e7ac-108">Die Datentypen der Elemente beider Arrays müssen Referenztypen sein.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-108">The data types of the elements of both arrays must be reference types.</span></span> <span data-ttu-id="7e7ac-109">Sie können nicht konvertiert werden ein `Integer` array an eine `Long` array oder sogar um eine `Object` array, da mindestens ein Werttyp ist.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-109">You cannot convert an `Integer` array to a `Long` array, or even to an `Object` array, because at least one value type is involved.</span></span> <span data-ttu-id="7e7ac-110">Weitere Informationen finden Sie unter [Value Types and Reference Types](../../../../visual-basic/programming-guide/language-features/data-types/value-types-and-reference-types.md).</span><span class="sxs-lookup"><span data-stu-id="7e7ac-110">For more information, see [Value Types and Reference Types](../../../../visual-basic/programming-guide/language-features/data-types/value-types-and-reference-types.md).</span></span>  
  
- <span data-ttu-id="7e7ac-111">**Die Konvertierung.**</span><span class="sxs-lookup"><span data-stu-id="7e7ac-111">**Convertibility.**</span></span> <span data-ttu-id="7e7ac-112">Eine Konvertierung, erweiternd oder einschränkend, muss zwischen den Elementtypen der beiden Arrays sein.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-112">A conversion, either widening or narrowing, must be possible between the element types of the two arrays.</span></span> <span data-ttu-id="7e7ac-113">Ein Beispiel, das diese Anforderung nicht erfüllt, wird ein Versuch einer Konvertierung zwischen einem `String` Array und einem Array von einer Klasse abgeleitet <xref:System.Attribute?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-113">An example that fails this requirement is an attempted conversion between a `String` array and an array of a class derived from <xref:System.Attribute?displayProperty=nameWithType>.</span></span> <span data-ttu-id="7e7ac-114">Diese beiden Typen haben nichts gemeinsam, und keine Konvertierung jeglicher Art, die zwischen ihnen vorhanden ist.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-114">These two types have nothing in common, and no conversion of any kind exists between them.</span></span>  
  
 <span data-ttu-id="7e7ac-115">Eine Konvertierung von einem Arraytyp ist erweiternde oder einschränkende, je nachdem, ob die Konvertierung der jeweiligen Elemente erweiternde oder einschränkende ist.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-115">A conversion of one array type to another is widening or narrowing depending on whether the conversion of the respective elements is widening or narrowing.</span></span> <span data-ttu-id="7e7ac-116">Weitere Informationen finden Sie unter [Widening and Narrowing Conversions](../../../../visual-basic/programming-guide/language-features/data-types/widening-and-narrowing-conversions.md).</span><span class="sxs-lookup"><span data-stu-id="7e7ac-116">For more information, see [Widening and Narrowing Conversions](../../../../visual-basic/programming-guide/language-features/data-types/widening-and-narrowing-conversions.md).</span></span>  
  
## <a name="conversion-to-an-object-array"></a><span data-ttu-id="7e7ac-117">Konvertierung in ein Objektarray</span><span class="sxs-lookup"><span data-stu-id="7e7ac-117">Conversion to an Object Array</span></span>  
 <span data-ttu-id="7e7ac-118">Wenn Sie deklarieren eine `Object` Array ohne Initialisierung hinzuzufügen, den Elementtyp `Object` solange er nicht initialisiert ist.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-118">When you declare an `Object` array without initializing it, its element type is `Object` as long as it remains uninitialized.</span></span> <span data-ttu-id="7e7ac-119">Wenn Sie es in ein Array von einer bestimmten Klasse festlegen, dauert es für den Typ dieser Klasse.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-119">When you set it to an array of a specific class, it takes on the type of that class.</span></span> <span data-ttu-id="7e7ac-120">Die zugrunde liegende Typ ist jedoch weiterhin `Object`, können Sie ihn anschließend in ein anderes Array einer unabhängigen Klasse festlegen.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-120">However, its underlying type is still `Object`, and you can subsequently set it to another array of an unrelated class.</span></span> <span data-ttu-id="7e7ac-121">Da alle Klassen abgeleitet `Object`, können Sie Elementtyp des Arrays aus einer Klasse in einer beliebigen anderen Klasse ändern.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-121">Since all classes derive from `Object`, you can change the array's element type from any class to any other class.</span></span>  
  
 <span data-ttu-id="7e7ac-122">Im folgenden Beispiel keine Konvertierungen zwischen Typen `student` und `String`, aber beide abgeleitet `Object`, sodass alle Zuweisungen gültig sind.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-122">In the following example, no conversion exists between types `student` and `String`, but both derive from `Object`, so all assignments are valid.</span></span>  
  
```  
' Assume student has already been defined as a class.  
Dim testArray() As Object  
' testArray is still an Object array at this point.  
Dim names() As String = New String(3) {"Name0", "Name1", "Name2", "Name3"}  
testArray = New student(3) {}  
' testArray is now of type student().  
testArray = names  
' testArray is now a String array.  
```  
  
### <a name="underlying-type-of-an-array"></a><span data-ttu-id="7e7ac-123">Der zugrunde liegende Typ eines Arrays</span><span class="sxs-lookup"><span data-stu-id="7e7ac-123">Underlying Type of an Array</span></span>  
 <span data-ttu-id="7e7ac-124">Wenn Sie ursprünglich ein Array mit einer bestimmten Klasse deklarieren, ist die zugrunde liegende Elementtyp dieser Klasse.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-124">If you originally declare an array with a specific class, its underlying element type is that class.</span></span> <span data-ttu-id="7e7ac-125">Wenn Sie anschließend auf ein Array von einer anderen Klasse festlegen, muss eine Konvertierung zwischen den beiden Klassen vorhanden sein.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-125">If you subsequently set it to an array of another class, there must be a conversion between the two classes.</span></span>  
  
 <span data-ttu-id="7e7ac-126">Im folgenden Beispiel `students` ist eine `student` Array.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-126">In the following example, `students` is a `student` array.</span></span> <span data-ttu-id="7e7ac-127">Da keine Konvertierung vorhanden, zwischen ist `String` und `student`, die letzte Anweisung schlägt fehl.</span><span class="sxs-lookup"><span data-stu-id="7e7ac-127">Since no conversion exists between `String` and `student`, the last statement fails.</span></span>  
  
```  
Dim students() As student  
Dim names() As String = New String(3) {"Name0", "Name1", "Name2", "Name3"}  
students = New Student(3) {}  
' The following statement fails at compile time.  
students = names  
```
