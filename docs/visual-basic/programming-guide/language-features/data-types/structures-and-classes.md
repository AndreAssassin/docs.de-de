- <span data-ttu-id="83116-117">Beide können freigegebene Konstruktoren, mit oder ohne Parameter verfügen.</span><span class="sxs-lookup"><span data-stu-id="83116-117">Both can have shared constructors, with or without parameters.</span></span>

## <a name="differences"></a><span data-ttu-id="83116-120">Unterschiede</span><span class="sxs-lookup"><span data-stu-id="83116-120">Differences</span></span>  
 <span data-ttu-id="83116-121">Strukturen und Klassen unterscheiden sich in den folgenden Angaben:</span><span class="sxs-lookup"><span data-stu-id="83116-121">Structures and classes differ in the following particulars:</span></span>  
  
- <span data-ttu-id="83116-122">Strukturen sind *Werttypen*; Klassen sind *Verweistypen*.</span><span class="sxs-lookup"><span data-stu-id="83116-122">Structures are *value types*; classes are *reference types*.</span></span> <span data-ttu-id="83116-123">Eine Variable eines Strukturtyps enthält die Struktur der Daten, anstatt mit einem Verweis auf die Daten als ein Klassentyp ist.</span><span class="sxs-lookup"><span data-stu-id="83116-123">A variable of a structure type contains the structure's data, rather than containing a reference to the data as a class type does.</span></span>  
  
- <span data-ttu-id="83116-124">Verwenden von Strukturen stapelreservierung; Klassen verwenden Heapzuordnung.</span><span class="sxs-lookup"><span data-stu-id="83116-124">Structures use stack allocation; classes use heap allocation.</span></span>  
  
- <span data-ttu-id="83116-125">Alle Strukturelemente sind `Public` standardmäßig-Klasse Variablen und Konstanten sind `Private` standardmäßig, während andere Klassenmember werden `Public` standardmäßig.</span><span class="sxs-lookup"><span data-stu-id="83116-125">All structure elements are `Public` by default; class variables and constants are `Private` by default, while other class members are `Public` by default.</span></span> <span data-ttu-id="83116-126">Dieses Verhalten für Klassenmember bietet Kompatibilität mit dem Visual Basic 6.0-System von Standardwerten.</span><span class="sxs-lookup"><span data-stu-id="83116-126">This behavior for class members provides compatibility with the Visual Basic 6.0 system of defaults.</span></span>  
  
- <span data-ttu-id="83116-127">Eine Struktur müssen mindestens eine nicht freigegebene Variable oder nicht benutzerdefiniertes Ereigniselement enthalten Event-Element; eine Klasse kann vollständig leer sein.</span><span class="sxs-lookup"><span data-stu-id="83116-127">A structure must have at least one nonshared variable or nonshared, noncustom event element; a class can be completely empty.</span></span>  
  
- <span data-ttu-id="83116-128">Können nicht Strukturelemente deklariert werden, als `Protected`;-Klasse, Elemente zu können.</span><span class="sxs-lookup"><span data-stu-id="83116-128">Structure elements cannot be declared as `Protected`; class members can.</span></span>  
  
- <span data-ttu-id="83116-129">Eine Strukturprozedur nur, wenn sie Ereignisse behandeln eine [Shared](../../../../visual-basic/language-reference/modifiers/shared.md) `Sub` Prozedur, und nur von der [AddHandler-Anweisung](../../../../visual-basic/language-reference/statements/addhandler-statement.md); jede Klassenprozedur kann Behandeln von Ereignissen, die mit entweder der [ Behandelt](../../../../visual-basic/language-reference/statements/handles-clause.md) Schlüsselwort oder `AddHandler` Anweisung.</span><span class="sxs-lookup"><span data-stu-id="83116-129">A structure procedure can handle events only if it is a [Shared](../../../../visual-basic/language-reference/modifiers/shared.md)`Sub` procedure, and only by means of the [AddHandler Statement](../../../../visual-basic/language-reference/statements/addhandler-statement.md); any class procedure can handle events, using either the [Handles](../../../../visual-basic/language-reference/statements/handles-clause.md) keyword or the `AddHandler` statement.</span></span> <span data-ttu-id="83116-130">Weitere Informationen finden Sie unter [Ereignisse](../../../../visual-basic/programming-guide/language-features/events/index.md)definiert sind.</span><span class="sxs-lookup"><span data-stu-id="83116-130">For more information, see [Events](../../../../visual-basic/programming-guide/language-features/events/index.md).</span></span>  
  
- <span data-ttu-id="83116-131">Struktur-Variablendeklarationen können keine Initialisierer oder Anfangsgröße für Arrays angeben; Klasse Variablendeklarationen können.</span><span class="sxs-lookup"><span data-stu-id="83116-131">Structure variable declarations cannot specify initializers or initial sizes for arrays; class variable declarations can.</span></span>  
  
- <span data-ttu-id="83116-132">Strukturen erben implizit vom die <xref:System.ValueType?displayProperty=nameWithType> Klasse und kann nicht von irgendeinem anderen Typ; erben Klassen können von einer Klasse oder Klassen erben, außer <xref:System.ValueType?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="83116-132">Structures implicitly inherit from the <xref:System.ValueType?displayProperty=nameWithType> class and cannot inherit from any other type; classes can inherit from any class or classes other than <xref:System.ValueType?displayProperty=nameWithType>.</span></span>  
  
- <span data-ttu-id="83116-133">Strukturen können nicht vererbt werden; Klassen sind.</span><span class="sxs-lookup"><span data-stu-id="83116-133">Structures are not inheritable; classes are.</span></span>  
  
- <span data-ttu-id="83116-134">Strukturen werden nie beendet, nie aufruft und die common Language Runtime (CLR) die <xref:System.Object.Finalize%2A> Methode für jede Struktur; Klassen werden vom Garbage Collector (GC), die aufruft beendet <xref:System.Object.Finalize%2A> für eine Klasse, wenn er erkennt keine aktiven Verweise vorhanden sind Verbleibende.</span><span class="sxs-lookup"><span data-stu-id="83116-134">Structures are never terminated, so the common language runtime (CLR) never calls the <xref:System.Object.Finalize%2A> method on any structure; classes are terminated by the garbage collector (GC), which calls <xref:System.Object.Finalize%2A> on a class when it detects there are no active references remaining.</span></span>  
  
- <span data-ttu-id="83116-135">Eine Struktur ist einen Konstruktor nicht erforderlich; eine Klasse ist.</span><span class="sxs-lookup"><span data-stu-id="83116-135">A structure does not require a constructor; a class does.</span></span>  
  
- <span data-ttu-id="83116-136">Strukturen aufweisen können nicht freigegebene Konstruktoren, die nur dann, wenn sie Parameter annehmen; Klassen können mit oder ohne Parameter verfügen.</span><span class="sxs-lookup"><span data-stu-id="83116-136">Structures can have nonshared constructors only if they take parameters; classes can have them with or without parameters.</span></span>  
  
 <span data-ttu-id="83116-137">Jede Struktur verfügt über einen impliziten öffentlichen Konstruktor ohne Parameter.</span><span class="sxs-lookup"><span data-stu-id="83116-137">Every structure has an implicit public constructor without parameters.</span></span> <span data-ttu-id="83116-138">Dieser Konstruktor initialisiert alle der Struktur Datenelemente auf ihre Standardwerte zurück.</span><span class="sxs-lookup"><span data-stu-id="83116-138">This constructor initializes all the structure's data elements to their default values.</span></span> <span data-ttu-id="83116-139">Sie können dieses Verhalten nicht neu definieren.</span><span class="sxs-lookup"><span data-stu-id="83116-139">You cannot redefine this behavior.</span></span>  
  
## <a name="instances-and-variables"></a><span data-ttu-id="83116-140">Instanzen und Variablen</span><span class="sxs-lookup"><span data-stu-id="83116-140">Instances and Variables</span></span>  
 <span data-ttu-id="83116-141">Da Strukturen Werttypen sind, wird jede Strukturvariable permanent auf eine einzelne Struktur-Instanz gebunden.</span><span class="sxs-lookup"><span data-stu-id="83116-141">Because structures are value types, each structure variable is permanently bound to an individual structure instance.</span></span> <span data-ttu-id="83116-142">Jedoch sind Klassen Verweistypen, und eine Objektvariablen auf verschiedenen Klasseninstanzen, zu unterschiedlichen Zeitpunkten verweisen kann.</span><span class="sxs-lookup"><span data-stu-id="83116-142">But classes are reference types, and an object variable can refer to various class instances at different times.</span></span> <span data-ttu-id="83116-143">Dieser Unterschied wirkt sich auf die die Nutzung von Strukturen und Klassen gibt folgenden Möglichkeiten:</span><span class="sxs-lookup"><span data-stu-id="83116-143">This distinction affects your usage of structures and classes in the following ways:</span></span>  
  
- <span data-ttu-id="83116-144">**Die Initialisierung.**</span><span class="sxs-lookup"><span data-stu-id="83116-144">**Initialization.**</span></span> <span data-ttu-id="83116-145">Eine Strukturvariable enthält implizit eine Initialisierung der Elemente, die mit der Struktur parameterlosen Konstruktor.</span><span class="sxs-lookup"><span data-stu-id="83116-145">A structure variable implicitly includes an initialization of the elements using the structure's parameterless constructor.</span></span> <span data-ttu-id="83116-146">Aus diesem Grund `Dim s As struct1` entspricht `Dim s As struct1 = New struct1()`.</span><span class="sxs-lookup"><span data-stu-id="83116-146">Therefore, `Dim s As struct1` is equivalent to `Dim s As struct1 = New struct1()`.</span></span>

     <span data-ttu-id="83116-157">Strukturelemente, sind jedoch in einer eigenen Instanz isoliert.</span><span class="sxs-lookup"><span data-stu-id="83116-157">Structure elements, however, are isolated within their own instance.</span></span> <span data-ttu-id="83116-158">Änderungen an deren Werte werden nicht in anderen Strukturvariablen, sogar in anderen Instanzen der gleichen wiedergegeben `Structure` Deklaration.</span><span class="sxs-lookup"><span data-stu-id="83116-158">Changes to their values are not reflected in any other structure variables, even in other instances of the same `Structure` declaration.</span></span>  
  
- <span data-ttu-id="83116-159">**Gleichheit.**</span><span class="sxs-lookup"><span data-stu-id="83116-159">**Equality.**</span></span> <span data-ttu-id="83116-160">Gleichheitsüberprüfung von zwei Strukturen muss für ein Element für Element durchgeführt werden.</span><span class="sxs-lookup"><span data-stu-id="83116-160">Equality testing of two structures must be performed with an element-by-element test.</span></span> <span data-ttu-id="83116-161">Zwei Objektvariablen verglichen werden können, mit der <xref:System.Object.Equals%2A> Methode.</span><span class="sxs-lookup"><span data-stu-id="83116-161">Two object variables can be compared using the <xref:System.Object.Equals%2A> method.</span></span> <span data-ttu-id="83116-162"><xref:System.Object.Equals%2A> Gibt an, ob die zwei Variablen auf dieselbe Instanz verweisen.</span><span class="sxs-lookup"><span data-stu-id="83116-162"><xref:System.Object.Equals%2A> indicates whether the two variables point to the same instance.</span></span>
