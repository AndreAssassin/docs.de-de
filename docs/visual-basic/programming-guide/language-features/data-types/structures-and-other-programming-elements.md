## <a name="structures-and-procedures"></a><span data-ttu-id="87059-120">Strukturen und Prozeduren</span><span class="sxs-lookup"><span data-stu-id="87059-120">Structures and Procedures</span></span>  
 <span data-ttu-id="87059-121">Sie können eine Struktur als ein Prozedurargument übergeben.</span><span class="sxs-lookup"><span data-stu-id="87059-121">You can pass a structure as a procedure argument.</span></span> <span data-ttu-id="87059-122">Dies wird anhand des folgenden Beispiels veranschaulicht.</span><span class="sxs-lookup"><span data-stu-id="87059-122">The following example illustrates this.</span></span>  
  
```vb  
Public currentCPUName As String = "700MHz Pentium compatible"  
Public currentMemorySize As Long = 256  
Public Sub fillSystem(ByRef someSystem As systemInfo)  
    someSystem.cPU = currentCPUName  
    someSystem.memory = currentMemorySize  
    someSystem.purchaseDate = Now  
End Sub  
```  
  
 <span data-ttu-id="87059-123">Im vorherige Beispiel übergibt der Struktur *als Verweis*, dadurch, dass die Prozedur die Elemente zu ändern, sodass die Änderungen im aufrufenden Code wirksam.</span><span class="sxs-lookup"><span data-stu-id="87059-123">The preceding example passes the structure *by reference*, which allows the procedure to modify its elements so that the changes take effect in the calling code.</span></span> <span data-ttu-id="87059-124">Wenn Sie eine Struktur vor solchen Änderungen schützen möchten, können übergeben sie als Wert.</span><span class="sxs-lookup"><span data-stu-id="87059-124">If you want to protect a structure against such modification, pass it by value.</span></span>  
  
 <span data-ttu-id="87059-125">Sie können auch eine Struktur von Zurückgeben einer `Function` Verfahren.</span><span class="sxs-lookup"><span data-stu-id="87059-125">You can also return a structure from a `Function` procedure.</span></span> <span data-ttu-id="87059-126">Dies wird anhand des folgenden Beispiels veranschaulicht.</span><span class="sxs-lookup"><span data-stu-id="87059-126">The following example illustrates this.</span></span>  
  
```vb  
Dim allSystems(100) As systemInfo  
Function findByDate(ByVal searchDate As Date) As systemInfo  
    Dim i As Integer  
    For i = 1 To 100  
        If allSystems(i).purchaseDate = searchDate Then Return allSystems(i)  
    Next i  
   ' Process error: system with desired purchase date not found.  
End Function  
```
