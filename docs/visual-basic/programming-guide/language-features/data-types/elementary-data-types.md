# <a name="elementary-data-types-visual-basic"></a>Elementare Datentypen (Visual Basic)
Visual Basic stellt einen Satz von vordefinierten Datentypen, die Sie für viele der von Programmierelementen verwenden können. Dieser Abschnitt beschreibt diese Typen und deren Verwendung.  
  
> [!NOTE]
>  Jeder elementaren Datentyp in Visual Basic wird unterstützt, von einer Struktur oder eine Klasse, die in der <xref:System> Namespace. Der Compiler verwendet jedes Data-Type-Schlüsselwort als Alias für die zugrunde liegende Struktur oder Klasse. Deklarieren z. B. das reservierte Wort mit einer Variablen `Byte` ist dasselbe wie das Deklarieren es mit dem vollqualifizierten Namen <xref:System.Byte?displayProperty=nameWithType>.
