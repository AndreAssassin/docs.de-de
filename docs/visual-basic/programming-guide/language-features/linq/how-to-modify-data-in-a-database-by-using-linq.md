### <a name="to-add-code-to-modify-the-database-and-display-the-results"></a><span data-ttu-id="f816e-128">Zum Hinzufügen von Code zum Ändern der Datenbank und die Ergebnisse anzeigen</span><span class="sxs-lookup"><span data-stu-id="f816e-128">To add code to modify the database and display the results</span></span>

2. <span data-ttu-id="f816e-130">Wenn Sie Tabellen in den O/R-Designer hinzugefügt haben, wird der Designer hinzugefügt eine <xref:System.Data.Linq.DataContext> Objekt, das Ihr Projekt.</span><span class="sxs-lookup"><span data-stu-id="f816e-130">When you added tables to the O/R Designer, the designer added a <xref:System.Data.Linq.DataContext> object to your project.</span></span> <span data-ttu-id="f816e-131">Dieses Objekt enthält Code, mit denen Sie Zugriff auf die Customers-Tabelle.</span><span class="sxs-lookup"><span data-stu-id="f816e-131">This object contains code that you can use to access the Customers table.</span></span> <span data-ttu-id="f816e-132">Sie enthält auch Code, ein lokales Kundenobjekt und eine Sammlung "Kunden" für die Tabelle definiert.</span><span class="sxs-lookup"><span data-stu-id="f816e-132">It also contains code that defines  a local Customer object and a Customers collection for the table.</span></span> <span data-ttu-id="f816e-133">Die <xref:System.Data.Linq.DataContext> Objekt für das Projekt mit dem Namen wird anhand des Namens der DBML-Datei.</span><span class="sxs-lookup"><span data-stu-id="f816e-133">The <xref:System.Data.Linq.DataContext> object for your project is named based on the name of your .dbml file.</span></span> <span data-ttu-id="f816e-134">Für dieses Projekt die <xref:System.Data.Linq.DataContext> Objekt mit dem Namen `northwindDataContext`.</span><span class="sxs-lookup"><span data-stu-id="f816e-134">For this project, the <xref:System.Data.Linq.DataContext> object is named `northwindDataContext`.</span></span>

     <span data-ttu-id="f816e-137">Doppelklicken Sie auf das Windows Form, Form1, um zum Hinzufügen von Code die <xref:System.Windows.Forms.Form.Load> Ereignis, um die Abfrage, die Customers-Tabelle als eine Eigenschaft verfügbar gemacht wird Ihre <xref:System.Data.Linq.DataContext>.</span><span class="sxs-lookup"><span data-stu-id="f816e-137">Double-click the Windows Form, Form1, to add code to the <xref:System.Windows.Forms.Form.Load> event to query the Customers table that is exposed as a property of your <xref:System.Data.Linq.DataContext>.</span></span> <span data-ttu-id="f816e-138">Fügen Sie den folgenden Code hinzu:</span><span class="sxs-lookup"><span data-stu-id="f816e-138">Add the following code:</span></span>  
  
    ```vb

    Private Sub Form1_Load(ByVal sender As System.Object,   
                           ByVal e As System.EventArgs  
                          ) Handles MyBase.Load  
      db = New northwindDataContext()  
  
      RefreshData()  
    End Sub  
  
    Private Sub RefreshData()  
      Dim customers = From cust In db.Customers   
                      Where cust.City(0) = "W"   
                      Select cust  
  
      DataGridView1.DataSource = customers  
    End Sub  
    ```

4. <span data-ttu-id="f816e-144">Doppelklicken Sie auf die **hinzufügen** Schaltfläche Hinzufügen von Code, dessen `Click` Ereignis.</span><span class="sxs-lookup"><span data-stu-id="f816e-144">Double-click the **Add** button to add code to its `Click` event.</span></span> <span data-ttu-id="f816e-145">Fügen Sie den folgenden Code hinzu:</span><span class="sxs-lookup"><span data-stu-id="f816e-145">Add the following code:</span></span>

                                ByVal e As System.EventArgs  
                               ) Handles AddButton.Click

        .City = "Wellington",   
        .CompanyName = "Blue Yonder Airlines",   
        .ContactName = "Jill Frank",

        .CustomerID = "JILLF"}  
  
      db.Customers.InsertOnSubmit(cust)  
  
      Try  
        db.SubmitChanges()  
      Catch  
        ' Handle exception.  
      End Try  
  
      RefreshData()  
    End Sub  
    ```  
  
5. <span data-ttu-id="f816e-146">Doppelklicken Sie auf die **Update** Schaltfläche Hinzufügen von Code, dessen `Click` Ereignis.</span><span class="sxs-lookup"><span data-stu-id="f816e-146">Double-click the **Update** button to add code to its `Click` event.</span></span> <span data-ttu-id="f816e-147">Fügen Sie den folgenden Code hinzu:</span><span class="sxs-lookup"><span data-stu-id="f816e-147">Add the following code:</span></span>

                                  ) Handles UpdateButton.Click  
      Dim updateCust = (From cust In db.Customers   
                        Where cust.CustomerID = "JILLF").ToList()(0)  
  
      updateCust.ContactName = "Jill Shrader"
      updateCust.Country = "Wales"
      updateCust.CompanyName = "Red Yonder Airlines"
      updateCust.City = "Cardiff"
  
      Try  
        db.SubmitChanges()  
      Catch  
        ' Handle exception.  
      End Try  
  
      RefreshData()  
    End Sub  
    ```  
  
6. <span data-ttu-id="f816e-148">Doppelklicken Sie auf die **löschen** Schaltfläche Hinzufügen von Code, dessen `Click` Ereignis.</span><span class="sxs-lookup"><span data-stu-id="f816e-148">Double-click the **Delete** button to add code to its `Click` event.</span></span> <span data-ttu-id="f816e-149">Fügen Sie den folgenden Code hinzu:</span><span class="sxs-lookup"><span data-stu-id="f816e-149">Add the following code:</span></span>

      End Try  
  
      RefreshData()
