## <a name="rules"></a><span data-ttu-id="6435b-104">Regeln</span><span class="sxs-lookup"><span data-stu-id="6435b-104">Rules</span></span>  
 <span data-ttu-id="6435b-105">Ein Elementname in Visual Basic muss die folgenden Regeln beachten:</span><span class="sxs-lookup"><span data-stu-id="6435b-105">An element name in Visual Basic must observe the following rules:</span></span>

 <span data-ttu-id="6435b-111">Das folgende Beispiel zeigt einige gültige Elementnamen.</span><span class="sxs-lookup"><span data-stu-id="6435b-111">The following example shows some valid element names.</span></span>

 `_567`

 `_`  
  
 `12ABC`  
  
 `xyz$wv`

## <a name="case-sensitivity-in-names"></a><span data-ttu-id="6435b-133">Groß-/Kleinschreibung in Namen</span><span class="sxs-lookup"><span data-stu-id="6435b-133">Case Sensitivity in Names</span></span>
