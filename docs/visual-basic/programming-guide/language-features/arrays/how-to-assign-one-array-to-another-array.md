# <a name="how-to-assign-one-array-to-another-array-visual-basic"></a><span data-ttu-id="19561-102">Vorgehensweise: Zuweisen eines Arrays zu einem anderen Array (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="19561-102">How to: Assign One Array to Another Array (Visual Basic)</span></span>
<span data-ttu-id="19561-103">Da Arrays Objekte sind, können Sie sie in zuweisungsanweisungen wie andere Objekttypen.</span><span class="sxs-lookup"><span data-stu-id="19561-103">Because arrays are objects, you can use them in assignment statements like other object types.</span></span> <span data-ttu-id="19561-104">Eine Array-Variable enthält einen Zeiger auf die Daten enthalten sind, die die Elemente des Arrays und die Informationen Rang und die Länge und eine Zuordnung kopiert nur this-Zeiger.</span><span class="sxs-lookup"><span data-stu-id="19561-104">An array variable holds a pointer to the data constituting the array elements and the rank and length information, and an assignment copies only this pointer.</span></span>  
  
### <a name="to-assign-one-array-to-another-array"></a><span data-ttu-id="19561-105">Zuweisen eines Arrays zu einem anderen array</span><span class="sxs-lookup"><span data-stu-id="19561-105">To assign one array to another array</span></span>  
  
1. <span data-ttu-id="19561-106">Stellen Sie sicher, dass die beiden Arrays den gleichen Rang (Anzahl der Dimensionen) und kompatiblen Elements-Datentypen aufweisen.</span><span class="sxs-lookup"><span data-stu-id="19561-106">Ensure that the two arrays have the same rank (number of dimensions) and compatible element data types.</span></span>  
  
2. <span data-ttu-id="19561-107">Verwenden Sie eine standardmäßige zuweisungsanweisung, um das Quellarray in den Zielarray zuzuweisen.</span><span class="sxs-lookup"><span data-stu-id="19561-107">Use a standard assignment statement to assign the source array to the destination array.</span></span> <span data-ttu-id="19561-108">Führen Sie nicht entweder Arrayname mit Klammern.</span><span class="sxs-lookup"><span data-stu-id="19561-108">Do not follow either array name with parentheses.</span></span>  
  
    ```  
    Dim formArray() As System.Windows.Forms.Form  
    Dim controlArray() As System.Windows.Forms.Control  
    controlArray = formArray  
    ```  
  
 <span data-ttu-id="19561-109">Wenn Sie ein Array zu einem anderen zuweisen, gelten die folgenden Regeln:</span><span class="sxs-lookup"><span data-stu-id="19561-109">When you assign one array to another, the following rules apply:</span></span>  
  
-   <span data-ttu-id="19561-110">**Gleicher Rang.**</span><span class="sxs-lookup"><span data-stu-id="19561-110">**Equal Ranks.**</span></span> <span data-ttu-id="19561-111">Der Rang (Anzahl der Dimensionen) des Zielarrays muss des Quellarrays, identisch sein.</span><span class="sxs-lookup"><span data-stu-id="19561-111">The rank (number of dimensions) of the destination array must be the same as that of the source array.</span></span>  
  
     <span data-ttu-id="19561-112">Sofern die Ränge von zwei Arrays gleich sind, müssen die Dimensionen nicht gleich sind.</span><span class="sxs-lookup"><span data-stu-id="19561-112">Provided the ranks of the two arrays are equal, the dimensions do not need to be equal.</span></span> <span data-ttu-id="19561-113">Die Anzahl der Elemente in der angegebenen Dimension kann bei der Zuordnung ändern.</span><span class="sxs-lookup"><span data-stu-id="19561-113">The number of elements in a given dimension can change during assignment.</span></span>  
  
-   <span data-ttu-id="19561-114">**Elementtypen.**</span><span class="sxs-lookup"><span data-stu-id="19561-114">**Element Types.**</span></span> <span data-ttu-id="19561-115">Es müssen entweder beide Arrays *Verweistyp* Elemente oder beide Arrays müssen *Werttyp* Elemente.</span><span class="sxs-lookup"><span data-stu-id="19561-115">Either both arrays must have *reference type* elements or both arrays must have *value type* elements.</span></span> <span data-ttu-id="19561-116">Weitere Informationen finden Sie unter [Value Types and Reference Types](../../../../visual-basic/programming-guide/language-features/data-types/value-types-and-reference-types.md).</span><span class="sxs-lookup"><span data-stu-id="19561-116">For more information, see [Value Types and Reference Types](../../../../visual-basic/programming-guide/language-features/data-types/value-types-and-reference-types.md).</span></span>  
  
    -   <span data-ttu-id="19561-117">Wenn beide Arrays Elementen mit Werttyp haben, müssen die Element-Datentypen genau übereinstimmen.</span><span class="sxs-lookup"><span data-stu-id="19561-117">If both arrays have value type elements, the element data types must be exactly the same.</span></span> <span data-ttu-id="19561-118">Die einzige Ausnahme hierbei ist, dass Sie ein Array von zuweisen können `Enum` Elemente in ein Array des Basistyps dieses `Enum`.</span><span class="sxs-lookup"><span data-stu-id="19561-118">The only exception to this is that you can assign an array of `Enum` elements to an array of the base type of that `Enum`.</span></span>  
  
    -   <span data-ttu-id="19561-119">Wenn beide Arrays Elemente aufweisen, muss der Elementtyp der Datenquelle von der Ziel-Elementtyp abgeleitet werden.</span><span class="sxs-lookup"><span data-stu-id="19561-119">If both arrays have reference type elements, the source element type must derive from the destination element type.</span></span> <span data-ttu-id="19561-120">Wenn dies der Fall ist, müssen die beiden Arrays die gleiche vererbungsbeziehung als ihre Elemente.</span><span class="sxs-lookup"><span data-stu-id="19561-120">When this is the case, the two arrays have the same inheritance relationship as their elements.</span></span> <span data-ttu-id="19561-121">Dies wird als bezeichnet *Array-Kovarianz*.</span><span class="sxs-lookup"><span data-stu-id="19561-121">This is called *array covariance*.</span></span>  
  
 <span data-ttu-id="19561-122">Der Compiler meldet, dass ein Fehler, wenn die oben genannten Regeln, z. B. verletzt werden die Datentypen nicht kompatibel sind oder die Ränge ungleich sind.</span><span class="sxs-lookup"><span data-stu-id="19561-122">The compiler reports an error if the above rules are violated, for example if the data types are not compatible or the ranks are unequal.</span></span> <span data-ttu-id="19561-123">Sie können die Fehlerbehandlung in Ihrem Code, um sicherzustellen, dass die Arrays kompatibel sind, bevor Sie versuchen, eine Zuordnung hinzufügen.</span><span class="sxs-lookup"><span data-stu-id="19561-123">You can add error handling to your code to make sure that the arrays are compatible before attempting an assignment.</span></span> <span data-ttu-id="19561-124">Sie können auch die [TryCast-Operator](../../../../visual-basic/language-reference/operators/trycast-operator.md) Schlüsselwort, wenn Sie das Auslösen einer Ausnahme vermeiden möchten.</span><span class="sxs-lookup"><span data-stu-id="19561-124">You can also use the [TryCast Operator](../../../../visual-basic/language-reference/operators/trycast-operator.md) keyword if you want to avoid throwing an exception.</span></span>
