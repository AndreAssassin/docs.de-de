4. <span data-ttu-id="a37ed-116">Fügen Sie nach `Sub Main` von `Module1` eine neue Methode (`Sub`) hinzu, um einen \<description-> Knoten in das angegebene HTML-Format umzuwandeln.</span><span class="sxs-lookup"><span data-stu-id="a37ed-116">After `Sub Main` of `Module1`, add a new method (`Sub`) to transform a \<Description> node into the specified HTML format.</span></span> <span data-ttu-id="a37ed-117">Diese Methode wird vom Code im vorherigen Schritt aufgerufen und wird verwendet, um das Format der \<description > Elemente beizubehalten.</span><span class="sxs-lookup"><span data-stu-id="a37ed-117">This method is called by the code in the previous step and is used to preserve the format of the \<Description> elements.</span></span>  
  
     <span data-ttu-id="a37ed-118">Diese Methode ersetzt unter Elemente der \<description >-Elements durch HTML.</span><span class="sxs-lookup"><span data-stu-id="a37ed-118">This method replaces sub-elements of the \<Description> element with HTML.</span></span> <span data-ttu-id="a37ed-119">Die `ReplaceWith`-Methode wird verwendet, um den Speicherort der untergeordneten Elemente beizubehalten.</span><span class="sxs-lookup"><span data-stu-id="a37ed-119">The `ReplaceWith` method is used to preserve the location of the sub-elements.</span></span> <span data-ttu-id="a37ed-120">Der transformierte Inhalt der \<description >-Elements ist in einem HTML-Absatz Element (\<P >) enthalten.</span><span class="sxs-lookup"><span data-stu-id="a37ed-120">The transformed content of the \<Description> element is included in an HTML paragraph (\<p>) element.</span></span> <span data-ttu-id="a37ed-121">Die <xref:System.Xml.Linq.XContainer.Nodes%2A>-Eigenschaft wird verwendet, um den transformierten Inhalt des \<description >-Elements abzurufen.</span><span class="sxs-lookup"><span data-stu-id="a37ed-121">The <xref:System.Xml.Linq.XContainer.Nodes%2A> property is used to retrieve the transformed content of the \<Description> element.</span></span> <span data-ttu-id="a37ed-122">Dadurch wird sichergestellt, dass untergeordnete Elemente in den transformierten Inhalt eingeschlossen werden.</span><span class="sxs-lookup"><span data-stu-id="a37ed-122">This ensures that sub-elements are included in the transformed content.</span></span>  
  
     <span data-ttu-id="a37ed-123">Fügen Sie den folgenden Code nach `Sub Main` von `Module1` hinzu.</span><span class="sxs-lookup"><span data-stu-id="a37ed-123">Add the following code after `Sub Main` of `Module1`.</span></span>  
  
    ```vb  
    Public Function TransformDescription(ByVal desc As XElement) As XElement  
  
      ' Replace <technology> elements with <b>.  
      Dim content = (From element In desc...<technology>).ToList()  
  
      If content.Count > 0 Then  
        For i = 0 To content.Count - 1  
          content(i).ReplaceWith(<b><%= content(i).Value %></b>)  
        Next  
      End If  
  
      ' Replace <audience> elements with <i>.  
      content = (From element In desc...<audience>).ToList()  
  
      If content.Count > 0 Then  
        For i = 0 To content.Count - 1  
          content(i).ReplaceWith(<i><%= content(i).Value %></i>)  
        Next  
      End If  
  
      ' Return the updated contents of the <Description> element.  
      Return <p><%= desc.Nodes %></p>  
    End Function  
    ```
