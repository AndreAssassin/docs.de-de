<span data-ttu-id="02f86-120">Wenn die `Overloads` -Schlüsselwort ausgelassen wird, beim Überladen von eines geerbten Members mit einem Member, die über identische Parameter und die Parametertypen verfügt, und klicken Sie dann die Überladung wird aufgerufen, *nach Namen shadowing*.</span><span class="sxs-lookup"><span data-stu-id="02f86-120">If the `Overloads` keyword is omitted when overloading an inherited member with a member that has identical parameters and parameter types, then the overloading is called *shadowing by name*.</span></span> <span data-ttu-id="02f86-121">Nach dem Namen Shadowing die geerbte Implementierung eines Elements ersetzt, und es allen anderen Überladungen für Instanzen von der abgeleiteten Klasse und die Decedents nicht verfügbar ist.</span><span class="sxs-lookup"><span data-stu-id="02f86-121">Shadowing by name replaces the inherited implementation of a member, and it makes all other overloads unavailable to instances of the derived class and its decedents.</span></span>
