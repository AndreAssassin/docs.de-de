# <a name="programming-guide-linq-to-xml-visual-basic"></a>Programming Guide (LINQ to XML) (Visual Basic)
Dieser Abschnitt enthält grundlegende Informationen und Anleitungen zum Programmieren mit [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)].  
  
## <a name="who-should-read-this-documentation"></a>Zielgruppe  
 This documentation targets developers who already understand Visual Basic and some basic aspects of the .NET Framework.  
  
 Ziel dieser Dokumentation ist es, Entwicklern das Arbeiten mit [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] zu erleichtern. [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] vereinfacht die XML-Programmierung. und kann auch von Entwicklern verwendet werden, die keine ausgesprochenen Experten auf diesem Gebiet sind.  
  
 [!INCLUDE[sqltecxlinq](~/includes/sqltecxlinq-md.md)] basiert in weiten Teilen auf generischen Klassen. Therefore, is very important that you understand the use of generic types and classes. For more information, see [Generic Types in Visual Basic](../../../../visual-basic/programming-guide/language-features/data-types/generic-types.md).
