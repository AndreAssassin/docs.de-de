## <a name="advantages-of-pure-functions"></a><span data-ttu-id="a224f-147">Vorteile von reinen Funktionen</span><span class="sxs-lookup"><span data-stu-id="a224f-147">Advantages of Pure Functions</span></span>  
 <span data-ttu-id="a224f-148">Der Hauptgrund für die Implementierung von funktionalen Transformationen als reinen Funktionen (Pure-Funktionen) besteht darin, dass reine Funktionen zusammenstellbar sind, d. h., sie sind in sich abgeschlossen und zustandslos.</span><span class="sxs-lookup"><span data-stu-id="a224f-148">The primary reason to implement functional transformations as pure functions is that pure functions are composable: that is, self-contained and stateless.</span></span> <span data-ttu-id="a224f-149">Diese Eigenschaften bieten u. a. die folgenden Vorteile:</span><span class="sxs-lookup"><span data-stu-id="a224f-149">These characteristics bring a number of benefits, including the following:</span></span>  
  
- <span data-ttu-id="a224f-150">Bessere Lesbarkeit und Verwaltbarkeit:</span><span class="sxs-lookup"><span data-stu-id="a224f-150">Increased readability and maintainability.</span></span> <span data-ttu-id="a224f-151">Jede Funktion ist für die Erledigung einer bestimmten Aufgabe anhand ihrer Argumente vorgesehen,</span><span class="sxs-lookup"><span data-stu-id="a224f-151">This is because each function is designed to accomplish a specific task given its arguments.</span></span> <span data-ttu-id="a224f-152">ohne sich dabei auf einen externen Zustand zu verlassen.</span><span class="sxs-lookup"><span data-stu-id="a224f-152">The function does not rely on any external state.</span></span>
