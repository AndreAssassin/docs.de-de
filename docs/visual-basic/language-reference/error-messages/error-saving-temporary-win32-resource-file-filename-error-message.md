# <a name="error-saving-temporary-win32-resource-file-filename-error-message"></a>Error saving temporary Win32 resource file '\<filename>': \<error message>
The Visual Basic compiler calls the Assembly Linker (Al.exe, also known as Alink) to generate an assembly with a manifest. Der Linker meldet einen Fehler beim Abrufen eines Dateinamen zum Schreiben einer speicherinternen Ressource.  
  
 **Error ID:** BC30137  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
1. Examine the quoted error message and consult the topic [Al.exe](../../../framework/tools/al-exe-assembly-linker.md). for further explanation and advice.  
  
2. Wenn der Fehler weiterhin besteht, tragen Sie Informationen zu den Umständen zusammen, und benachrichtigen Sie den Produktsupport von Microsoft.
