# <a name="array-subscript-expression-missing"></a>Fehlender Arraysubscriptausdruck
Eine Array Initialisierung verlässt mindestens einen der Index, der die Array Begrenzungen definiert. Beispielsweise kann die-Anweisung den Ausdruck `myArray (5,5,,10)` enthalten, der den dritten Index verlässt.  
  
 **Fehler-ID:** BC30306  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Geben Sie den fehlenden Index an.
