# <a name="subscript-out-of-range-visual-basic"></a>Index außerhalb des definierten Bereichs (Visual Basic)
Ein Array-Subscript ist ungültig, da sie außerhalb des zulässigen Bereichs liegt. Der niedrigste Indexwert für eine Dimension ist immer 0, während die höchsten Indexwert wird zurückgegeben, durch die `GetUpperBound` Methode für diese Dimension.  
  
## <a name="to-correct-this-error"></a>So beheben Sie diesen Fehler  
  
- Ändern Sie den Index aus, damit sie innerhalb des gültigen Bereichs liegt.
