# <a name="copying-the-value-of-byref-parameter-parametername-back-to-the-matching-argument-narrows-from-type-typename1-to-type-typename2"></a><span data-ttu-id="a9080-102">Zurückkopieren des ByRef-Parameters '\<Parametername >' zurück in das entsprechende Argument führt zu einer Einschränkung vom Typ "\<Typname1 >' in den Typ"\<Typname2 >'</span><span class="sxs-lookup"><span data-stu-id="a9080-102">Copying the value of 'ByRef' parameter '\<parametername>' back to the matching argument narrows from type '\<typename1>' to type '\<typename2>'</span></span>
<span data-ttu-id="a9080-103">Eine Prozedur wird aufgerufen, mit der ein Argument, das den entsprechenden Typ erweitert werden kann, und die Konvertierung aus dem Parameter mit dem Argument einschränkend.</span><span class="sxs-lookup"><span data-stu-id="a9080-103">A procedure is called with an argument that widens to the corresponding parameter type, and the conversion from the parameter to the argument is narrowing.</span></span>  
  
 <span data-ttu-id="a9080-104">Wenn Sie eine Klasse oder Struktur definieren, können Sie einen oder mehrere Konvertierungsoperatoren zum Konvertieren dieses Klassen- oder Strukturtyps in andere Typen definieren.</span><span class="sxs-lookup"><span data-stu-id="a9080-104">When you define a class or structure, you can define one or more conversion operators to convert that class or structure type to other types.</span></span> <span data-ttu-id="a9080-105">Sie können auch Operatoren für die umgekehrte Konvertierung definieren, um dieser anderen Typen zurück in den Klassen- oder Strukturtyp zu konvertieren.</span><span class="sxs-lookup"><span data-stu-id="a9080-105">You can also define reverse conversion operators to convert those other types back to your class or structure type.</span></span> <span data-ttu-id="a9080-106">Wenn Sie den Typ Klasse oder Struktur in einem Prozeduraufruf verwenden, kann Visual Basic diese Konvertierungsoperatoren verwenden, um den Typ eines Arguments in den Typ des entsprechenden Parameters konvertieren.</span><span class="sxs-lookup"><span data-stu-id="a9080-106">When you use your class or structure type in a procedure call, Visual Basic can use these conversion operators to convert the type of an argument to the type of its corresponding parameter.</span></span>  
  
 <span data-ttu-id="a9080-107">Wenn Sie das Argument übergeben [ByRef](../../../visual-basic/language-reference/modifiers/byref.md), kopiert Visual Basic zuweilen den Argumentwert in eine lokale Variable in der Prozedur, statt einen Verweis zu übergeben.</span><span class="sxs-lookup"><span data-stu-id="a9080-107">If you pass the argument [ByRef](../../../visual-basic/language-reference/modifiers/byref.md), Visual Basic sometimes copies the argument value into a local variable in the procedure instead of passing a reference.</span></span> <span data-ttu-id="a9080-108">In diesem Fall müssen Visual Basic klicken Sie dann die Prozedur gibt den Wert den lokalen Variablen zurück in das Argument im aufrufenden Code kopieren.</span><span class="sxs-lookup"><span data-stu-id="a9080-108">In such a case, when the procedure returns, Visual Basic must then copy the local variable value back into the argument in the calling code.</span></span>  
  
 <span data-ttu-id="a9080-109">Wenn ein `ByRef` -Argumentwert in die Prozedur kopiert wird und das Argument und der Parameter denselben Typ aufweisen, ist keine Konvertierung erforderlich.</span><span class="sxs-lookup"><span data-stu-id="a9080-109">If a `ByRef` argument value is copied into the procedure and the argument and parameter are of the same type, no conversion is necessary.</span></span> <span data-ttu-id="a9080-110">Aber wenn die Typen unterschiedlich sind, müssen in beide Richtungen Visual Basic konvertieren.</span><span class="sxs-lookup"><span data-stu-id="a9080-110">But if the types are different, Visual Basic must convert in both directions.</span></span> <span data-ttu-id="a9080-111">Wenn einer der Typen der Klasse oder Struktur-Typ ist, muss Visual Basic es sowohl vom anderen Typ konvertieren.</span><span class="sxs-lookup"><span data-stu-id="a9080-111">If one of the types is your class or structure type, Visual Basic must convert it both to and from the other type.</span></span> <span data-ttu-id="a9080-112">Wenn eine dieser Konvertierungen eine erweiternde Konvertierung handelt, kann die umgekehrte Konvertierung einschränken.</span><span class="sxs-lookup"><span data-stu-id="a9080-112">If one of these conversions is widening, the reverse conversion might be narrowing.</span></span>  
  
 <span data-ttu-id="a9080-113">**Fehler-ID:** BC32053</span><span class="sxs-lookup"><span data-stu-id="a9080-113">**Error ID:** BC32053</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a9080-114">So beheben Sie diesen Fehler</span><span class="sxs-lookup"><span data-stu-id="a9080-114">To correct this error</span></span>  
  
- <span data-ttu-id="a9080-115">Verwenden Sie möglichst ein aufrufendes Argument desselben Typs wie der Prozedurparameter ein, damit Visual Basic nicht Konvertierung vornehmen muss.</span><span class="sxs-lookup"><span data-stu-id="a9080-115">If possible, use a calling argument of the same type as the procedure parameter, so Visual Basic does not need to do any conversion.</span></span>  
  
- <span data-ttu-id="a9080-116">Wenn Sie die Prozedur mit einem Argument aufrufen müssen, dessen Typ sich vom Parametertyp unterscheidet, jedoch kein Wert in das aufrufende Argument zurückgegeben werden muss, definieren Sie den Parameter als [ByVal](../../../visual-basic/language-reference/modifiers/byval.md) anstelle von `ByRef`.</span><span class="sxs-lookup"><span data-stu-id="a9080-116">If you need to call the procedure with an argument type different from the parameter type but do not need to return a value into the calling argument, define the parameter to be [ByVal](../../../visual-basic/language-reference/modifiers/byval.md) instead of `ByRef`.</span></span>  
  
- <span data-ttu-id="a9080-117">Wenn ein Wert in das aufrufende Argument zurückgegeben werden sollen, definieren Sie den Operator für die umgekehrte Konvertierung als [Widening](../../../visual-basic/language-reference/modifiers/widening.md), sofern möglich.</span><span class="sxs-lookup"><span data-stu-id="a9080-117">If you need to return a value into the calling argument, define the reverse conversion operator as [Widening](../../../visual-basic/language-reference/modifiers/widening.md), if possible.</span></span>
