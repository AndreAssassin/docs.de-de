# <a name="doloop-statement-visual-basic"></a><span data-ttu-id="2a157-102">Do...Loop-Anweisung (Visual Basic)</span><span class="sxs-lookup"><span data-stu-id="2a157-102">Do...Loop Statement (Visual Basic)</span></span>
<span data-ttu-id="2a157-103">Wiederholt einen Anweisungsblock während einer `Boolean` Bedingung `True` oder bis die Bedingung wird `True`.</span><span class="sxs-lookup"><span data-stu-id="2a157-103">Repeats a block of statements while a `Boolean` condition is `True` or until the condition becomes `True`.</span></span>  
  
## <a name="syntax"></a><span data-ttu-id="2a157-104">Syntax</span><span class="sxs-lookup"><span data-stu-id="2a157-104">Syntax</span></span>  
  
```  
Do { While | Until } condition  
    [ statements ]  
    [ Continue Do ]  
    [ statements ]  
    [ Exit Do ]  
    [ statements ]  
Loop  
-or-  
Do  
    [ statements ]  
    [ Continue Do ]  
    [ statements ]  
    [ Exit Do ]  
    [ statements ]  
Loop { While | Until } condition  
```
