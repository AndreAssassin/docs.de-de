# <a name="visual-basic-language-specification"></a>Visual Basic-Sprachspezifikation

Die Visual Basic-Sprachspezifikation ist die autoritative Quelle für Antworten auf alle Fragen zu Visual Basic Grammatik und Syntax. Sie enthält ausführliche Informationen zur Sprache, darunter auch viele Punkte, die in der Visual Basic Referenz Dokumentation nicht behandelt werden.  
  
Die Spezifikation ist im [Microsoft Download Center](https://go.microsoft.com/fwlink/?LinkId=188623) verfügbar.  
  
Diese Site enthält die [Spezifikation Visual Basic 11](../../../../_vblang/spec/introduction.md). Diese wird aus den Markdowndateien erstellt, die im [dotnet/vblang-GitHub-Repository](https://github.com/dotnet/vblang/blob/master/spec/README.md) enthalten sind.

Probleme mit der Entwurfsspezifikation sollten im Repository [dotnet/vblang](https://github.com/dotnet/vblang/issues) erstellt werden. Wenn Sie alle auftretenden Fehler beheben möchten, können Sie auch einen [Pull Request](https://github.com/dotnet/vblang/pulls) an dasselbe Repository senden.
