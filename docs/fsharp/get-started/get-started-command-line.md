<span data-ttu-id="34852-116">Nach dem Ausführen des vorherigen Befehls wird die folgende Verzeichnisstruktur erstellt:</span><span class="sxs-lookup"><span data-stu-id="34852-116">The following directory structure is produced after running the previous command:</span></span>

        └── Library

<span data-ttu-id="34852-120">Führen Sie `dotnet build`zum Erstellen des Projekts aus.</span><span class="sxs-lookup"><span data-stu-id="34852-120">Run `dotnet build` to build the project.</span></span> <span data-ttu-id="34852-121">Beim Erstellen werden nicht aufgelöste Abhängigkeiten wiederhergestellt.</span><span class="sxs-lookup"><span data-stu-id="34852-121">Unresolved dependencies will be restored when building.</span></span>

<span data-ttu-id="34852-124">Nach dem Ausführen des vorherigen Befehls wird die folgende Verzeichnisstruktur erstellt:</span><span class="sxs-lookup"><span data-stu-id="34852-124">The following directory structure is produced after running the previous command:</span></span>

<span data-ttu-id="34852-125">Ersetzen Sie den Inhalt von der Datei `Program.fs`durch den folgenden Code:</span><span class="sxs-lookup"><span data-stu-id="34852-125">Replace the contents of the `Program.fs` file with the following code:</span></span>
