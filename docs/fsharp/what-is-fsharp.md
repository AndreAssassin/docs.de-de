## <a name="rich-data-types"></a><span data-ttu-id="d124b-117">Rich-Datentypen</span><span class="sxs-lookup"><span data-stu-id="d124b-117">Rich data types</span></span>

<span data-ttu-id="d124b-118">Mit Datentypen wie [Datensätzen](./language-reference/records.md) und Unterscheidungs-[Unions](./language-reference/discriminated-unions.md) können Sie komplexe Daten und Domänen darstellen.</span><span class="sxs-lookup"><span data-stu-id="d124b-118">Data types such as [Records](./language-reference/records.md) and [Discriminated Unions](./language-reference/discriminated-unions.md) let you represent complex data and domains.</span></span>

```fsharp
// Group data with Records
type SuccessfulWithdrawal = {
    Amount: decimal
    Balance: decimal
}

type FailedWithdrawal = {
    Amount: decimal
    Balance: decimal
    IsOverdraft: bool
}

// Use discriminated unions to represent data of 1 or more forms
type WithdrawalResult =
    | Success of SuccessfulWithdrawal
    | InsufficientFunds of FailedWithdrawal
    | CardExpired of System.DateTime
    | UndisclosedFailure
```
