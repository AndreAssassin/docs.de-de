<span data-ttu-id="f90f1-104">*Abstrakte Klassen* sind Klassen, die einige oder alle Member nicht implementiert lassen, sodass Implementierungen von abgeleiteten Klassen bereitgestellt werden können.</span><span class="sxs-lookup"><span data-stu-id="f90f1-104">*Abstract classes* are classes that leave some or all members unimplemented, so that implementations can be provided by derived classes.</span></span>

```fsharp
// Abstract class syntax.
[<AbstractClass>]

[ inherit base-class-or-interface-name ]
[ abstract-member-declarations-and-member-definitions ]

```
