## <a name="value-restriction"></a><span data-ttu-id="d19eb-122">Wertebeschränkung</span><span class="sxs-lookup"><span data-stu-id="d19eb-122">Value Restriction</span></span>

<span data-ttu-id="d19eb-123">Der Compiler führt die automatische Verallgemeinerung nur für vollständige Funktionsdefinitionen, die expliziten Argumente haben, und klicken Sie auf einfache unveränderlichen Werten.</span><span class="sxs-lookup"><span data-stu-id="d19eb-123">The compiler performs automatic generalization only on complete function definitions that have explicit arguments, and on simple immutable values.</span></span>

<span data-ttu-id="d19eb-124">Dies bedeutet, dass der Compiler einen Fehler ausgibt, wenn Sie versuchen, Code zu kompilieren, die nicht ausreichend sollen einen bestimmten Typ beschränkt ist, jedoch ist auch nicht generalisierbar.</span><span class="sxs-lookup"><span data-stu-id="d19eb-124">This means that the compiler issues an error if you try to compile code that is not sufficiently constrained to be a specific type, but is also not generalizable.</span></span> <span data-ttu-id="d19eb-125">Die Fehlermeldung für dieses Problem bezieht sich auf die diese Einschränkung für die automatische Verallgemeinerung für Werte als die *wertebeschränkung*.</span><span class="sxs-lookup"><span data-stu-id="d19eb-125">The error message for this problem refers to this restriction on automatic generalization for values as the *value restriction*.</span></span>

- <span data-ttu-id="d19eb-129">Wenn das Problem ein Konstrukt verallgemeinerbaren definieren Sie eine generische Funktion, wie z. - <span data-ttu-id="d19eb-130">Wenn das Problem ein Ausdruck ist, der ist zu komplex für den generalisiert werden, stellen es in eine Funktion, durch das Hinzufügen eines zusätzlichen, nicht verwendeten Parameters.</span><span class="sxs-lookup"><span data-stu-id="d19eb-130">If the problem is an expression that is too complex to be generalized, make it into a function by adding an extra, unused parameter.</span></span>

- <span data-ttu-id="d19eb-131">Fügen Sie explizite generische Typparameter hinzu.</span><span class="sxs-lookup"><span data-stu-id="d19eb-131">Add explicit generic type parameters.</span></span> <span data-ttu-id="d19eb-132">Diese Option wird nur selten verwendet.</span><span class="sxs-lookup"><span data-stu-id="d19eb-132">This option is rarely used.</span></span>
