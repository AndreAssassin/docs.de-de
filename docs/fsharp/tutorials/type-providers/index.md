<span data-ttu-id="c72eb-104">Ein F#-Typanbieter ist eine Komponente, die Typen, Eigenschaften und Methoden zur Verwendung im Programm bereitstellt.</span><span class="sxs-lookup"><span data-stu-id="c72eb-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="c72eb-105">Typanbieter generiert, so genannten **bereitgestellten Typen**, die von F#-Compiler generiert werden und basieren auf einer externen Datenquelle.</span><span class="sxs-lookup"><span data-stu-id="c72eb-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="c72eb-108">Vorausgesetzt, dass die Typen von Eingabeparametern für ein Typanbieter abhängig sind.</span><span class="sxs-lookup"><span data-stu-id="c72eb-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="c72eb-109">Diese Eingabe kann eine Beispieldatenquelle (z. Onlinedatenanbietern.</span><span class="sxs-lookup"><span data-stu-id="c72eb-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="c72eb-112">Produktive und gelöschten Typanbietern</span><span class="sxs-lookup"><span data-stu-id="c72eb-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="c72eb-113">Typanbieter gibt zwei Arten: Produktive und gelöschten.</span><span class="sxs-lookup"><span data-stu-id="c72eb-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="c72eb-114">Produktive Typanbieter erzeugen, Typen, die als Typen in .NET in die Assembly geschrieben werden können in denen sie erstellt wurden.</span><span class="sxs-lookup"><span data-stu-id="c72eb-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="c72eb-115">Dadurch können sie aus Code in anderen Assemblys verwendet werden.</span><span class="sxs-lookup"><span data-stu-id="c72eb-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="c72eb-116">Dies bedeutet, dass die typisierte Darstellung der Datenquelle in der Regel eine sein muss, die mit .NET-Typen darstellen möglich ist.</span><span class="sxs-lookup"><span data-stu-id="c72eb-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="c72eb-117">Löschung Typanbieter erzeugen, Typen, die nur genutzt werden können, in der Assembly oder das Projekt, das sie generiert werden.</span><span class="sxs-lookup"><span data-stu-id="c72eb-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="c72eb-118">Die Typen sind kurzlebige; d. h. sie werden nicht in eine Assembly geschrieben und können nicht von Code in anderen Assemblys genutzt werden.</span><span class="sxs-lookup"><span data-stu-id="c72eb-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="c72eb-119">Sie können enthalten *verzögert* Member, da Sie die bereitgestellten Typen aus einer potenziell unbegrenzte Informationen.</span><span class="sxs-lookup"><span data-stu-id="c72eb-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="c72eb-120">Sie eignen sich für eine kleine Teilmenge der eine große und miteinander verknüpfte Datenquelle verwenden.</span><span class="sxs-lookup"><span data-stu-id="c72eb-120">They are useful for using a small subset of a large and interconnected data source.</span></span>
